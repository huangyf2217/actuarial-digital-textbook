第2章 风险模型

本章学习目标

  1. 基于索赔频率和索赔强度，构建适合短期保险合同的模型并描述其假设。
  2. 理解N个独立随机变量之和的分布。
  3. 了解常见的复合分布，并推导相应的数字特征。
  4. 掌握复合泊松分布（compound Poisson distribution）的重要性质。
  5. 认识短期个体风险模型，理解其与短期聚合风险模型的区别。

2.1 短期聚合风险模型

许多非人寿保险可以视为短期合同，例如汽车保险，某些形式的人寿保险也可视为短期合同，例如团体人寿一年期保单。此类短期保险合同通常具有以下特点。

第一，保单持续一段固定的、相对较短的时间，通常为一年。

第二，保险人支付保单有效期内产生的索赔，但索赔金额不是确定的，而是取决于实际损失情况。

第三，发生索赔并不会使保单立即终止，不同于某些寿险合同，并非“索赔一次即终止”的保单。

第四，在非寿险合同中，索赔可以在保单期间的任何时间发生。尽管投保人通常有义务尽快向保险人报告索赔，但如果损失不严重，投保人可能会出现报案延迟。在一些复杂情况下，如果无法立即确定损失的程度，或者需要长期的法律程序，索赔可能会持续较长时间。但无论索赔发生的时机如何，从引起索赔的事件发生之时起，最终结算金额即为保险公司的责任。估计这些负债所需的准备金是精算师的重要工作。

保单期限届满，投保人可以续保，也可以不续保；续保后，投保人应付保险费可以与上期相同，也可以不相同。保险人可以选择将部分保险费转给再保险人；作为回报，再保险人应当按照约定，在保单有效期内向保险人支付部分赔偿费用。

更具体地，本章将考虑一份涵盖风险的短期保险合同，风险包括单个保单或一组保单。保单期限通常假定为一年（但也可以是其他任何期限，如6个月），随机变量$S$表示保险人在保单期限内就这一风险支付的累积索赔金额。本章主要讨论随机变量$S$的分布和特征，首先研究聚合风险模型，随后将聚合风险模型的概念扩展到个体风险模型。

2.1.1 基本模型假设

定义 2.1（短期聚合风险模型）

聚合风险模型的核心是应用保单期限内发生的索赔次数（以离散型随机变量$N$表示）和每次索赔的金额（以随机变量$X_i~(i=1,2,\cdots,N)$表示）刻画累积索赔金额，即：

$$S = X_1+X_2+\cdots+X_N = \sum_{i=1}^{N} X_i$$

$S$表示为$N$个随机变量$X_i$之和，如果索赔次数$N$为零，则累积索赔金额为零。该框架称为聚合风险模型（collective risk model），因其通常是将所有保单或事件的风险进行聚合，将$N$作为一个风险集体的索赔次数进行分析。

$S$的分解允许对索赔次数和每次索赔金额分别建模，这样做的一个优势是，实际中影响索赔次数和索赔金额的因素可能是不同的。以汽车保险为例，长时间的恶劣天气可能会对索赔次数产生显著影响，但对单次索赔金额的分布几乎没有影响。另一方面，通货膨胀可能会对汽车维修成本产生重大影响，从而影响单次索赔金额的分布，但对索赔次数影响很小或没有影响。

与实际的短期保险业务相比，聚合风险模型进行了一些简化。此处假定$N$和$X_{i}$的矩（或分布）是确定的，且一旦引起索赔的事件发生，索赔就会无时滞解决。此外，利息是短期保险的一个相对次要的特征，短期保险模型有可能包括利率，但聚合风险模型中通常忽略利率的影响。

将要研究的问题是：在不考虑再保险的情况下，如何根据$N$和$X_i$的矩和分布推导出$S$的矩和分布。为此，首先作出以下两个重要假设：

  • 随机变量 $\{X_i\}_{i=1}^{N}$ 是独立同分布的。
  • 随机变量 $N$ 与 $\{X_i\}_{i=1}^{N}$ 相互独立。

上述假设意味着投保人的索赔次数与索赔金额之间互不影响，各次索赔的金额互不影响，且投保人索赔金额的分布在保单期限（短期）内不会发生变化。

记$F(x)$和$G(x)$分别为$X_i$和$S$的分布函数，并且假设所有索赔金额均为非负值，即$F(x)=0,\ \forall x<0$。通常假定$F(x)$的密度函数存在，记为$f(x)$。在密度函数不存在的情况下，$X_i$的各阶原点矩$m_k$也可以被导出。

对于随机变量$N$，一般假设其服从常见的离散型分布，如泊松分布和负二项分布。每种分布具有不同的适用场景，例如：泊松分布通常用于描述固定时间内发生的索赔次数；而负二项分布则适用于索赔次数“过度分散”的情况，即事件发生的次数在统计上呈现出较大的波动性，通常出现在某些高风险的险种。

对于随机变量$X$，一般可以考虑伽马分布、逆高斯分布等常见的连续型分布（也可以假定$X$服从取值为非负整数的离散型分布）。分布的选择应考虑索赔金额的规律，例如：伽马分布常用于索赔金额较小且大部分集中在低值区间的情况；而当数据呈较严重的偏斜、特别是存在极端大额索赔时，则更适合考虑对数正态分布或威布尔分布。

2.1.2 $S$的分布和特征

$S$的分布函数$G(x)$可以通过事件$\{S \leq x\}$导出。注意到若该事件发生，则必有以下其中一个事件发生：

  • $\{N = 0 \}$，即无索赔；
  • $\{N = 1,\ S = X_1 \leq x \}$，即只有一次索赔，索赔金额 $\leq x$；
  • $\{N = 2,\ S = X_1+X_2 \leq x \}$，即发生两次索赔，累积索赔金额 $\leq x$；
  • $~~~~~~~~~~~~~~~~~~~\vdots$
  • $\{N = r,\ S = X_1+ \cdots + X_r \leq x \}$，即发生 $r$ 次索赔，累积索赔金额 $\leq x$；
  • $~~~~~~~~~~~~~~~~~~~\vdots$

这些事件是互斥和无穷的，因此有：

$$\begin{gathered}\{S \leq x\} = \bigcup_{n=0}^{\infty} \{S \leq x, N = n\} \\ \text{P}(S \leq x) = \sum_{n=0}^{\infty} \text{P}(S \leq x, N = n) = \sum_{n=0}^{\infty} \text{P}(N = n) \cdot \text{P}(S \leq x | N = n)\end{gathered}$$

注意，当 $N = n$ 时，$S$ 是固定个随机变量 $\{X_i\}_{i=1}^{n}$ 之和，即：

$$\text{P}(S \leq x | N = n) = F^{n*}(x)$$

其中，$F^{n*}(x)$ 是分布 $F(x)$ 的 $n$ 重卷积。注意 $F^{1*}(x)$ 即为 $F(x)$，为方便起见，定义$F^{0*}(x)$ 在所有非负 $x$ 上的值为1，否则取值为0。

综合上面两式，可以得到：

$$G(x) = \sum_{n=0}^{\infty} \text{P}(N = n) F^{n*}(x)$$

该式为累积索赔金额$S$的分布函数的一般表达式，对于$N$和$X_i$的分布无特殊要求。

当$X_i$仅在正整数上取值时，可以计算累积索赔金额等于$x = 1, 2, 3, \ldots$的概率为：

$$\begin{aligned} \text{P}(S = x) &= G(x) - G(x - 1) \\ &= \sum_{n=1}^{\infty} \text{P}(N = n)\left[F^{n*}(x) - F^{n*}(x - 1)\right] \\ &= \sum_{n=1}^{\infty} \text{P}(N = n) f^{n*}(x) \end{aligned}$$

其中，$f^{n*}(x) = F^{n*}(x) - F^{n*}(x - 1)$是$\sum_{i=1}^{n} X_i=x$的概率函数。

当$X_i$为连续随机变量时，$\text{P}(S = 0)$当且仅当$\text{P}(N = 0)$。

对于$S$的期望和方差，可以使用索赔次数$N$作为条件，通过重期望公式和重方差公式计算。首先，根据重期望公式，有：

$$\mathbb{E}(S) = \mathbb{E}[\mathbb{E}(S | N)]$$

其中，给定$N$时的条件期望易计算为：

$$\mathbb{E}(S | N = n) = \sum_{i=1}^{n} \mathbb{E}(X_i) = n \mathbb{E}(X)$$

因此，$S$的期望等于：

$$\mathbb{E}(S) = \mathbb{E}[N \cdot \mathbb{E}(X)] = \mathbb{E}(N) \mathbb{E}(X)$$

上式具有非常自然的解释，它表明累积索赔金额的期望是预期索赔次数和每次索赔金额期望的乘积。

其次推导累积索赔金额的方差，应用重方差公式，有：

$$\text{Var}(S) = \mathbb{E}[\text{Var}(S|N)] + \text{Var}[\mathbb{E}(S|N)]$$

已知每次索赔的金额是独立的，则可以求得$\text{Var}(S|N)$等于：

$$\text{Var}(S|N = n) = \text{Var} \left( \sum_{i=1}^{n} X_i \right) = \sum_{i=1}^{n} \text{Var}(X_i) = n(m_2 - m_1^2)$$

其中，$m_k$是$X_i$的$k$阶原点矩。

将()和()代入重方差公式，得到：

$$\begin{aligned} \text{Var}(S) &= \mathbb{E}[N(m_2 - m_1^2)] + \text{Var}(Nm_1) \\ &= \mathbb{E}(N)(m_2 - m_1^2) + \text{Var}(N)m_1^2 \end{aligned}$$

即$S$的方差可以由$N$和$X_i$的期望和方差表示。

$S$的矩母函数也可以根据条件期望推导。由定义$M_S(t) = \mathbb{E}[\exp(tS)]$，应用重期望公式可得：

$$M_S(t) = \mathbb{E}\left\{ \mathbb{E}\left[ \exp(tS) | N \right] \right\}$$

其中，条件期望$\mathbb{E}\left[ \exp(tS) | N \right]$等于：

$$\begin{aligned} \mathbb{E}\left[\exp(tS) | N = n\right] &= \mathbb{E}\left[\exp(tX_1 + tX_2 + \cdots + tX_n)\right] \\ &= \mathbb{E}\left[\prod_{i=1}^{n} \exp(tX_i)\right] \end{aligned}$$

由于$\{X_i\}_{i=1}^{n}$为独立同分布的随机变量，具有相同的矩母函数函数$M_X(t)$；且根据概率论的相关性质，指数变换后的各$\exp(tX_i)$仍然相互独立，则：

$$\begin{aligned} \mathbb{E}\left[\exp(tS) | N = n\right] &= \mathbb{E}\left[\prod_{i=1}^{n} \exp(tX_i)\right] \\ &= \prod_{i=1}^{n} \mathbb{E}\left[\exp(tX_i)\right] \\ &= \prod_{i=1}^{n} M_X(t) = [M_X(t)]^n \end{aligned}$$

将上式代入上式，可得：

$$\begin{aligned} M_S(t) &= \mathbb{E}\left\{\left[M_X(t)\right]^N\right\} \\ &= \mathbb{E}\left\{\exp[N \cdot \ln M_X(t)]\right\} \\ &= M_N(\ln M_X(t)) \end{aligned}$$

即$S$的矩母函数可以用$N$和$X_i$的矩母函数表示。与前述结果类似，对于$N$和$X_i$的分布无特殊要求。

例 2.1

假设：索赔次数以概率$(0.3, 0.5, 0.2)$取值为$(0, 1, 2)$；每次索赔的金额（索赔强度）以概率$(0.2, 0.4, 0.2, 0.1, 0.1)$取值于$(0, 100, 200, 300, 400)$。计算累积损失金额的期望和方差。

首先，根据索赔次数和索赔强度的概率分布，计算期望值$\mathbb{E}(N)$和$\mathbb{E}(X)$：

$$\begin{gathered}\mathbb{E}(N) = 0 \cdot 0.3 + 1 \cdot 0.5 + 2 \cdot 0.2 = 0.9 \\ \mathbb{E}(X) = 0 \cdot 0.2 + 100 \cdot 0.4 + 200 \cdot 0.2 + 300 \cdot 0.1 + 400 \cdot 0.1 = 150\end{gathered}$$

根据上式，可计算累积损失金额的期望值$\mathbb{E}(S)$为：

$$\mathbb{E}(S) = \mathbb{E}[\mathbb{E}(S|N)] = \mathbb{E}(N) \cdot \mathbb{E}(X) = 135$$

为计算$S$的方差，需要先计算$N$和$X$的方差。对于索赔次数$N$，有

$$\begin{gathered}\mathbb{E}(N^2) = 0^2 \cdot 0.3 + 1^2 \cdot 0.5 + 2^2 \cdot 0.2 = 1.3 \\ \text{Var}(N) = \mathbb{E}(N^2) - [\mathbb{E}(N)]^2 = 1.3 - 0.9^2 = 0.49\end{gathered}$$

对于索赔强度$X$，有：

$$\begin{gathered}\mathbb{E}(X^2) = 0^2 \cdot 0.2 + 100^2 \cdot 0.4 + 200^2 \cdot 0.2 + 300^2 \cdot 0.1 + 400^2 \cdot 0.1 = 370000 \\ \text{Var}(X) = \mathbb{E}(X^2) - [\mathbb{E}(X)]^2 = 370000 - 150^2 = 370000 - 22500 = 347500\end{gathered}$$

因此，将结果代入累积索赔金额的方差公式，得到$S$的方差为：

$$\text{Var}(S) = \mathbb{E}(N) \text{Var}(X) + \text{Var}(N) [\mathbb{E}(X)]^2 = 323775$$

def panjer_poisson(p, lam):
    """Panjer递推（泊松分布）"""
    cumul = np.exp(-lam * np.sum(p))
    f = [cumul]
    s = 0
    while cumul <= 0.99999999:
        s += 1
        m = min(s, len(p))
        last = lam / s * np.sum(np.arange(1, m+1) * p[:m] * f[s-m:s][::-1])
        f.append(last)
        cumul += last
    return np.array(f)

2.1.3 复合泊松分布

本节将继续讨论当$N$服从常见的离散型分布时，累积索赔金额$S$的分布规律。首先考虑当$N$服从参数为$\lambda$的泊松分布时的累积索赔金额，此时$S$的分布称为复合泊松分布（compound Poisson distribution）。

根据泊松分布的性质，有：

$$\begin{gathered}\mathbb{E}(N) = \text{Var}(N) = \lambda \\ M_N (t) = \exp [\lambda(e^t - 1)]\end{gathered}$$

代入2.1.2节的结果，可以得到$S$的期望、方差和矩母函数分别为：

$$\begin{gathered}\mathbb{E}(S) = \lambda m_1 \\ \text{Var}(S) = \lambda m_2 \\ M_S(t) = \exp \{\lambda[M_X(t) - 1]\} \end{gathered}$$

可见期望和方差的结果非常简单。注意，$S$的方差用$X_i$的二阶原点矩表示（而不是用$X_i$的方差表示）。

同时，当$S$服从复合泊松分布时，$S$的3阶中心矩有一个简单的形式，等于$\lambda m_3$。计算$S$的3阶中心矩的最简单方法是应用累积量生成函数$C_S(t) = \ln M_S(t)=\lambda[M_X(t) - 1]$，将其关于$t$求三阶导，并令$t=0$，结果为：

$$\begin{aligned} \mathbb{E}[(S - \mathbb{E}S)^3] &= \left.\frac{\mathrm{d}^3}{\mathrm{d}t^3}\ln M_S(t)\right|_{t=0} \\ &= \lambda \left. \frac{\mathrm{d}^3}{\mathrm{d}t^3}[M_X(t) - 1]\right|_{t=0} = \lambda m_3 \end{aligned}$$

因此，可知累积索赔金额分布的偏态系数为$\lambda m_3 / (\lambda m_2)^{3/2}$。

上述结果表明$S$的分布是右偏的，因为$m_3$是非负随机变量$X_i$关于原点的3阶矩，恒大于零；故无论$X_i$分布的偏态如何，$S$的分布总是呈右偏。注意到$S$的偏态系数当$\lambda \to \infty$时趋向于$0$，故当$\lambda$取较大值时，$S$的分布将趋向于对称。

泊松分布的一个重要性质是具有可加性，即两个独立的泊松随机变量之和仍然服从泊松分布。该性质可通过卷积方法进行证明。

例 2.2

设$X \sim \text{Poi}(\lambda)$和$Y \sim \text{Poi}(\mu)$是两个相互独立的泊松随机变量，使用卷积求$Z = X + Y$的概率分布。

根据离散型随机变量的卷积公式，有：

$$\text{P}(Z = z) = \sum_{x=0}^{z} \text{P}(X = x) \text{P}(Y = z - x),\ z=0,1,2,\cdots$$

代入 $\text{P}(X = x) = \frac{\lambda^x e^{-\lambda}}{x!}$ 和 $\text{P}(Y = z - x) = \frac{\mu^{z-x} e^{-\mu}}{(z-x)!}$ 得：

$$\begin{aligned}\text{P}(Z = z) &= \sum_{x=0}^{z} \frac{\lambda^x e^{-\lambda}}{x!} \cdot \frac{\mu^{z-x} e^{-\mu}}{(z-x)!} \\ &= \frac{e^{-(\lambda + \mu)}}{z!} \sum_{x=0}^{z} \frac{z!}{x!(z - x)!} \lambda^x \mu^{z - x}\end{aligned}$$

注意到 $\sum_{x=0}^{z} \frac{z!}{x!(z - x)!} \lambda^x \mu^{z - x}$ 是二项式展开式，因此可以得到：

$$\text{P}(Z = z) = \frac{(\lambda + \mu)^z}{z!} e^{-(\lambda + \mu)}$$

上式恰好为$\text{Poi}(\lambda + \mu)$的概率分布。通过矩母函数可以更容易证明这个结果。

类似地，复合泊松分布的一个重要性质是：独立复合泊松随机变量之和仍然服从复合泊松分布。该性质的表述及证明如下。

记随机变量$S_1, \dots, S_n$表示$n$个不同保单组合的累积索赔金额，$\{S_j\}_{j=1}^{n}$之间相互独立：各$S_j$服从复合泊松分布，每个保单组合的索赔次数服从$\text{Poi}(\lambda_j)$，每次索赔金额的分布函数记为$F_j(x)$。定义$A = S_1 + \cdots + S_n$表示汇总保单组合的累积索赔金额，则$A$仍然服从复合泊松分布，对应索赔次数$N$的泊松参数等于$\Lambda = \sum_{j=1}^{n} \lambda_j$，每次索赔金额$X$的分布函数具有如下形式：

$$F(x) = \frac{1}{\Lambda} \sum_{j=1}^{n} \lambda_j F_j(x)$$

为证明这一性质，首先注意到$F(x)$是分布函数$F_j(x)$的加权平均值，权重为正且加和为1。因此$F(x)$是一个分布函数，且矩母函数可以表示为：

$$\begin{aligned} M(t) &= \int_{0}^{\infty} \exp(tx) \frac{1}{\Lambda} \sum_{j=1}^{n} \lambda_j f_j(x) \mathrm{d}x \\ &= \frac{1}{\Lambda} \sum_{j=1}^{n} \lambda_j \int_{0}^{\infty} \exp(tx) f_j(x) \mathrm{d}x \\ &= \frac{1}{\Lambda} \sum_{j=1}^{n} \lambda_j M_j(t) \end{aligned}$$

其中，$f_j(x)$ 是 $F_j(x)$ 的概率密度函数，$M_j(t)$是$F_j(x)$对应的矩母函数。

记$M_A(t)$表示$A$的矩母函数，根据定义有：

$$\begin{aligned} M_A(t) &= \mathbb{E}[\exp(tA)] = \mathbb{E}[\exp(tS_1 + tS_2 + \dots + tS_n)] \\ &= \prod_{j=1}^{n} \mathbb{E}[\exp(tS_j)] \end{aligned}$$

由于$\{S_j\}_{j=1}^{n}$相互独立，随机变量的乘积和期望可以换序。

当$S_j$为复合泊松随机变量时，矩母函数的形式可由下式给出：

$$\mathbb{E}[\exp(tS_j)] = \exp\left\{\lambda_j[M_j(t) - 1]\right\}$$

将该结果代入上式，最终得到：

$$\begin{aligned} M_A(t) &= \prod_{j=1}^{n} \exp\left\{\lambda_j[M_j(t) - 1]\right\} \\ &= \exp\left\{\sum_{j=1}^{n} \lambda_j [M_j(t) - 1]\right\} \\ &= \exp\left\{\Lambda [M(t) - 1]\right\} \end{aligned}$$

其中，$\Lambda = \sum_{j=1}^{n} \lambda_j$且$M(t) = \frac{1}{\Lambda} \sum_{j=1}^{n} \lambda_j M_j(t)$。

将上式与上式进行对比，同时注意到分布函数和矩母函数的一一对应关系，可知$A$仍然服从复合泊松分布，索赔次数的泊松分布参数为$\Lambda$，且每次索赔金额的分布函数恰好为$F(x)$。

## 2.4.1 Panjer递推
def panjer_poisson(p, lam):
    """Panjer递推（泊松分布）"""
    cumul = np.exp(-lam * np.sum(p))
    f = [cumul]
    s = 0
    while cumul <= 0.99999999:
        s += 1
        m = min(s, len(p))
        last = lam / s * np.sum(np.arange(1, m+1) * p[:m] * f[s-m:s][::-1])
        f.append(last)
        cumul += last
    return np.array(f)

2.1.4 其他常用复合分布

1. 复合二项分布

在某些情况下，二项分布是$N$的自然选择。例如，假设每个被保险人的生命都受到相同死亡率的影响，且各人的死亡相互独立，那么对于包含$n$个个体的团体人寿保单组合，一年中的死亡人数（索赔次数）服从二项分布。选择二项分布的一个重要特点是，索赔次数存在一个自然上限$n$。

记$N \sim \text{B(n,p)}$服从二项分布。根据二项分布的性质，有：

$$\begin{gathered}\mathbb{E}(N) = np \\ \text{Var}(N) = np(1 - p) \\ M_N (t) = (pe^t + 1 - p)^n\end{gathered}$$

当$N$服从二项分布时，称$S$的分布为复合二项分布（compound binomial distribution）。代入2.1.2节的结果，可以得到$S$的期望、方差和矩母函数分别为：

$$\begin{gathered}\mathbb{E}(S) = np m_1 \\ \text{Var}(S) = np(m_2 - m_1^2) + np(1 - p)m_1^2 = np m_2 - np^2 m_1^2 \\ M_S(t) = [p M_X(t) + 1 - p]^n\end{gathered}$$

3阶中心矩同样可以通过矩母函数求出：

$$\frac{\mathrm{d}^3}{\mathrm{d}t^3} \ln M_S(t) = \frac{\mathrm{d}^3}{\mathrm{d}t^3} \left\{ n \ln[pM_X(t) + q] \right\}$$

其中，$q = 1 - p$。

对上式进一步展开，结果为：

$$\begin{aligned} \frac{\mathrm{d}^3}{\mathrm{d}t^3} \ln M_S(t) &= n \cdot \frac{\mathrm{d}^2}{\mathrm{d}t^2} \left[ \frac{p M_X^{'}(t)}{p M_X(t) + q} \right] \\ &= n \cdot \frac{\mathrm{d}}{\mathrm{d}t} \left\{ \frac{p M_X^{''}(t)[p M_X(t) + q] - [p M_X^{'}(t)]^2}{[p M_X(t) + q]^2} \right\} \\ &= n \cdot \frac{p M_X^{'''}(t)[p M_X(t) + q] - 3p M_X^{''}(t) p M_X^{'}(t) + 2[p M_X^{'}(t)]^3}{[p M_X(t) + q]^3} \end{aligned}$$

令$t = 0$，可求得复合二项分布的3阶中心矩等于：

$$\mathbb{E}[(S - np m_1)^3] = np m_3 - 3np^2 m_2 m_1 + 2np^3 m_1^3$$

偏态系数为：

$$\frac{np m_3 - 3np^2 m_2 m_1 + 2np^3 m_1^3}{(np m_2 - np^2 m_1^2)^{3/2}}$$

由上式可以得知，复合二项分布可能是左偏（负偏斜）的。一个简单例子是当每次索赔金额固定为$B>0$时，$S = NB$的三阶中心矩为：

$$\mathbb{E}[(S - \mathbb{E}S)^3] = B^3 \mathbb{E}[(N - \mathbb{E}N)^3]$$

故$S$的偏态系数是$N$的偏态系数的正数倍数。如果$p > 0.5$，则$N$的二项分布呈左偏。

例 2.4

如果索赔次数$N \sim \text{B}(100, 0.01)$服从二项分布，且每次索赔的金额$X \sim \text{Ga}(\alpha=10,\lambda=0.2)$服从伽马分布，试求累积索赔金额$S$的期望、方差以及矩母函数。

首先，计算$S$的期望：

$$\mathbb{E}(S) = \mathbb{E}(N) \mathbb{E}(X) = 100 \times 0.01 \times \frac{10}{0.2} = 50$$

为计算$S$的方差，需要先计算$N$和$X$的方差，分别为：

$$\begin{gathered}\text{Var}(N) = np(1-p) = 100 \times 0.01 \times 0.99 = 0.99 \\ \text{Var}(X) = \frac{\alpha}{\lambda^2} = \frac{10}{0.2^2} = 250\end{gathered}$$

因此，将结果代入累积索赔金额的方差公式，得到$S$的方差为：

$$\text{Var}(S) = \mathbb{E}(N) \text{Var}(X) + \text{Var}(N) [\mathbb{E}(X)]^2 = 2725$$

根据二项分布、伽马分布的矩母函数，有：

$$M_N(t) = \left( 0.99 + 0.01 e^t \right)^{100},   M_X(t) = \left( 1 - \frac{t}{0.2} \right)^{-10}$$

代入复合分布的矩母函数形式，结果为：

$$M_S(t) = M_N(\ln M_X(t)) = \left[ 0.99 + 0.01 \left( 1 - \frac{t}{0.2} \right)^{-10} \right]^{100}$$

此处对$S$的矩母函数求导可同样求得期望和方差的结果，但这种方法会产生大量计算，直接使用复合分布的期望和方差公式更加简单。

## 2.4.1 Panjer递推
def panjer_poisson(p, lam):
    """Panjer递推（泊松分布）"""
    cumul = np.exp(-lam * np.sum(p))
    f = [cumul]
    s = 0
    while cumul <= 0.99999999:
        s += 1
        m = min(s, len(p))
        last = lam / s * np.sum(np.arange(1, m+1) * p[:m] * f[s-m:s][::-1])
        f.append(last)
        cumul += last
    return np.array(f)

2. 复合负二项分布

当索赔次数$N$服从负二项分布时，对应累积索赔金额$S$的分布称为复合负二项分布（compound negative binomial distribution）。在上式中，令$k=r,p=1/(1+\beta)$，可以得到负二项分布的另一种形式为：

$$\text{P}(N = n) = \binom{k+n-1}{n} p^k q^n,\ n=0,1,2,\cdots$$

记为$N \sim \text{NB}(k,p)$。其中，参数$k>0$，参数$p \in (0,1)$，且$q=1-p$。

此时，$N$的期望、方差和矩母函数分别为：

$$\begin{gathered}\mathbb{E}(N) = kq/p \\ Var(N) = kq/p^2 \\ M_N(t) = \left(\frac{p}{1 - qe^t}\right)^k\end{gathered}$$

当$k = 1$时，负二项分布退化成几何分布。

负二项分布是泊松分布的一种替代，与泊松分布相比的特点在于负二项分布的方差大于均值（泊松分布的方差等于均值）。因此，由于现实中索赔次数的方差往往超过其均值，此时负二项分布可能更适合刻画其分布规律。

代入2.1.2阶的结果，可以得到$S$的期望、方差和矩母函数分别为：

$$\begin{gathered}\mathbb{E}(S) = \frac{kq}{p} m_1 \\ \text{Var}(S) = \frac{kq}{p}(m_2 - m_1^2) + \frac{kq}{p^2} m_1^2 = \frac{kq}{p} m_2 + \frac{kq^2}{p^2} m_1^2 \\ M_S(t) = \left[\frac{p}{1 - qM_X(t)}\right]^k\end{gathered}$$

与前述分布类似，复合负二项分布的3阶中心矩可以通过$S$的矩母函数得出，连续求三阶导，结果为：

$$\frac{\mathrm{d}}{\mathrm{d}t} \ln M_S(t) = \frac{\mathrm{d}}{\mathrm{d}t} \left\{ k \ln{p} - k \ln[1-qM_X(t)] \right\} = \frac{kq}{1 - q M_X(t)} M_X^{'}(t)$$
$$\frac{\mathrm{d}^2}{\mathrm{d}t^2} \ln M_S(t) = \frac{kq^2}{[1 - q M_X(t)]^2} [M_X^{'}(t)]^2 + \frac{kq}{1 - q M_X(t)} M_X^{''}(t)$$
$$\begin{aligned} \frac{\mathrm{d}^3}{\mathrm{d}t^3} \ln M_S(t) &= \frac{3kq^2}{[1 - q M_X(t)]^2} M_X^{'}(t) M_X^{''}(t) \\ &~~~+ \frac{2kq^3}{[1 - q M_X(t)]^3} [M_X^{'}(t)]^3 + \frac{kq}{1 - q M_X(t)} M_X^{'''}(t) \end{aligned}$$

在三阶导函数中令$t = 0$，得到3阶中心矩为：

$$\mathbb{E}[(S - \mathbb{E}S)^3] = \frac{3kq^2m_1m_2}{p^2} + \frac{2kq^3m_1^3}{p^3} + \frac{kqm_3}{p}$$

由于参数$k$和$p$均为正，且$X$的各阶原点矩也为正，因此可知复合负二项分布是右偏的。偏态系数可以根据定义直接计算。

def panjer_poisson(p, lam):
    """Panjer递推（泊松分布）"""
    cumul = np.exp(-lam * np.sum(p))
    f = [cumul]
    s = 0
    while cumul <= 0.99999999:
        s += 1
        m = min(s, len(p))
        last = lam / s * np.sum(np.arange(1, m+1) * p[:m] * f[s-m:s][::-1])
        f.append(last)
        cumul += last
    return np.array(f)

2.2 短期个体风险模型

2.2.1 基本模型假设

定义 2.2（短期个体风险模型）

短期个体风险模型主要研究由固定数量的风险（个体）构成的组合。假设风险个体在保险期间内的数量不变，记为$n$，个体的索赔情况相互独立（即索赔次数和索赔金额互不影响），且不同个体的索赔金额可以来自不同的分布；用$S$表示风险组合的总索赔金额，则有：

$$S = Y_1 + Y_2 + \cdots + Y_n$$

该框架称为个体风险模型（individual risk model）。其中，$ Y_i $表示第$ i $个个体的索赔金额。组合中的某些个体在保险期间内可能不发生索赔，因此$ \{Y_i\}_{i=1}^n $中可能有部分观察值等于0。

短期个体风险模型包含两个重要的假设：

第一，每个个体至多发生一次索赔，即第$i$个个体的索赔次数取值仅限于0或1；

第二，记第$i$个个体发生索赔的概率为$q_i$，每个个体是否发生索赔相互独立。

由于该模型中的每个个体最多发生一次索赔，故在寿险精算中具有比较广泛的应用；而对于非寿险精算，则可以将个体的多次赔付合并为一次赔付来分析。

个体风险模型与聚合风险模型相比有三个主要区别，在实际应用中需要注意：

其一，个体风险模型需要指定组合中的个体数量；而在聚合风险模型中则不必，甚至无需假设其在保险期间内保持不变。

其二，个体风险模型主要研究单个个体的索赔情况，每个个体的索赔次数受到约束；而聚合风险模型则是将所有风险汇总，组合的索赔次数$N$可以在正整数上取值。

其三，在个体风险模型中，每个个体的索赔金额可以来自不同分布；而对于聚合风险模型，则假设每次索赔的金额具有相同分布。

2.2.2 $S$的分布和特征

首先，记$N_i=\{0,1\}$表示第$i$个个体的索赔次数；若第$i$个个体有发生索赔，则用随机变量$X_i$表示其索赔金额，记$\mu_i,\ \sigma_i^2$和$F_i(x)$分别代表$X_i$的期望、方差和分布函数。

在个体风险模型中，每个个体的索赔次数服从伯努利分布（二项分布），即$N_i \sim \text{B}(1,q_i)$。易知$Y_i$服从复合二项分布，期望和方差可以分别表示为：

$$\begin{gathered}\mathbb{E}(Y_i) = q_i \mu_i \\ \text{Var}(Y_i) = q_i \sigma_i^2 + q_i (1 - q_i) \mu_i^2\end{gathered}$$

总索赔金额$S$是$n$个独立复合二项随机变量的和，根据期望和方差的性质，易计算$S$的期望和方差为：

$$\begin{gathered}\mathbb{E}(S) = \mathbb{E}\left( \sum_{i=1}^n Y_i \right) = \sum_{i=1}^n \mathbb{E}(Y_i) = \sum_{i=1}^n q_i \mu_i \\ \text{Var}(S) = \text{Var}\left( \sum_{i=1}^n Y_i \right) = \sum_{i=1}^n \text{Var}(Y_i) = \sum_{i=1}^n \left[ q_i \sigma_i^2 + q_i (1 - q_i) \mu_i^2 \right]\end{gathered}$$

在个体风险模型中，$S$的分布函数的计算通常较为复杂，只有在某些特定场景下才易于推导。例如，当$\{Y_i\}_{i=1}^n$为独立同分布的随机变量时（即对于所有个体，$q_i,\mu_i,\sigma_i^2$的值均相同，索赔金额$X_i$具有相同的分布$F(x)$），该模型等价于聚合风险模型，$S$的期望和方差可以简化为：

$$\begin{gathered}\mathbb{E}(S) = nq\mu \\ \text{Var}(S) = nq\sigma^2 + nq(1 - q)\mu^2\end{gathered}$$
例 2.6

假设有1000份一年期保单构成的组合，其中每份保单发生的索赔概率为0.004，且索赔金额均服从伽马分布$\text{Ga}(\alpha=5,\lambda=0.002)$。计算该组合的总索赔金额$S$的期望和方差。

由伽马分布的性质，易知：

$$\mu_j = \frac{5}{0.002} = 2500,    \sigma_j^2 = \frac{5}{0.002^2} = 1250000$$

因此，$S$的期望和方差分别为：

$$\begin{gathered}\mathbb{E}(S) = n q \mu = 1000 \times 0.004 \times 2500 = 10000 \\ \text{Var}(S) = nq \sigma^2 + nq(1 - q) \mu^2 = (5468)^2\end{gathered}$$

def panjer_poisson(p, lam):
    """Panjer递推（泊松分布）"""
    cumul = np.exp(-lam * np.sum(p))
    f = [cumul]
    s = 0
    while cumul <= 0.99999999:
        s += 1
        m = min(s, len(p))
        last = lam / s * np.sum(np.arange(1, m+1) * p[:m] * f[s-m:s][::-1])
        f.append(last)
        cumul += last
    return np.array(f)

2.3 参数不确定性的影响

本章目前介绍了当索赔次数及每次索赔的金额服从特定分布时，累积索赔金额的分布特征。其中，索赔次数分布和每次索赔金额分布的参数均假定为固定常数；尽管在实际应用中这些参数通常是未知的，但可以采用适当的方法进行估计。

本节将通过几个例题对前述模型进行拓展，以允许分布的参数为不确定/可变时的情况。多数例子主要考虑索赔次数分布的不确定性，并假定索赔次数服从泊松分布。典型场景可以表述为：考虑一个由$n$份独立保单构成的组合，$S_j$表示第$j$份保单的累积索赔金额，服从复合泊松分布；每份保单的索赔次数服从泊松分布$\text{Poi}(\lambda_j)$，每次索赔金额（索赔强度）的分布函数记为$F(x)$，为简化起见，假设所有保单的索赔强度具有相同分布。通常假定索赔强度的分布函数$F(x)$已知，而不同保单的索赔次数分布的参数值$\lambda_j$未知，是一组来自已知分布的独立同分布随机变量。这意味着每份保单的索赔次数分布是不确定的，但可以对泊松参数的取值概率进行描述。

例 2.8

假设组合中每份保单的索赔次数服从泊松分布，泊松参数值$\lambda_j$未知，但有相同可能性为0.1或0.3：

(1) 对于任一保单，计算其累积索赔金额$S_j$的期望和方差（以索赔强度分布的原点矩$m_1,m_2$表示）。

(2) 计算整个保单组合的总索赔金额$\sum_{j=1}^{n} S_j$的期望和方差（以$n,m_1,m_2$表示）。

该例子在非寿险精算实务中十分常见。以汽车保险为例，根据投保人的不同驾驶风险水平，通常可以分为“好”司机和“差”司机两类。保险公司依据其经验，认为两类司机在出险后的索赔金额具有相同分布，但“好”司机的索赔次数（平均每年0.1次）比“差”司机（平均每年0.3次）更低。假设保单组合中两类司机的比例相等，但任意抽取一名投保人$j$，保险公司不知道其到底属于“好”司机还是“差”司机。

根据题目条件，已知$\text{P}(\lambda_j=0.1)=\text{P}(\lambda_j=0.3)=0.5$，则：

$$\mathbb{E}(\lambda_j)=0.2,   \text{Var}(\lambda_j)=0.01$$

(1) $S_j$的矩可以通过$\lambda_j$的条件值计算。由于$S_j | \lambda_j$服从复合泊松分布，则有：

$$\begin{aligned}\mathbb{E}(S_j) &= \mathbb{E}[\mathbb{E}(S_j | \lambda_j)] = \mathbb{E}(\lambda_j m_1) = 0.2 m_1 \\ \text{Var}(S_j) &= \mathbb{E}[\text{Var}(S_j|\lambda_j)] + \text{Var}[\mathbb{E}(S_j|\lambda_j)] \\ &= \mathbb{E}(\lambda_j m_2) + \text{Var}(\lambda_j m_1) = 0.2 m_2 + 0.01 m_1^2\end{aligned}$$

(2) 由于各$\{S_j\}_{j=1}^n$相互独立，且具有相同的边际分布，因此：

$$\begin{gathered}\mathbb{E}(\sum_{j=1}^{n} S_j) = n E(S_j) = 0.2 n m_1 \\ \text{Var}(\sum_{j=1}^{n} S_j) = n \text{Var}(S_j) = 0.2 n m_2 + 0.01 n m_1^2\end{gathered}$$

现在考虑另一个例子。与之前类似，假设有一个包含$n$份保单的组合，每份保单的累积索赔金额服从复合泊松分布，索赔次数的泊松分布参数值为$\lambda$，每次索赔金额的分布函数为$F(x)$。此处仍假定每次索赔金额的分布$F(x)$是确定的；组合中所有保单的索赔次数都具有相同的泊松参数，且对于任意固定的$\lambda$，不同保单的累积索赔金额$S_i$相互独立。参数$\lambda$可能随时间而变化，因此可视为随机变量进行建模。

例 2.9

假设泊松参数$\lambda$以相同概率取值为0.1或0.3：

(1) 对于任一保单，计算其累积索赔金额$S_j$的期望和方差（以索赔强度分布的原点矩$m_1,m_2$表示）。

(2) 计算整个保单组合的总索赔金额$\sum_{j=1}^{n} S_j$的期望和方差（以$n,m_1,m_2$表示）。

已知$\text{P}(\lambda=0.1) = \text{P}(\lambda=0.3) = 0.5$，则$\mathbb{E}(\lambda)=0.2,\ \text{Var}(\lambda)=0.01$。

(1) 应用重期望和重方差公式，有：

$$\begin{aligned}\mathbb{E}(S_j) &= \mathbb{E}[\mathbb{E}(S_j | \lambda)] = \mathbb{E}(\lambda m_1) = 0.2 m_1 \\ \text{Var}(S_j) &= \mathbb{E}[\text{Var}(S_j|\lambda)] + \text{Var}[\mathbb{E}(S_j|\lambda)] \\ &= \mathbb{E}(\lambda m_2) + \text{Var}(\lambda m_1) = 0.2 m_2 + 0.01 m_1^2\end{aligned}$$

(2) 在给定$\lambda$的条件下，随机变量$\{S_j|\lambda\}_{j=1}^n$独立同分布，因此：

$$\begin{aligned}\mathbb{E}(\sum_{j=1}^{n} S_j) &= \mathbb{E}[\mathbb{E}(\sum_{j=1}^{n} S_j | \lambda)] = \mathbb{E}[\sum_{j=1}^{n} \mathbb{E}(S_j | \lambda)] = 0.2 n m_1 \\ \text{Var}(\sum_{j=1}^{n} S_j) &= \mathbb{E}[\text{Var}(\sum_{j=1}^{n} S_j | \lambda)] + \text{Var}[\mathbb{E}(\sum_{j=1}^{n} S_j | \lambda)] \\ &= \mathbb{E}(n\lambda m_2) + \text{Var}(n\lambda m_1) = 0.2 n m_2 + 0.01 n^2 m_1^2\end{aligned}$$

注意本例与前例~中，累积索赔金额独立性的差异。

例 2.10

一家保险公司使用以下假设来模拟家财险保单遭受风暴索赔的情况：

  • 假设每年发生的风暴数 $K$ 服从参数为 $\lambda$ 的泊松分布；
  • 第 $i$ 次风暴导致的索赔次数 $N_i\ (i=1,2,\cdots,K)$ 服从参数为 $\Theta_i$ 的泊松分布；
  • 参数 $\{\Theta_i\}_{i=1}^n$ 假设为一组独立同分布的随机变量，且$\mathbb{E}(\Theta_i)=n$，$\text{Var}(\Theta_i)=s_1^2$；
  • 记第$i$次风暴导致的第$j$个索赔的金额为$X_{ij}\ (j=1,2,\cdots,N_i)$，则$X_{ij}$服从对数正态分布$\text{LN}(\mu_i,\sigma^2)$，其中$\sigma^2$为已知；
  • 第$i$次风暴导致的平均索赔金额$\Lambda_i = \exp(\mu_i + 0.5\sigma^2)$假设为独立同分布的随机变量，均值为$p$、方差为$s_2^2$；
  • $\Theta_i$ 和 $\Lambda_i$ 独立。

(1) 证明 $\mathbb{E}(X_{ij}) = p$ 和 $\text{Var}(X_{ij}) = \exp(\sigma^2)(p^2 + s_2^2) - p^2$。

(2) 记 $S_i$ 表示第 $i$ 次风暴导致的累积索赔金额，$S_i | \{\Theta_i, \Lambda_i\}$是一个复合泊松随机变量。证明 $\mathbb{E}(S_i) = np$ 以及 $\text{Var}(S_i) = (p^2 + s_2^2)(n^2 + s_1^2 + n\exp(\sigma^2)) - n^2p^2 $。

(3) 计算一年内所有风暴导致的总索赔金额的期望和方差。

(1) 对于$X_{ij}$，应用重期望和重方差公式，有

$$\begin{aligned}\mathbb{E}(X_{ij}) &= \mathbb{E}[\mathbb{E}(X_{ij} | \Lambda_i)] = \mathbb{E}(\Lambda_i) = p \\ \text{Var}(X_{ij}) &= \mathbb{E}[\text{Var}(X_{ij} | \Lambda_i)] + \text{Var}[\mathbb{E}(X_{ij} | \Lambda_i)] \\ &= \mathbb{E}\{\Lambda_i^2 [\exp(\sigma^2) - 1]\} + \text{Var}(\Lambda_i) \\ &= (p^2 + s_2^2)[\exp(\sigma^2) - 1] + s_2^2 \\ &= (p^2 + s_2^2) \exp(\sigma^2) - p^2\end{aligned}$$

其中，$\text{Var}(X_{ij} | \Lambda_i) = \Lambda_i^2 [\exp(\sigma^2) - 1]$。

(2) 首先，由重期望公式，并注意$\Theta_i$和$\Lambda_i$的独立性，有：

$$\begin{aligned}\mathbb{E}(S_i) &= \mathbb{E}[\mathbb{E}(S_i | \Theta_i, \Lambda_i)] = \mathbb{E}(\Theta_i \Lambda_i) \\ &= \mathbb{E}(\Theta_i) \cdot \mathbb{E}(\Lambda_i) = np\end{aligned}$$

为应用重方差公式，分别计算条件方差的期望和条件期望的方差。其中，条件方差的结果为：

$$\text{Var}(S_i | \Theta_i, \Lambda_i) = \Theta_i \cdot \mathbb{E}(X_{ij}^2 | \Lambda_i) = \Theta_i \Lambda_i^2 \exp(\sigma^2)$$

由于$\Theta_i$和$\Lambda_i$独立，对上式求期望有：

$$\mathbb{E}[\text{Var}(S_i | \Theta_i, \Lambda_i)] = \mathbb{E}(\Theta_i) \mathbb{E}(\Lambda_i^2) \exp(\sigma^2) = n(p^2+s_2^2)\exp(\sigma^2)$$

同时，条件期望的方差可计算为：

$$\begin{aligned}\text{Var}[\mathbb{E}(S_i | \Theta_i, \Lambda_i)] &= \text{Var}(\Theta_i \Lambda_i) \\ &= \mathbb{E}(\Theta_i^2 \Lambda_i^2) - [\mathbb{E}(\Theta_i\Lambda_i)]^2 \\ &= \mathbb{E}(\Theta_i^2)\mathbb{E}(\Lambda_i^2) - \mathbb{E}^2(\Theta_i)\mathbb{E}^2(\Lambda_i) \\ &= (n^2 + s_1^2)(p^2 + s_2^2) - n^2 p^2\end{aligned}$$

加总得到$S_i$的总方差为：

$$\text{Var}(S_i) = (n^2 + s_1^2)(p^2 + s_2^2) - n^2 p^2 + n(p^2 + s_2^2) \exp(\sigma^2)$$

(3) 令 $R$ 表示一年内所有风暴的总索赔金额，则其可以表示为：$R = \sum_{i=1}^K S_i$。其中 $K \sim \text{Poi}(\lambda)$，且各 $\{S_i\}_{i=1}^n$ 独立同分布。

已知 $R$ 服从复合泊松分布，则根据2.1.3节的性质，有：

$$\begin{aligned}\mathbb{E}(R) &= \lambda \mathbb{E}(S_i) = \lambda np \\ \text{Var}(R) &= \lambda \mathbb{E}(S_i^2) = \lambda [\text{Var}(S_i) + \mathbb{E}^2(S_i)] \\ &= \lambda (p^2 + s_2^2) [n^2 + s_1^2 + n\exp(\sigma^2)]\end{aligned}$$
例 2.11

某保险公司每年发行一系列家财险保单，每张保单的年保费为80元。假设每张保单的年度累积索赔金额服从复合泊松分布：泊松参数为0.4，每次索赔的金额服从伽马分布$\text{Ga}(\alpha,\lambda)$。每次理赔支出的费用是一个随机变量，服从$[50,b]$区间上的均匀分布，理赔费用独立于相关索赔的金额。记随机变量$S$表示保险公司对该保单组合的年度支出（包括索赔金额和理赔费用），可以假设$S$近似服从正态分布。

(1) 假定 $\alpha = 1,~\lambda = 0.01,~b = 100$。试计算保险公司需售出多少份保单，才能以 99% 概率确保该年度家财险业务的收入不低于总支出。

(2) 假设参数 $\alpha,~\lambda,~b$ 的值不确定，但服从下列区间中的均匀分布：$0.95 \leq \alpha \leq 1.05,~0.009 \leq \lambda \leq 0.011,~90 \leq b \leq 110$。保险公司可以考虑 $\alpha,~\lambda,~b$ 的组合，通过计算最坏情况下的索赔和支出来评估风险。为以 99% 的概率确保该年度家财险业务的收入不低于总支出，计算保险公司应出售的保单数量。

记 $N$ 表示保单组合在一年内的索赔次数，$X_i$ 表示第 $i$ 次索赔的金额，$Y_i$ 表示该次索赔的相关理赔费用。令 $n$ 表示保险公司售出保单的数量，则索赔次数 $N$ 服从泊松分布，参数为 $0.4n$。

保险公司的年度支出 $S$ 可表示为：$S = \sum_{i=1}^{N} (X_i + Y_i)$，其中$\{X_i+Y_i\}_{i=1}^N$是独立同分布的随机变量序列，与索赔次数$N$无关。因此可认为$S$服从复合泊松分布，其中 $(X_i + Y_i)$ 表示“每次索赔的金额”。

根据复合泊松分布的性质，$S$ 的期望和方差具有如下计算公式：

$$\begin{gathered}\mathbb{E}(S) = 0.4n\mathbb{E}(X_i + Y_i) \\ \text{Var}(S) = 0.4n\mathbb{E}[(X_i + Y_i)^2] = 0.4n[\mathbb{E}(X_i^2) + 2\mathbb{E}(X_iY_i) + \mathbb{E}(Y_i^2)]\end{gathered}$$

将上述结果用参数 $\alpha,~\lambda,~b$ 表示，则有：

$$\begin{gathered}\mathbb{E}(X_i) = \frac{\alpha}{\lambda},   \mathbb{E}(Y_i) = \frac{b + 50}{2} \\ \mathbb{E}(X_i^2) = \frac{\alpha(\alpha + 1)}{\lambda^2},   \mathbb{E}(Y_i^2) = \frac{b^2 + 50b + 2500}{3} \\ \mathbb{E}(X_iY_i) = \mathbb{E}(X_i)\mathbb{E}(Y_i)\end{gathered}$$

本例的关键是注意到 $X_i$ 和 $Y_i$ 独立。

(1) 将 $\alpha = 1,~\lambda = 0.01,~b = 100$ 代入上述公式，得到：

$$\mathbb{E}(S) = 70n,    \text{Var}(S) = 127.80^2n$$

由题目条件可知，$S$ 近似服从均值为$70n$、标准差为$127.80\sqrt{n}$的正态分布。保险公司的年度保费收入为$80n$。

因此，题目等价于求使概率不等式$\text{P}(S < 80n) \geq 0.99$成立的$n$的最小值。将$S$标准化为正态分布$Z$，得到：

$$\text{P}\left(Z < \frac{80n - 70n}{127.80\sqrt{n}}\right) \geq 0.99$$

标准正态分布的 99% 分位数为 2.326，因此 $n$ 应满足：

$$\frac{80n - 70n}{127.80\sqrt{n}} \geq 2.326$$

解方程得到 $n \geq 883.7$，因此至少需售出884份保单。

(2) 记$\mu=0.4\mathbb{E}(X_i+Y_i),~\sigma^2=0.4\mathbb{E}[(X_i+Y_i)^2]$，则$S$的期望和方差分别为$\mathbb{E}(S)=n\mu,~\text{Var}(S)=n\sigma^2$。

根据第(1)问的结果，$n$应满足以下条件：

$$\frac{(80 - \mu)\sqrt{n}}{\sigma} \geq 2.326,   n \geq (\frac{2.326 \cdot \sigma}{80-\mu})^2$$

注意$\mu<80$。对于保险公司而言，最坏情况是$\alpha,~\lambda,~b$在参数空间中的取值使$n$极大化，此时$\mu$和$\sigma$也达到最大值。

根据$X_i$和$Y_i$的矩的公式，当$\alpha$和$b$尽可能大、而$\lambda$尽可能小时，$\mu$和$\sigma^2$取到最大值。此时有：

$$\begin{gathered}\alpha = 1.05,    \lambda = 0.009,    b = 110 \\ \mu = 78.67,   \sigma = 144.14\end{gathered}$$

因此对保险公司而言，$n$ 必须至少为 63546，才能以 99% 的概率确保该年度家财险业务的收入不低于总支出。

2.4 真题练习

1. 假设 $N$ 是一年内某个风险组合的索赔次数；每次索赔的金额 $X_1, X_2, \cdots, X_N$ 是独立同分布的随机变量，与 $N$ 无关。记$S$表示一年内的累积索赔金额。

(1) 推导 $\mathbb{E}(S)$ 和 $\text{Var}(S)$，并表示为 $X_1$ 的期望和方差的形式。

(2) 根据 $X_1$ 和 $N$ 的矩母函数 $M_X(t)$ 和 $M_N(t)$，推导 $S$ 矩母函数 $M_S(t)$ 的表达式。

(3) 如果 $N$ 服从均值为 $\lambda$ 的泊松分布，证明：$M_S(t) = \exp[\lambda(M_X(t) - 1)]$。

(4) 如果 $N$ 服从参数为 $m$ 和 $q$ 的二项分布，试用 $m$、$q$ 和 $M_X(t)$ 表达 $S$ 的矩母函数。 【解答】由题可知，$S=X_1+X_2+\cdots+X_N$。

(1) 应用重期望公式，有：

$$\begin{aligned}\mathbb{E}(S) &= \mathbb{E}[\mathbb{E}(S|N)] = \mathbb{E}[\mathbb{E}(X_1 + X_2 + \dots + X_N | N)] \\ &= \mathbb{E}[\mathbb{E}(X_1|N) + \mathbb{E}(X_2|N) + \dots + \mathbb{E}(X_N|N)] \\ &= \mathbb{E}[N \cdot \mathbb{E}(X_1)] = \mathbb{E}[N]\mathbb{E}[X_1]\end{aligned}$$

类似地，应用重方差公式，有：

$$\begin{aligned}\text{Var}(S) &= \text{Var}[\mathbb{E}(S | N)] + \mathbb{E}[\text{Var}(S | N)] \\ &= \text{Var}[N \mathbb{E}(X_1)] + \mathbb{E}[N \text{Var}(X_1)] \\ &= [\mathbb{E}(X_1)]^2 \text{Var}(N) + \mathbb{E}(N) \text{Var}(X_1)\end{aligned}$$

(2) $S$ 的矩母函数的定义为：$M_S(t) = \mathbb{E}(e^{tS})$。

仍然使用重期望公式，并注意$X_i$间的独立性，有：

$$\begin{aligned}M_S(t) &= \mathbb{E}[\mathbb{E}(e^{tS} | N)] = \mathbb{E} \left\{\mathbb{E} [e^{t(X_1+X_2+\cdots+X_N)} | N ] \right\} \\ &= \mathbb{E} \left\{ \mathbb{E}[(e^{tX_1}|N)(e^{tX_2}|N)\cdots (e^{tX_N}|N) ] \right\} = \mathbb{E} \left\{ [\mathbb{E}(e^{tX_1})]^N \right\} \\ &= \mathbb{E}\left\{ [M_X(t)]^N \right\} = \mathbb{E} \left\{ \exp[N \cdot \ln M_X(t)] \right\} \\ &= M_N(\ln M_X(t))\end{aligned}$$

(3) 根据泊松分布的性质，可知若 $N \sim \text{Poi}(\lambda)$，则$M_N(t) = \exp\left[\lambda(e^t - 1)\right]$。

将第(2)问的结果代入，可得此时$S$的矩母函数为：

$$M_S(t) = \exp\left\{\lambda\left[M_X(t) - 1\right]\right\}$$

(4) 根据二项分布的性质，可知若 $N \sim \text{B}(m, q)$, 则$M_N(t) = (1 - q + q e^t)^m$。

将第(2)问的结果代入，可得此时$S$的矩母函数为：

$$M_S(t) = [1 - q + q M_X(t)]^m$$

2. 某个保单组合由500个独立风险组成。对于第$i$个风险，假设其发生索赔的概率为$q_i$，各风险的索赔金额（在索赔发生时）具有相同的期望$\mu$和方差$\sigma^2$。定义$T=Y_1+Y_2+\cdots+Y_{500}$表示保单组合在一年内的总索赔金额，$M_T(t)$为其矩母函数。

(1) 确定 $T$ 的期望和方差。

若第 $i$ 个风险在一年内的累积索赔金额近似服从复合泊松分布，泊松参数为 $q_i$，每次索赔金额的期望为 $\mu$、方差为 $\sigma^2$，且不受个体差异的影响。记 $\tilde{T}$ 表示在该模型中保单组合一年内的总索赔金额。

(2) 确定 $\tilde{T}$ 的期望和方差，并将其与第(1)问的答案进行比较。

假设对所有$i$，$q_i = 0.02$；且每次索赔发生时，索赔金额为一个固定的常数值 $\mu$。

(3) 推导 $T$ 的矩母函数，并证明 $T$ 满足复合二项分布。

(4) 确定 $\tilde{T}$ 的矩母函数，并证明 $\tilde{T}$ 服从复合泊松分布。 【解答】(1) 根据个体风险模型的性质，$T$的期望和方差分别为：

$$\begin{aligned}\mathbb{E}(T) &= \mathbb{E}(Y_1) + \mathbb{E}(Y_2) + \dots + \mathbb{E}(Y_{500}) = \mu \sum_{i=1}^{500} q_i \\ \text{Var}(T) &= \text{Var}(Y_1) + \text{Var}(Y_2) + \cdots + \text{Var}(Y_{500}) \\ &= \sigma^2 \sum_{i=1}^{500}q_i + \mu^2 \sum_{i=1}^{500}q_i(1-q_i)\end{aligned}$$

(2) 记 $C_i$ 表示第$i$个风险在一年内的累积索赔金额。根据题目，有$C_i = X_1 + X_2 + \dots + X_{N_i}$，其中$N_i \sim \text{Poi}(q_i),~\mathbb{E}(X_j) = \mu,~\text{Var}(X_j) = \sigma^2$。

保单组合在一年内的总索赔金额等于$\tilde{T} = C_1 + C_2 + \dots + C_{500}$。且根据复合泊松分布的性质，可知$C_i$的期望和方差分别为：$\mathbb{E}(C) = q_i \mu,\ \text{Var}(C) = q_i (\mu^2 + \sigma^2)$。

对 $\tilde{T}$ 求期望和方差，结果等于：

$$\begin{aligned}\mathbb{E}(\tilde{T}) &= \mathbb{E}(C_1 + C_2 + \cdot + C_{500}) = \mu \sum_{i=1}^{500} q_i \\ \text{Var}(\tilde{T}) &= \text{Var}(C_1 + C_2 + \cdot + C_{500}) = (\mu^2 + \sigma^2) \sum_{i=1}^{500} q_i\end{aligned}$$

因此，$\tilde{T}$ 与 $T$ 的期望相同，但$\tilde{T}$的方差大于$T$的方差（$T$的方差中额外包含一项$-\mu^2 \sum_{i=1}^{500}q_i^2$）。

(3) 根据定义，$T$的矩母函数可以表示为：

$$M_T(t) = \mathbb{E}(e^{tT}) = \mathbb{E}[e^{t(Y_1 + Y_2 + \dots + Y_{500})}] = M_{Y_1}(t) M_{Y_2}(t) \cdots M_{Y_{500}}(t)$$

根据题目给出的信息，第$i$个风险的索赔金额$Y_i$是一个二值随机变量：以0.98的概率取值为0，以0.02的概率取值为$\mu$。因此，可以按定义计算其矩母函数：

$$M_{Y_i}(t) = \mathbb{E}(e^{tY_i}) = e^0 \times 0.98 + e^{t \mu} \times 0.02 = 0.98 + 0.02 e^{t \mu}$$

将上式代入 $T$ 的矩母函数表达式，得到：

$$M_T(t) = (0.98 + 0.02 e^{t \mu})^{500}$$

恰好为复合二项分布的矩母函数形式，其中二项分布参数为 $n=500,~p=0.02$，每次索赔的金额为常数值$\mu$。

(4) 根据定义，$\tilde{T}$的矩母函数可以表示为：

$$M_{\tilde{T}}(t) = \mathbb{E}(e^{t \tilde{T}}) = \mathbb{E}[e^{t(C_1 + C_2 + \dots + C_{500})}] = M_{C_1}(t) M_{C_2}(t) \cdots M_{C_{500}}(t)$$

由于 $C_i$ 服从复合泊松分布，其矩母函数具有如下表达式：

$$M_{C_i}(t) = \exp\{q_i[M_X(t) - 1]\} = \exp\{0.02[M_X(t) - 1]\}$$

根据题目给出的信息，$X$ 是取值于 $\mu$ 的常数值，即：$M_X(t) = e^{t \mu}$。因此，$C_i$的矩母函数为：$M_{C_i}(t) = \exp[0.02(e^{t \mu} - 1)]$。

将其代入$\tilde{T}$的矩母函数表达式，得到：

$$\begin{aligned}M_{\tilde{T}}(t) &= \exp[0.02(e^{t \mu} - 1)] \cdot \exp[0.02(e^{t \mu} - 1)] \cdots \exp[0.02(e^{t \mu} - 1)] \\ &= \exp[10(e^{t \mu} - 1)]\end{aligned}$$

恰好为复合泊松分布的矩母函数形式，其中泊松分布参数为$\lambda=10$，每次索赔的金额为常数值$\mu$。

3. 一家公司正在分析其工厂每年发生的事故次数。该公司认为，每年的事故数量 $N$ 服从参数为 $p=0.8$ 的几何分布，即：$ \text{P}(N = n) = 0.8 \times 0.2^n,\ n = 0, 1, 2, \cdots$。对于每起事故，设受伤员工的数量 $Y = 1+X $，其中 $X$ 服从参数为 $\lambda=2.2$ 的泊松分布。

该公司目前投保了一份保单，为事故中的每名受伤员工提供1000元赔付。保险公司约定每次事故最多负责3名员工的赔付，但对一年内索赔的次数没有限制。

记$S$表示保险公司在一年内赔付的受伤员工数量，计算$S$的期望和方差。

【解答】根据题意，$S$可以表示为$S = Z_1 + Z_2 + \dots + Z_N$，其中 $N$ 为每年的事故数量，$Z_i$ 为每次事故中保险公司负责赔付的受伤员工数量，则：

$$Z_i = \left\{ \begin{aligned} &Y_i, & Y_i < 3 \\ &3, & Y_i \geq 3 \end{aligned} \right.$$

由于$Y_i =1+ X_i,\ X_i \sim \text{Poi}(2.2)$，可以计算$Z_i$的概率分布为：

$$\begin{gathered}\text{P}(Z_i = 1)=\text{P}(Y_i = 1)=\text{P}(X_i = 0)= e^{-2.2} \\ \text{P}(Z_i = 2)=\text{P}(Y_i = 2)=\text{P}(X_i = 1) = 2.2e^{-2.2} \\ \text{P}(Z_i = 3)=P(Y_i \geq 3) = 1 - 3.2e^{-2.2}\end{gathered}$$

因此，可以计算 $Z_i$ 的均值和方差为：

$$\begin{gathered}\mathbb{E}(Z_i) = e^{-2.2} + 4.4e^{-2.2} + 3(1 - 3.2e^{-2.2}) = 3 - 4.2e^{-2.2} = 2.53463 \\ \mathbb{E}(Z_i^2) = e^{-2.2} + 8.8e^{-2.2} + 9(1 - 3.2e^{-2.2}) = 9 - 19e^{-2.2} = 6.89474 \\ \text{Var}(Z_i) = \mathbb{E}(Z_i^2) - \mathbb{E}^2(Z_i) = 6.8934 - 2.53463^2 = 0.4704\end{gathered}$$

对于事故数量$N$，易知其分布等价于$\text{NB}(k=1,~p=0.8)$，即负二项分布。则$N$的期望和方差为：

$$\begin{gathered}\mathbb{E}(N) = \frac{kq}{p} = \frac{0.2}{0.8} = 0.25 \\ \text{Var}(N) = \frac{kq}{p^2} = \frac{0.2}{0.8^2} = 0.3125\end{gathered}$$

$S$服从复合负二项分布，索赔次数$N \sim \text{NB}(k=1,~p=0.8)$，每次索赔的金额为独立同分布的随机变量$\{Z_i\}_{i=1}^N$，且$Z_i$的期望和方差为已知。因此，根据复合分布的性质，$S$的期望和方差可计算为：

$$\begin{aligned}\mathbb{E}(S) &= \mathbb{E}(N) \mathbb{E}(Z_i) = 0.25 \times 2.53463 = 0.63366 \\ \text{Var}(S) &= \mathbb{E}(N)\text{Var}(Z_i) + \text{Var}(N) \mathbb{E}^2(Z_i) \\ &= 0.25 \times 0.4704 + 0.3125 \times 2.53463^2 = 2.12521\end{aligned}$$

对于保险公司在一年内支付的总索赔金额 $1000S$，可应用期望和方差的性质，直接计算得：

$$\begin{gathered}\mathbb{E}(1000S) = 1000 \mathbb{E}(S) = 634 \\ \text{Var}(1000S) = 1000^2 \text{Var}(S) = 2125000\end{gathered}$$
