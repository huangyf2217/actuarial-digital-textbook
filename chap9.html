9.1 风险度量

在金融和保险领域，风险度量（risk measures）是评估和管理损失不确定性的重要工具。常见的数字特征如期望和方差等，通常仅关注风险的整体分布规律；然而，对于现实世界中发生概率较低、但对企业经营管理和财务稳定性具有重大潜在威胁的“黑天鹅”事件，如金融危机、流行病和巨灾损失等，其关注往往不足。特别从经营风险的保险公司的角度出发，更需要准确量化这些极端事件在风险尾部的行为，从而制定合理的资本要求，确保充足的流动性，增强抵御市场危机的能力。

风险度量是将风险损失随机变量$X$转化为一个实数值$\rho(X)$的过程，函数$\rho(\cdot)$表示对应的风险度量方法。本节将介绍两个常用的风险度量——在险价值（value at risk，VaR）和尾部在险价值（tail value at risk，TVaR）。

9.1.1 在险价值VaR

记$X$表示风险损失，对应的分布函数为$F(x)$。在险价值表示在给定置信水平（如95%或99%）下，该风险可能发生的最大损失（maximum possible loss）。定义为：

$$\text{VaR}_\alpha(X) = \inf \{x|F(x) \geq \alpha\}$$

易知$\text{P}(X>\text{VaR}_\alpha(X)) \leq 1-\alpha$，故在险价值还可理解为，风险损失超过$\text{VaR}_\alpha(X)$的概率不高于$1-\alpha$。若保险公司按照该要求储备资本，则可将破产概率控制在相对较低的水平。由于VaR通过一个单一数值描述了保单组合或资产组合的风险敞口，计算简单且易于理解，因此在风险管理和金融保险监管中被广泛使用。

从上式可知，VaR本质上等价于$X$在右侧尾部的分位数。故当$X$为连续型变量、且分布函数已知时，$\text{VaR}_\alpha(X) = F^{-1}(\alpha)$；当$X$为离散型变量时，$\text{VaR}_\alpha(X)$可取为使分布函数大于$\alpha$的最小值。然而，VaR只关注特定损失水平下的分位数，未考虑分位数以上的损失程度和分布特征，因此无法全面反映极端风险的情况。

例 9.1

假设风险损失$X$服从正态分布$\text{N}(\mu,\sigma^2)$，计算其对应的在险价值$\text{VaR}_\alpha(X)$。

根据定义$\text{P}(X \leq \text{VaR}_\alpha(X)) = \alpha$，有：

$$\begin{gathered}\text{P}(X \leq \text{VaR}_\alpha(X)) = \text{P}(\frac{X-\mu}{\sigma} \leq \frac{\text{VaR}_\alpha(X)-\mu}{\sigma}) = \alpha \\ \frac{\text{VaR}_\alpha(X)-\mu}{\sigma} = z_\alpha ~~\Rightarrow~~ \text{VaR}_\alpha(X) = \mu + \sigma z_\alpha\end{gathered}$$

其中，$z_\alpha$是标准正态分布的$100\alpha$%分位数。

9.1.2 一致性风险度量与TVaR

在介绍尾部在险价值TVaR前，首先引入一致性风险度量的概念。风险度量的一致性（coherence）是指风险度量方法$\rho(\cdot)$应满足的一组数学性质，以确保其适应金融与保险风险管理的实际需求。具体而言，包括以下四个方面：

(1) 单调性：若$X \leq Y$，则应有$\rho(X) \leq \rho(Y)$，即损失较低的风险，其风险度量结果也应较低，符合风险管理的直觉；

(2) 次可加性：$\rho(X+Y) \leq \rho(X) + \rho(Y)$，即组合$X+Y$的总风险不应大于各个单独部分的风险之和；

(3) 正齐次性：给定常数$c>0$，有$\rho(cX) = c \rho(X)$，即若损失$X$按照比例$c$进行缩放，则风险度量结果也应按照相同比例缩放；

(4) 平移不变性：给定常数$c$，有$\rho(X+c) = \rho(X)+c$，即在损失中增加一个确定性金额，风险度量结果也应线性增加。

当风险损失$X$和$Y$都服从椭圆分布（elliptically distributed）时，在险价值VaR满足一致性要求。但在其他情况下，VaR可能不满足次可加性，不是一致性风险度量。

例 9.2

假设$X$和$Y$都依赖于$[0,1]$区间上的均匀分布随机变量$U$，风险损失分别为：

$$X = \left\{ \begin{aligned} &1000, ~~\text{if}~~U < 0.04 \\ &0, ~~~~~~\text{if}~~U \geq 0.04 \end{aligned} \right.      Y = \left\{ \begin{aligned} &1000, ~~\text{if}~~U > 0.96 \\ &0, ~~~~~~\text{if}~~U \leq 0.96 \end{aligned} \right.$$

则易知$\text{VaR}_{0.95}(X) = \text{VaR}_{0.95}(Y) = 0$，但$\text{VaR}_{0.95}(X+Y) = 1000$。

尾部在险价值TVaR是一种改进的风险度量方法，特别适用于衡量极端损失的影响。该方法克服了VaR对尾部风险特征考虑不足、不满足次可加性的缺陷，使保险公司对极端风险具有更充分的认识。

TVaR表示风险损失高于$\text{VaR}_{\alpha}(X)$部分的平均值，定义为：

$$\text{TVaR}_{\alpha}(X) = \mathbb{E}[X|X \geq \text{VaR}_{\alpha}(X)]$$

对于连续型风险损失，上述定义等价于：

$$\begin{aligned} \text{TVaR}_{\alpha}(X) &= \frac{1}{1-F(\text{VaR}_{\alpha}(X))} \int_{\text{VaR}_{\alpha}(X)}^{\infty} xf(x) \mathrm{d}x \\ &= \frac{1}{1-\alpha} \int_{\text{VaR}_{\alpha}(X)}^{\infty} x \mathrm{d}F(x) \\ &\xlongequal{u=F(x)} \frac{1}{1-\alpha} \int_{\alpha}^1 \text{VaR}_{u}(X) \mathrm{d}u \end{aligned}$$

即$\text{VaR}_{u}(X)$在区间$u \in(\alpha,1)$上的平均数。因此，$\text{TVaR}_{\alpha}(X)$的结果大于$\text{VaR}_{\alpha}(X)$的结果，对尾部风险的评估更加保守。

定义$\frac{1}{1-\alpha} \int_{\alpha}^1 \text{VaR}_{u}(X) \mathrm{d}u$也被称为预期不足（expected shortfall，ES）。当$X$为连续型随机变量时，其与TVaR完全相等。

对于连续型风险损失，由定义易知$\text{TVaR}_{\alpha}(X)$还可以写为：

$$\begin{aligned} \text{TVaR}_{\alpha}(X) &= \text{VaR}_{\alpha}(X) + \mathbb{E}[X-\text{VaR}_{\alpha}(X) \mid X \geq \text{VaR}_{\alpha}(X)] \\ &= \text{VaR}_{\alpha}(X) + \frac{\mathbb{E}(X) - \mathbb{E}[X \wedge \text{VaR}_{\alpha}(X)]}{1-\alpha} \end{aligned}$$

即可以应用$X$的有限期望函数和平均超越函数计算。

尾部在险价值TVaR满足一致性风险度量的四个性质，且与VaR相比，能够更全面地衡量风险尾部的极端损失情况，是理论上更优秀的风险度量。在现代风险管理和监管框架（如Solvency II和巴塞尔协议III）中，TVaR已逐步取代VaR，成为衡量资本要求的重要工具。

def gev_nll(parms, x):
    mu = parms[0]
    sigma = np.exp(parms[1])
    xi = parms[2]
    z = (x - mu) / sigma
    if np.any(1 + xi * z <= 0):
        return 1e10
    if abs(xi) < 1e-8:
        t = np.exp(-z)
    else:
        t = (1 + xi * z) ** (-1/xi)
    return np.sum(np.log(sigma) + (1 + 1/xi) * np.log(t) + t)

result_gev = minimize(gev_nll, x0=[3, 0, 0.1], args=(maxima,),
                       method='Nelder-Mead')

xi_hat = result_gev.x[2]

/* 9.2 广义极值分布（GEV）                                                  */
/****************************************************************************/
proc iml;
  /* GEV分布函数:
     - ξ > 0 (Frechet): F(x) = exp(-(1+ξx/σ)^(-1/ξ)), 1+ξx/σ > 0
     - ξ = 0 (Gumbel): F(x) = exp(-exp(-x/σ))
     - ξ < 0 (Weibull): F(x) = exp(-(1+ξx/σ)^(-1/ξ)), 1+ξx/σ > 0
  */

  /* 绘制不同ξ的GEV密度 */
  x = do(-4, 4, 0.1)`;
  sigma = 1;

  /* ξ = 0.5 (Frechet) */
  xi1 = 0.5;
  f1 = j(nrow(x), 1, 0);
  do i = 1 to nrow(x);
    if 1 + xi1 * x[i] / sigma > 0 then do;
      t = (1 + xi1 * x[i] / sigma)##(-1/xi1);
      f1[i] = t##(xi1 + 1) * exp(-t) / sigma;
    end;
  end;

  /* ξ = 0 (Gumbel) */
  xi2 = 0;
  f2 = j(nrow(x), 1, 0);
  do i = 1 to nrow(x);
    t = exp(-x[i] / sigma);
    f2[i] = t * exp(-t) / sigma;
  end;

  /* ξ = -0.5 (Weibull) */
  xi3 = -0.5;
  f3 = j(nrow(x), 1, 0);
  do i = 1 to nrow(x);
    if 1 + xi3 * x[i] / sigma > 0 then do;
      t = (1 + xi3 * x[i] / sigma)##(-1/xi3);
      f3[i] = t##(xi3 + 1) * exp(-t) / sigma;
    end;
  end;

  create gev_data from x [colname={'x'}];
  append from x;
  close;
  create gev_f1 from f1 [colname={'f1'}];
  append from f1;
  close;
  create gev_f2 from f2 [colname={'f2'}];
  append from f2;
  close;
  create gev_f3 from f3 [colname={'f3'}];
    append from f3;
  close;
quit;

9.2 区块最大值模型

在金融领域中，极端损失事件虽然发生频率较低，却可能对公司造成重大打击。以2007年爆发的次贷危机为例，在这一过程中股票价格的波动程度远超以往常规水平。此类事件往往对应分布的“尾部”，一旦发生便意味着巨大损失。因此，准确描述损失分布在尾部的形态，刻画“低频高损”事件的分布特征，是风险管理的一个核心问题。

然而，由于极端事件的数据记录较少，对分布的尾部特征进行建模往往存在一定的挑战。一方面，参数估计过程往往会受到分布中心大量数据的主导，使得尾部特征难以准确体现。另一方面，传统的连续型分布假设如正态分布、伽马分布等，通常也无法有效描述金融损失数据的特性。此类数据在其分布的中心位置通常更加集中、陡峭，且尾部也存在一定厚度（即事件发生的概率无法被忽略），称为尖峰厚尾（leptokurtic）的特性；若仍采用传统分布假设建模，往往会低估尾部风险，从而制约极端事件的预测和管理。其他更复杂的特征还包括波动率随时间的不稳定变化（异方差性）等。

极值理论（extreme value theory，EVT）为刻画数据的尾部行为提供了一套有效的工具。其核心思想是：无论原始数据的分布形态如何，其尾部在样本量足够大时均会呈现特定的渐近性质。具体而言，极值理论给出了两项重要结论：

(1) 随着样本量增加，一组样本中的极大值将收敛于广义极值分布族（generalized extreme value distribution，GEV）；

(2) 随机变量超过某一预设阈值后的极端数据（称为“超越值”）将收敛于广义帕累托分布（generalized Pareto distribution，GPD）。

基于上述理论，精算师可以更准确地估计极端事件的发生概率和可能的损失规模。本节首先介绍区块最大值模型。

9.2.1 区块最大值的概念

假设有一组独立同分布的随机变量序列$\{X_1,X_2,\cdots\}$，分布函数为$F(x)$。按照顺序可将其划分成若干个等长度的区块，每个区块包含$n$个变量（数据点），例如：

  • 第1块：$X_1, X_2, \cdots, X_n$
  • 第2块：$X_{n+1}, X_{n+2}, \cdots, X_{2n}$
  • $~~~~~~~~~~~~~~~~~~~~~~~~\vdots$
  • 第$k$块：$X_{(k-1)n+1}, X_{(k-1)n+2}, \cdots, X_{kn}$
  • $~~~~~~~~~~~~~~~~~~~~~~~~\vdots$

其中，定义$M^k_n = \max(X_{(k-1)n+1}, X_{(k-1)n+2}, \cdots, X_{kn})$称为区块最大值（block maximum），是对应区块所包含的$n$个随机变量中的最大值；$n$称为区块长度。

通过上述划分，可以得到一组新的随机变量序列$\{M^1_n, M^2_n, \cdots\}$。由于各$X_i$独立同分布，则易知各$M^k_n$也独立同分布，且分布函数等于：

$$\begin{aligned} \text{P}(M_n^k \leq x) &= \text{P}[ {\max \left(X_{(k-1)n+1}, X_{(k-1)n+2}, \cdots, X_{kn} \right) \leq x}] \\ &= \text{P}[ X_{(k-1)n+1} \leq x, X_{(k-1)n+2} \leq x, \cdots, X_{kn} \leq x] \\ &= \text{P}[X_{(k-1)n+1} \leq x] \cdot \text{P}[X_{(k-1)n+2} \leq x] \cdots \text{P}[X_{kn} \leq x] \\ &= \left[ {F(x)} \right]^n \end{aligned}$$
例 9.4

假设某个保单集合的索赔金额服从指数分布$F(x) = 1 - {e}^{- x / 10}$，试计算在100次观测记录中，索赔最大值超过50和100的概率。

本例可视为对一个$n=100$的区块，求其最大值的分布的问题。因此有：

$$\begin{gathered}\text{P} \left[ {M_n \leq x} \right] = \left[ {F(x)} \right]^n \\ \text{P} \left[ {M_{100} > x} \right] = 1 - \left[ {F\mbox{(}x)} \right]^{100} = 1 - \left( {1 - e^{ - x / 10}} \right)^{100}\end{gathered}$$

代入$x=50$和$x=100$，可计算两个概率分别为0.49023和0.00453。

当$F(x)$为指数分布，本例还存在如下的近似公式：

$$\begin{aligned} \text{P} \left( {\frac{M_n}{10} - \ln n \leq x} \right) &= \text{P}[M_n \leq 10(x + \ln n)] \\ &= \left[ {F\left( {10(x + \ln n\mbox{)}} \right)} \right]^n = \left(1 - e^{-x}/n \right)^n \\ \end{aligned}$$

当区块长度$n$足够大时，有：

$$\lim\limits_{n \to \infty} \text{P}(\frac{M_n }{10} - \ln n \leq x) = \lim\limits_{n \to \infty} \left(1 - e^{-x}/n \right)^n = e^{-e^{-x}}$$

故$M_n$的分布函数也可写为：

$$\begin{gathered}\text{P} \left[ {M_n \leq z} \right] = \text{P}\left( {\frac{M_n }{10} - \ln n \leq \frac{z}{10} - \ln n} \right) \approx \exp \left[-\exp(-\frac{z}{10} + \ln n) \right]\end{gathered}$$

容易验证两种方法计算的结果一致。

9.2.2 广义极值分布

定理 9.1（Fisher-Tippett-Gnedenko）

极值理论描述了当区块长度$n$足够大时，最大值$M_n$的分布。根据Fisher-Tippett-Gnedenko定理，对于一组独立同分布的随机变量$X_1,X_2,\cdots,X_n$，其中的最大值记为$M_n = \max(X_1,X_2,\cdots,X_n)$，若存在常数序列$\{a_n>0\}$和$\{b_n\}$，使得$n \to \infty$时标准化最大值的分布函数收敛，即：

$$\begin{gathered}\text{P}\left(\frac{M_n - b_n}{a_n} \leq x \right) \to G(x),\ \forall x \in \mathbb{R}\end{gathered}$$

则$G(x)$必属于Gumbel分布、Fréchet分布或Weibull分布三者之一。

三种分布对应的累积分布函数分别为：

(1) Gumbel分布：

$$\begin{gathered}G_1 (x) = \exp \left\{ { -\exp \left[ { -\left({\frac{x - \mu}{\sigma}} \right)} \right]} \right\},\ -\infty < x < \infty,~\sigma > 0\end{gathered}$$

(2) Fréchet分布：

$$\begin{gathered}G_2 (x) = \exp \left\{ {-\left({\frac{x - \mu}{\sigma}} \right)^{ -\alpha}} \right\},\ x > \mu ,~\alpha > 0,~\sigma > 0\end{gathered}$$

(3) Weibull分布：

$$\begin{gathered}G_3 (x) = \exp \left\{ { -\left[ {-\left({\frac{x - \mu}{\sigma}} \right)} \right]^\alpha} \right\},\ x \leq \mu,~\alpha > 0\end{gathered}$$

def gev_pdf(x, mu, sigma, xi):
    """GEV概率密度函数"""
    z = (x - mu) / sigma
    if abs(xi) < 1e-8:
        t = np.exp(-z)
        return (1/sigma) * t * np.exp(-t)
    else:
        if np.any(1 + xi * z <= 0):
            return 0
        t = (1 + xi * z) ** (-1/xi)
        return (1/sigma) * t**(xi + 1) * np.exp(-t)

proc iml;
  /* GEV分布函数:
     - ξ > 0 (Frechet): F(x) = exp(-(1+ξx/σ)^(-1/ξ)), 1+ξx/σ > 0
     - ξ = 0 (Gumbel): F(x) = exp(-exp(-x/σ))
     - ξ < 0 (Weibull): F(x) = exp(-(1+ξx/σ)^(-1/ξ)), 1+ξx/σ > 0
  */

  /* 绘制不同ξ的GEV密度 */
  x = do(-4, 4, 0.1)`;
  sigma = 1;

  /* ξ = 0.5 (Frechet) */
  xi1 = 0.5;
  f1 = j(nrow(x), 1, 0);
  do i = 1 to nrow(x);
    if 1 + xi1 * x[i] / sigma > 0 then do;
      t = (1 + xi1 * x[i] / sigma)##(-1/xi1);
      f1[i] = t##(xi1 + 1) * exp(-t) / sigma;
    end;
  end;

  /* ξ = 0 (Gumbel) */
  xi2 = 0;
  f2 = j(nrow(x), 1, 0);
  do i = 1 to nrow(x);
    t = exp(-x[i] / sigma);
    f2[i] = t * exp(-t) / sigma;
  end;

  /* ξ = -0.5 (Weibull) */
  xi3 = -0.5;
  f3 = j(nrow(x), 1, 0);
  do i = 1 to nrow(x);
    if 1 + xi3 * x[i] / sigma > 0 then do;
      t = (1 + xi3 * x[i] / sigma)##(-1/xi3);
      f3[i] = t##(xi3 + 1) * exp(-t) / sigma;
    end;
  end;

  create gev_data from x [colname={'x'}];
  append from x;

  append from f2;
  close;
  create gev_f3 from f3 [colname={'f3'}];

  close;
quit;

事实上，上述分布可以统一为如下的形式，称为广义极值分布：

定义 9.1（广义极值分布）
$$G_{\xi}(x) = \left\{ \begin{aligned} &\exp \left\{-\left[1 + \xi \left( {\frac{x - \mu }{\sigma }} \right)\right]^{-1/\xi} \right\}, &\xi \neq 0 \\ &\exp \left\{-\exp\left[-\left(\frac{x - \mu}{\sigma} \right)\right]\right\}, &\xi = 0 \end{aligned} \right.$$

广义极值分布包含3个参数，其中$\mu$称为位置参数（location），$\sigma>0$称为尺度参数（scale），$\xi$称为形状参数（shape），且定义域为$\displaystyle 1+\xi(x-\mu)/\sigma>0$。位置参数$\mu$和尺度参数$\sigma$主要影响分布的中心位置和分散程度，而形状参数$\xi$决定了分布的类型和尾部行为：

  • 当$\xi = 0$时，退化为Gumbel分布（是$\xi \neq 0$时函数的极限情况）；
  • 当$\xi > 0$时，对应Fréchet分布；
  • 当$\xi < 0$时，对应Weibull分布。

当$\mu=0,~\sigma=1$时，上式可化简成标准化的广义极值分布：

$$G_\xi (x) = \left\{ \begin{aligned} &\exp [-(1 + \xi x)^{-1/\xi}], &\xi \neq 0 \\ &\exp (-e^{-x}), &\xi = 0 \end{aligned} \right.$$

极值理论表明：当区块长度$n$足够大时，可以使用广义极值分布近似最大值$M_n$的分布函数。

9.2.3 广义极值分布的性质

1. 广义极值分布的稳定性

广义极值分布的稳定性是指，若$X_1, X_2,\cdots,X_n \overset{\text{i.i.d.}}{\sim} G_{\xi}(x)$，则该组随机变量的最大值$M_n = \max(X_1,X_2,\cdots,X_n)$也服从广义极值分布（广义极值分布随机变量的最大值仍然服从广义极值分布）。

例如，不妨令$\mu=0,~\sigma=1$，则$X_i$的分布函数为$G_{\xi}(x) = \exp [-(1 + \xi x)^{-1/\xi}]$。类似可求其最大值$M_n$的分布函数为：

$$\begin{aligned} \text{P}(M_n \leq x) &= [G_{\xi}(x)]^n = \exp [-n (1 + \xi x)^{-1/\xi}] \\ &= \exp \left[- \left(\frac{1 + \xi x}{n^\xi} \right)^{-1/\xi} \right] \\ &= \exp \left[- \left(1 + \frac{\xi x + 1 - n^\xi}{n^\xi} \right)^{-1/\xi} \right] \\ &= \exp \left\{ - \left[ {1 + \xi \frac{x - \left(\xi^{-1} n^\xi - \xi^{-1} \right)}{n^\xi }} \right]^{-1/\xi} \right\} \end{aligned}$$

上式等价于位置参数为$(\xi^{-1} n^\xi - \xi^{-1})$、尺度参数为$n^\xi$的广义极值分布。

2. 最大吸引域

定义 9.4（最大吸引域）

给定一组随机变量$X_1, X_2,\cdots,X_n \overset{\text{i.i.d.}}{\sim} F(x)$，若在适当的线性变换下最大值$M_n$可收敛到广义极值分布$G(x)$，则称分布$F$属于广义极值分布$G$的最大吸引域（maximum domain of attraction），记为$F \in \text{MDA}(G)$。

(1) Fréchet分布的最大吸引域

定义 9.5（Fréchet最大吸引域）

若$F$属于Fréchet最大吸引域，则其生存函数应具有如下形式：

$$\begin{gathered}\bar{F}(x) = x^{-1/\xi} L(x),~\xi>0\end{gathered}$$

其中，$L(x)$是一个慢变函数（slowly varying function），满足：

$$\begin{gathered}\lim\limits_{x \to \infty} \frac{L(tx)}{L(x)} = 1,\ \forall t > 0\end{gathered}$$

Fréchet最大吸引域主要对应厚尾分布（heavy-tailed distributions），生存函数在尾部以幂次速率$x^{-1/\xi}$衰减。常见的属于Fréchet最大吸引域的分布包括：帕累托分布、对数伽马分布、逆伽马分布、t分布、F分布和Burr分布等。

典型的慢变函数如常数函数$L(x) = C$、对数函数$L(x) = \ln(x)$等。由于慢变函数在极限处的变化较慢，故$L(x)$不会改变生存函数尾部的衰减规律，主要仅起到对生存函数进行修正的作用。

由上述定理可知，Fréchet最大吸引域主要对应厚尾分布（heavy-tailed distributions），生存函数在尾部以幂次速率$x^{-1/\xi}$衰减，下降较缓且尾部较长。作为对比，指数分布的生存函数$\bar{F}(x)=e^{-\lambda x}$以指数速率衰减，下降速度远远高于幂次衰减。

衰减率$\alpha = 1/\xi$通常称为分布的尾部指数：若$\xi$较大，则分布的尾部较厚，极端值出现的概率越大；反之，则分布的尾部较薄。若$F \in \text{MDA}(G_\xi),~\xi>0$，则随机变量的矩在$k > 1/\xi$时不存在，即$\mathbb{E}(X^k)=\infty$。

幂次衰减的生存函数示例

常见的属于Fréchet最大吸引域的分布包括：帕累托分布、对数伽马（Log-Gamma）分布、逆伽马分布、t分布、F分布和Burr分布等。

(2) Gumbel分布的最大吸引域

定义 9.6（Gumbel最大吸引域）

Gumbel最大吸引域通常对应尾部呈指数型衰减的分布，生存函数的衰减速度比幂函数更快，且存在任意阶矩，即$\mathbb{E}(X^k)<\infty,~\forall k >0$。常见的属于Gumbel最大吸引域的分布包括：正态分布、指数分布、伽马分布、逆高斯分布、对数正态分布、卡方分布、双曲分布、标准威布尔分布、Benktander分布等。

(3) Weibull分布的最大吸引域

定义 9.7（Weibull最大吸引域）

Weibull最大吸引域对应的分布具有明确的上界（右端点），分布的尾部有限。常见的属于Weibull最大吸引域的分布包括：贝塔分布、均匀分布、三角分布等，在金融保险领域应用较少。

当随机变量$X$的分布已知时，根据其与广义极值分布最大吸引域的对应关系，即可使用适当的分布模拟$M_n$的分布特征。

需要注意的是，极值理论仅仅给出了区块最大值的渐进分布。而对于最小值的分布，需要对数据取负号$Y_i = -X_i$，转换为如下问题：

9.2.4 应用案例

现在回到~节的概念。假设有一组长期观察的时间序列数据$\{X_1,X_2,\cdots\}$，将数据按照固定的时间间隔（例如年、月、季等）分块，并提取各时间段内的最大值。已知当区块内的样本数量（即区块长度$n$）足够大时，最大值序列$\{M_n^1, M_n^2,\cdots\}$将近似服从广义极值分布。在此基础上，精算师可以进一步分析极端事件的统计特性，从而对未来极端事件的发生概率和严重程度进行估计。

例如，某河流的防洪设施（如堤坝、排水系统等）需要按照100年一遇的洪水标准进行设计，为获得该信息，研究者获取了该河流在过去50年（1975-2024年）的每日最大水位数据。此时，可以按照年份将数据分块，每个区块包含$n=365$或$366$天的日水位高度，提取各区块的最大值，即可得到年最大水位序列$\{M_n^1,M_n^2,\cdots,M_n^{50}\}$的观测值。进一步结合日最大水位$X$的分布特性，选择合适的极值分布并应用极大似然法估计参数$(\mu,\sigma,\xi)$，最终拟合得到年最大水位$M_n$的分布。在这一过程中，同样可以应用前述章节的方法，对极值分布的拟合优度进行检验和诊断。

重现期（return period）和重现水平（return level）是极端风险管理的两个重要概念。重现期是指某个极端事件在长时间内的平均发生时间间隔，反映了事件发生的频率和可能性。直观来讲，100年一遇洪水的重现期为$T=100$，意味着这种规模的洪水平均每100年发生一次。而重现水平是指与重现期对应的极端事件的预期最大强度。例如对于重现期为$T$的事件，其在任意一期中发生的概率均为$1/T$。因此，取随机变量$M_n$的水平$\alpha = 1-1/T$的分位数（VaR），即可得到每期的极端事件$M_n$的最大可能值。

已知若$M_n$服从上式所示的广义极值分布，则其水平为$\alpha$的VaR等于：

$$\text{VaR}_{\alpha}(M_n) = \left\{ \begin{aligned} &\mu + \frac{\sigma}{\xi}[(-\ln \alpha)^{-\xi} - 1], & \xi \neq 0 \\ &\mu - \sigma \ln(-\ln \alpha), & \xi =0 \end{aligned} \right.$$

应用上述公式，即可计算对应重现期$T$的重现水平。

例 9.5

某河流的100年一遇洪水的预期水位为10米，请解释各数字的含义。

100年是该洪水事件的重现期，表示在统计上，水位达到10米的洪水事件平均每100年发生一次，每年内发生发生类似规模洪水的概率为0.01。

若事先给定极端事件的水平$C$，则可以应用分布函数$G_{\xi}(\cdot)$反解与$C$对应的分位数水平$p$，即$G_{\xi}(C) = p$。每期内发生类似强度或更极端事件的概率等于$1-p$，故其对应的重现期为$T=1/(1-p)$。重现期与重现水平可以很容易地互相转换。

在实际应用中，区块长度$n$对模型偏差和估计方差具有重要影响。一般而言，当区块长度增加时，广义极值分布在理论上更接近区块最大值$M_n$的真实分布，模型的拟合效果更好；但在样本总量固定的情况下，每个区块越大，则能够提取的最大值观测样本越少，导致参数估计的方差增大、稳定性降低。两者之间需要进行平衡。此外，当原始观察数据$X_i$存在相依性时，通常应选择比独立情况下更大的区块长度；否则各区块最大值的独立性可能受到影响，导致与广义极值分布发生偏离。

除时间维度以外，区块最大值模型也可以扩展到空间分块，即对不同区域或地理空间上的事件极值进行分析。

     - ξ = 0 (Gumbel): F(x) = exp(-exp(-x/σ))

9.3 超阈值模型

9.3.1 超越值的概念

给定随机变量$X$和阈值$u$，可定义其超越值（exceedance）为：

$$Y = X - u \mid X > u$$

当$X$表示风险损失、$u$表示免赔额时，$Y$等价于保险公司在有免赔额情况下的赔付（即超额损失）。

记$X$的分布函数为$F(x)$，则超越值的分布函数可以表示为：

$$\begin{aligned} F_Y(y) &= \text{P}(Y \leq y) = \text{P}(X-u \leq y \mid X > u) \\ &= \frac{\text{P}(u < X \leq u+y)}{\text{P}(X > u)} = \frac{F(u+y)-F(u)}{1-F(u)},\ y>0 \end{aligned}$$

9.3.2 广义帕累托分布

定义 9.2（广义帕累托分布）

两参数广义帕累托分布具有如下的分布函数：

$$G(x) = 1 - \left(1 + \xi \frac{x}{\sigma}\right)^{-1/\xi}$$

其中，$\sigma>0$是尺度参数，$\xi$是形状参数，定义域需满足$x \geq 0$且$1 + \xi x / \sigma > 0$。当形状参数$\xi < 0$，广义帕累托分布存在有限上界，定义域为$[0,-\sigma/\xi)$；当$\xi > 0$时，定义域为$[0,+\infty)$。

当$\xi=0$时，广义帕累托分布退化为指数分布：

$$\begin{aligned} \lim_{\xi \to 0} G(x) &= 1-\lim_{\xi \to 0} (1+\xi \frac{x}{\sigma})^{-1/\xi} \\ &= 1 - \lim_{\xi \to 0}~ \exp \left[-\frac{1}{\xi} \ln(1+\frac{\xi x}{\sigma}) \right] \\ &= 1 - \lim_{\xi \to 0}~ \exp \left[-\frac{x}{\sigma} \cdot \frac{1}{1 + \xi x / \sigma} \right] \\ &= 1 - e^{-x/\sigma} \end{aligned}$$

容易求得广义帕累托分布的概率密度函数为：

$$g(x) = \left\{ \begin{aligned} &\frac{1}{\sigma}(1+\xi \frac{x}{\sigma})^{-1-1/\xi}, &\xi \neq 0 \\ &\frac{1}{\sigma}\exp(-\frac{x}{\sigma}), &\xi = 0 \end{aligned} \right.$$

广义帕累托分布的$k$阶仅当$\xi < 1/k$时存在，此处列出其期望和方差：

$$\begin{gathered}\mathbb{E}(X) = \frac{\sigma}{1 - \xi},\ \xi < 1 \\ \text{Var}(X) = \frac{\sigma ^2}{(1-\xi ^2)(1-2\xi)},\ \xi <\frac{1}{2}\end{gathered}$$

前图~展示了不同参数的广义帕累托分布的概率密度函数。当$\xi$固定时，增大$\sigma$将使概率密度函数变平缓、尾部变厚。当$\sigma$固定时，若形状参数$\xi>0$，则增大$\xi$将使尾部概率增加、尾部变厚；而若$\xi<0$，概率密度函数会快速下降并截断。

def gev_pdf(x, mu, sigma, xi):
    """GEV概率密度函数"""
    z = (x - mu) / sigma
    if abs(xi) < 1e-8:
        t = np.exp(-z)
        return (1/sigma) * t * np.exp(-t)
    else:
        if np.any(1 + xi * z <= 0):
            return 0
        t = (1 + xi * z) ** (-1/xi)
        return (1/sigma) * t**(xi + 1) * np.exp(-t)

def gev_nll(parms, x):
    mu = parms[0]
    sigma = np.exp(parms[1])
    xi = parms[2]
    z = (x - mu) / sigma
    if np.any(1 + xi * z <= 0):
        return 1e10
    if abs(xi) < 1e-8:
        t = np.exp(-z)
    else:
        t = (1 + xi * z) ** (-1/xi)
    return np.sum(np.log(sigma) + (1 + 1/xi) * np.log(t) + t)

/* 9.2 广义极值分布（GEV）                                                  */
/****************************************************************************/
proc iml;
  /* GEV分布函数:
     - ξ > 0 (Frechet): F(x) = exp(-(1+ξx/σ)^(-1/ξ)), 1+ξx/σ > 0
     - ξ = 0 (Gumbel): F(x) = exp(-exp(-x/σ))
     - ξ < 0 (Weibull): F(x) = exp(-(1+ξx/σ)^(-1/ξ)), 1+ξx/σ > 0
  */

  /* 绘制不同ξ的GEV密度 */
  x = do(-4, 4, 0.1)`;
  sigma = 1;

  /* ξ = 0.5 (Frechet) */
  xi1 = 0.5;
  f1 = j(nrow(x), 1, 0);
  do i = 1 to nrow(x);
    if 1 + xi1 * x[i] / sigma > 0 then do;
      t = (1 + xi1 * x[i] / sigma)##(-1/xi1);
      f1[i] = t##(xi1 + 1) * exp(-t) / sigma;
    end;
  end;

  /* ξ = 0 (Gumbel) */
  xi2 = 0;
  f2 = j(nrow(x), 1, 0);
  do i = 1 to nrow(x);
    t = exp(-x[i] / sigma);
    f2[i] = t * exp(-t) / sigma;
  end;

  /* ξ = -0.5 (Weibull) */
  xi3 = -0.5;
  f3 = j(nrow(x), 1, 0);
  do i = 1 to nrow(x);
    if 1 + xi3 * x[i] / sigma > 0 then do;
      t = (1 + xi3 * x[i] / sigma)##(-1/xi3);
      f3[i] = t##(xi3 + 1) * exp(-t) / sigma;
    end;
  end;

  create gev_data from x [colname={'x'}];
  append from x;
  close;
  create gev_f1 from f1 [colname={'f1'}];
  append from f1;
  close;
  create gev_f2 from f2 [colname={'f2'}];
  append from f2;
  close;
  create gev_f3 from f3 [colname={'f3'}];
    append from f3;
  close;
quit;

data gev_plot;

定理 9.2（Balkema-de Haan）

根据Pickands-Balkema-de Haan定理，若原始随机变量的分布$F$属于广义极值分布$G_{\xi}$的最大吸引域，则当阈值$u$足够大时，超越值$Y = X - u \mid X > u$将近似服从上式定义的广义帕累托分布，且形状参数$\xi$保持不变。

由于常见的连续型损失分布都属于广义极值分布的最大吸引域，故当阈值足够大时，超额损失$Y$均服从广义帕累托分布。

9.3.3 广义帕累托分布的性质

1. 广义帕累托分布的稳定性

当$X$服从广义帕累托分布时，$Y = X - u \mid X > u$仍然服从广义帕累托分布。

证明：根据超越值的分布函数定义，有：

$$\begin{aligned} F_Y(y) &= \frac{F(u+y)-F(u)}{1-F(u)} = \frac{\bar{F}(u) - \bar{F}(u+y)}{\bar{F}(u)} \\ &= 1 - \left[\frac{1 + \xi (u+y) / \sigma}{1 + \xi u / \sigma}\right]^{-1/\xi} \\ &= 1 - \left[\frac{\sigma + \xi (u+y)}{\sigma + \xi u}\right]^{-1/\xi} = 1 - \left(1 + \xi \frac{y}{\sigma + \xi u} \right)^{-1/\xi} \end{aligned}$$

恰好服从形状参数为$\xi$、尺度参数为$\sigma+\xi u$的广义帕累托分布。 $\square$

2. 更大阈值下的超越值分布

若给定阈值$u$，已知$Y = X-u \mid X>u$近似服从广义帕累托分布$\text{GPD}(\sigma,\xi)$。则对于更大的阈值$v \geq u$，超越值$Z = X-v \mid X>v$仍然近似服从广义帕累托分布，且形状参数$\xi$保持不变，尺度参数调整为$\sigma + \xi (v-u)$。

证明：首先考察$Y$的生存函数，有：

$$\begin{gathered}\bar{F}_Y(y) = \text{P}(X-u>y \mid X>u) = \frac{\bar{F}(u+y)}{\bar{F}(u)} \approx \bar{G}(y)\end{gathered}$$

可知$\bar{F}(u+y) \approx \bar{F}(u) \bar{G}(y)$。对于$Z$的生存函数，类似有：

$$\begin{aligned} \bar{F}_Z(z) &= \frac{\bar{F}(v+z)}{\bar{F}(v)} = \frac{\bar{F}(u + (v+z-u))}{\bar{F}(u+(v-u))} \\ &\approx \frac{\bar{F}(u) \bar{G}(v+z-u)}{\bar{F}(u) \bar{G}(v-u)} \\ &= \left[\frac{1+\xi(v+z-u)/\sigma}{1+\xi(v-u)/\sigma}\right]^{-1/\xi} = \left[1 + \xi \frac{z}{\sigma + \xi(v-u)} \right]^{-1/\xi} \end{aligned}$$

恰好是形状参数为$\xi$、尺度参数为$\sigma + \xi(v-u)$的广义帕累托分布的生存函数。 $\square$

3. $X$的平均超越函数和尾部特征

平均超越函数$e(v)$描述了$X$超过$v$部分的损失的条件期望。给定阈值$u$，若已知超越值$Y = X-u \mid X>u$近似服从广义帕累托分布，则当$v \geq u$时，$X$的平均超越函数可以写为：

$$\begin{gathered} e(v) = \mathbb{E}(X - v \mid X > v) = \mathbb{E}(Z) = \frac{\sigma + \xi (v-u)}{1-\xi},\ \xi<1\end{gathered}$$

其中，$Z = X - v \mid X > v$服从广义帕累托分布$\text{GPD}(\sigma + \xi (v-u),\xi)$。这是广义帕累托分布的一个重要性质，即原始损失$X$的平均超越值是关于$(v-u)$的线性函数。

$X$的生存函数$\bar{F}(x)$在$x \geq u$的部分可以表示为：

$$\begin{gathered} \bar{F}(x) \approx \bar{F}(u) \bar{G}(x-u) \approx \bar{F}(u) \left(1 + \xi \frac{x-u}{\sigma} \right)^{-1/\xi}\end{gathered}$$

该结果的证明可参考“更大阈值下的超越值分布”的思路。

根据形状参数$\xi$的不同，随机变量$X$的生存函数在其尾部$X \geq u$将表现出不同的衰减特性（要求$X$的分布属于广义极值分布的最大吸引域）。

对于原始风险损失$X$的VaR和TVaR，在已知超越值服从广义帕累托分布时，同样具有简单的结果。首先考虑$\text{VaR}_\alpha(X)$，根据定义需要求解$x$使$\bar{F}(x) = 1-\alpha$。假定$\alpha \geq F(u)$，则VaR的结果必大于等于$u$。此时可以将$X$的生存函数表示为上式的形式，解方程得到：

$$\text{VaR}_{\alpha}(X) = \left\{ \begin{aligned} &u+\frac{\sigma}{\xi} \left[\left(\frac{1-\alpha}{\bar{F}(u)}\right)^{-\xi} - 1\right], &\xi \neq 0 \\ &u - \sigma \ln \left( \frac{1-\alpha}{\bar{F}(u)} \right), &\xi = 0 \end{aligned} \right.$$

其次考虑$\text{TVaR}_\alpha(X)$，应用上式和上式，可得：

$$\begin{aligned} \text{TVaR}_{\alpha}(X) &= \text{VaR}_{\alpha}(X) + e(\text{VaR}_{\alpha}(X)) \\ &= \text{VaR}_{\alpha}(X) + \frac{\sigma + \xi [\text{VaR}_{\alpha}(X)-u]}{1-\xi} \\ &= \frac{\text{VaR}_{\alpha}(X) + \sigma - \xi u}{1-\xi},\ \xi<1 \end{aligned}$$

9.3.4 阈值选择和参数估计

在使用广义帕累托分布拟合尾部数据时，阈值$u$的选择至关重要。与区块最大值模型中的区块长度类似，$u$的选择同样需要在模型偏差和估计方差间进行权衡：若选取阈值太低，则尾部数据可能不满足广义帕累托分布的渐近性质，从而引入较大偏差；但若选取阈值太高，可用于拟合的样本量减少，又将导致估计结果的不确定性增大。

常用的阈值选择方法为图形诊断法。给定观察样本$(X_1,X_2,\cdots,X_n)$，可以定义该样本的经验平均超越函数$\hat{e}_n(v)$为：

$$\hat{e}_n(v) = \frac{\sum\limits_{i=1}^n (X_i-v) \mathbf{I}\{X_i > v\}}{\sum\limits_{i=1}^n \mathbf{I}\{X_i > v\}},\ v > 0$$

即对于超过$v$的样本点$X_i$，计算$(X_i-v)$的平均值。

由上式可知，当$v \geq u$时，经验平均超越函数$\hat{e}_n(v)$与$v$之间应呈线性趋势（此处假定$\xi < 1$），斜率为$\xi/(1-\xi)$。则绘制经验平均超越函数图$\hat{e}_n(v) \sim v$，寻找使该函数呈现线性趋势的起始点，即为使超越值$Y = X-u \mid X>u$渐进服从广义帕累托分布的最小阈值$u$。

当已知数据来自厚尾分布、即$\xi > 0$时，还可以使用Hill图确定合适的阈值。对于样本$(X_1,X_2,\cdots,X_n)$，首先按降序排列为次序统计量$X_{(1)} \geq X_{(2)} \geq \cdots \geq X_{(n)}$。给定正整数$k$，可以定义形状参数$\xi$的Hill估计量$\hat{\xi}(k)$为：

$$\hat{\xi}(k) = \frac{1}{k} \sum_{i=1}^k \ln \left(\frac{X_{(i)}}{X_{(k+1)}}\right)$$

Hill估计量主要使用样本中前$k$个最大值估计参数$\xi$。对于一系列不同的$k$值，分别计算对应的估计量并绘制$\hat{\xi}(k) \sim k$，称为Hill图。此时可从图中寻找估计量$\hat{\xi}(k)$相对稳定的区域，并使用区域起始点所对应的$X_{(k)}$作为阈值$u$。

在阈值确定后，广义帕累托分布的参数可以很容易地进行极大似然估计。对于固定的$u$，筛选观察值超过阈值的$(x'_1,x'_2,\cdots,x'_m)$，并记$y_j = x'_j-u~(j=1,2,\cdots,m)$。应用上式，即可得到如下的对数似然函数：

$$l \left(\sigma, \xi;~y_1, \cdots, y_m\right) = - m \ln\sigma - \frac{1 + \xi}{\xi}\sum\limits_{j=1}^m \ln \left(1 + \xi \frac{y_m}{\sigma} \right)$$

对上式求极大化，求得极大似然估计量$\hat{\sigma}, \hat{\xi}$。

进一步结合上式，得到$X$的生存函数的估计为：

$$\hat{\bar{F}}(x) = \frac{m}{n} \cdot \left(1 + \hat{\xi} \frac{x-u}{\hat{\sigma}} \right)^{-1/\hat{\xi}},\ x \geq u$$

9.3.5 应用案例

超阈值模型是极值理论中的另一类重要方法，主要用于对数据中的极端事件进行建模。其核心思想是应用阈值$u$将数据分为两部分：阈值以下的部分，仍然使用传统损失分布进行拟合；阈值以上的部分，使用广义帕累托分布对超越值进行拟合。因此，该模型实际上属于一类组合分布模型或剪接模型。

例如，对于一个取值为正的连续型随机变量$X$，假设其在$X \leq u$的部分服从截断伽马分布，概率密度函数为：

$$f(x) = \frac{f_{\text{Ga}}(x)}{F_{\text{Ga}}(u)} \cdot \mathbf{I}\{x \leq u\}$$

在$X > u$的部分，超越值$Y = X-u \mid X>u$服从广义帕累托分布，概率密度函数和累积分布函数分别如上式和上式。则在POT模型的框架下，$X$的概率密度函数和累积分布函数可以统一表示为：

$$h(x) = \left\{ \begin{aligned} &p \cdot f(x), & x \leq u \\ &(1-p) \cdot g(x-u), & x > u \end{aligned} \right.$$
$$H(x) = \left\{ \begin{aligned} &p \cdot \frac{F_{\text{Ga}}(x)}{F_{\text{Ga}}(u)}, & x \leq u \\ &p + (1-p) \cdot G(x-u), & x > u \end{aligned} \right.$$

分别使用对应部分的样本估计伽马分布和广义帕累托分布的参数，即可得到$X$分布规律的完整表述，且对于尾部极端事件的刻画更加准确。

def gev_pdf(x, mu, sigma, xi):

     - ξ = 0 (Gumbel): F(x) = exp(-exp(-x/σ))
     - ξ < 0 (Weibull): F(x) = exp(-(1+ξx/σ)^(-1/ξ)), 1+ξx/σ > 0

  x = do(-4, 4, 0.1)`;

  xi1 = 0.5;

      t = (1 + xi1 * x[i] / sigma)##(-1/xi1);
      f1[i] = t##(xi1 + 1) * exp(-t) / sigma;
    end;

  xi2 = 0;

  xi3 = -0.5;

quit;


/****************************************************************************/
/* 9.8 极值理论应用：VaR估计                                                */
/****************************************************************************/
proc iml;
  /* 使用GPD估计高置信水平VaR */
  call streaminit(101);
  n = 1000;
  x = j(n, 1, 0);
  do i = 1 to n;
    x[i] = rand('EXPONENTIAL', 1);
  end;

  /* 选择阈值 */
  u = 2;
  excess = x[loc(x > u)] - u;
  n_excess = nrow(excess);
  n_total = nrow(x);

  /* GPD参数估计（简化：使用矩估计） */
  xbar = mean(excess);
  s2 = var(excess);
  xi_hat = 0.5 * (xbar**2 / s2 - 1);
  sigma_hat = 0.5 * xbar * (xbar**2 / s2 + 1);

  /* VaR估计 */
  p_values = {0.95, 0.99, 0.999};
  do i = 1 to nrow(p_values);
    p = p_values[i];
    /* VaR_p = u + σ/ξ * [(n/n_u * (1-p))^(-ξ) - 1] */
    VaR = u + sigma_hat / xi_hat *
          ((n_total / n_excess * (1 - p))##(-xi_hat) - 1);
    print "p=" p "VaR_GPD =" VaR;
  end;

  /* 与经验VaR比较 */
  x_sorted = x;
  call sort(x_sorted, 1);
  do i = 1 to nrow(p_values);
    p = p_values[i];
    idx = ceil(p * n_total);
    VaR_emp = x_sorted[idx];
    print "p=" p "VaR_emp =" VaR_emp;
  end;
quit;

9.4 真题练习

1. 一家水电公司正在管理一个由河谷中的水坝围成的水库，该水坝最初的设计目的是使得水位在每300天内约有2天会超过50米的阈值。在极端情况下，水库会排放多余的水，以确保水位不会超过安全的50米限制。

据研究，该水库的每日水位服从一个平均值为$\mu$的指数分布。

(1) 估计$\mu$的值。

(2) 计算水位超过50米阈值的超越值的期望。

为了更好地管理多余的水量，现在假设超过阈值的水位服从尺度参数$\beta = 1$的广义帕累托分布。

(3) 解释在何种情况下，广义帕累托分布比指数分布更合适。

(4) 假设超越值的期望与第(2)问相同，估计广义帕累托分布的形状参数。

【解答】(1) 已知指数分布的生存函数为$S(x) = e^{-x/\mu}$，则

$$S(50) = e^{-50/\mu} = 2/300 ~~\Rightarrow~~ \mu = -50/\ln(2/300) = 9.978775$$

(2) 记$U = X-50 \mid X > 50$。由于指数分布具有无记忆性，则易知：

$$\mathbb{E}(U) = \mathbb{E}(X-50 \mid X > 50) = \mathbb{E}(X) = 9.978775$$

(3) 如果极端天气事件在未来更有可能发生，则广义帕累托分布更加合适，此时水位超越值将具有更厚的尾部（与超越值为指数分布时相比）。当情况恶化时，对尾部建模的重要性愈加凸显。

(4) 根据广义帕累托分布的性质，当尺度参数等于1时，有：

$$\mathbb{E}(U) = \frac{1}{1-\xi} = 9.978775 ~~\Rightarrow~~ \xi = 0.8997873,\ \gamma = \frac{1}{\xi} = 1.111374$$

2. 令$X_n$表示一组独立同分布的随机变量。试对以下情况，证明$X_n$的分布满足：

$$\begin{gathered}[F(\beta_n x+\alpha_n)]^n = \left\{1 - \frac{1}{n} \left[1 + \frac{C(x-A)}{B} \right]^{-1/C} \right\}^n\end{gathered}$$

(1) $X_n$服从$[0,1]$区间上的均匀分布，且$\alpha_n = 1-1/n,~ \beta_n = 1/n,~ A=0,~ B=1,~ C=-1$。

(2) $X_n$服从参数为$\delta,\lambda$的两参数帕累托分布，且$\alpha_n = \lambda(n^{1/\delta}-1),~ \beta_n = n^{1/\delta},~ A=0,~ B=\lambda/\delta,~ C=1/\delta$

(3) 考虑当$n \to \infty$时的极限行为，解释上述结果的意义。

【解答】(1) 已知$F(x)=x$，代入$\alpha_n$和$\beta_n$，有：

$$\begin{aligned} \left[ F(\beta_n x+\alpha_n) \right]^n &= (\beta_n x+\alpha_n)^n = (x/n + 1 - 1/n)^n \\ &= \left[1 - \frac{1}{n} \cdot (1-x) \right]^n \end{aligned}$$

代入$A,B,C$的值进行比较，可知上式服从题目要求的形式。

(2) 已知$\displaystyle F(x) = 1 - (\frac{\lambda}{\lambda + x})^\delta$，代入$\alpha_n$和$\beta_n$，有：

$$\begin{aligned} \left[ F(\beta_n x+\alpha_n) \right]^n &= \left[ 1 - \left(\frac{\lambda}{\lambda + \beta_n x + \alpha_n}\right)^\delta \right]^n = \left[1 - \left(\frac{\lambda + \beta_n x + \alpha_n}{\lambda}\right)^{-\delta} \right]^n \\ &= \left[1 - \left(\frac{\lambda n^{1/\delta} + n^{1/\delta} x}{\lambda}\right)^{-\delta} \right]^n = \left[1 - \frac{1}{n} \left(1 + \frac{x}{\lambda}\right)^{-\delta} \right]^n \\ \end{aligned}$$

代入$A,B,C$的值进行比较，可知上式服从题目要求的形式。

(3) 公式左侧是$n$个随机变量中的区块最大值小于等于$\beta_n x + \alpha_n$的概率，即要求所有随机变量的取值都小于等于$\beta_n x + \alpha_n$。

注意$\lim\limits_{n \to \infty} (1+x/n)^n \to \exp(x)$，故公式右侧将趋于$\exp \left\{- \left[1 + \frac{C(x-A)}{B} \right]^{-1/C} \right\}$，这是广义极值分布的分布函数。因此，本题证明了当区块长度$n$足够大时，区块最大值的线性函数的分布趋近于广义极值分布。

在第(1)问中，形状参数$C$小于0，表明其属于Weibull型广义极值分布。这一结果符合预期，因为均匀分布存在一个有限的上界。

在第(2)问中，形状参数$C$大于0，表明其属于Fréchet型广义极值分布。这同样符合预期，因为帕累托分布具有厚尾特性。

3. 一家航空公司关注因货币波动可能带来的财务损失。该公司聘请了一名分析师对多个国家的每日货币波动进行建模，目标是了解极端货币波动可能发生的概率，这些波动将对航空公司造成重大财务影响。

分析师建议使用几种标准统计分布拟合数据集，包括正态分布、伽马分布、广义帕累托分布和对数正态分布，采用最大似然估计方法。随后即可应用所选择的模型，进一步估计极端货币波动的概率。

(1) 讨论上述方法的潜在问题。

分析师随后使用广义帕累托分布（GPD）来评估极端货币损失的大小。在这一过程中，货币损失被视为正值，盈利则被视为负值。

设定每日损失的阈值为+2.2%，从3000个原始数据点中选出150个“极端”数据点用于极值分析。分析得到以下参数估计结果：$\beta = 0.009785,\ \gamma = 5.417$。

(2) 使用上述参数值计算$G(0.02)$，其中$G$为广义帕累托分布的分布函数，并解释其含义。

(3) 基于该GPD模型，估算下一日的货币损失超过5%的概率。

由前所述，正态分布也被用于拟合该数据，最优的拟合结果为$\text{N}(0, 0.014^2)$；但货币收益率通常被认为具有厚尾特征。

(4) 基于该正态分布，计算每日货币损失超过5%的概率，并对结果进行评论。

(5) 讨论分析师在拟合模型时需要注意的其他问题，尤其是使用极值理论和/或广义帕累托分布（GPD）时可能遇到的问题。

【解答】(1) 由于分布中心部分的数据点的数量通常比尾部更多，最大似然估计倾向于赋予分布中心部分更多的权重，可能导致估计分布在拟合尾部时表现较差。这将导致对极端货币波动概率的估计不准确。

此外，最大似然估计得到的最佳模型也可能因为其他原因而不适用。

(2) 已知广义帕累托的分布函数为$\displaystyle G(x) = 1 - \left(1 + \frac{x}{\gamma \beta} \right)^{-\gamma}$，代入数据可得：

$$G(0.02) = 1 - \left(1 + \frac{0.02}{5.417 \times 0.009785} \right)^{-5.417} = 0.823$$

上式为已知货币损失超过2.2%、该损失不超过4.2%的条件概率。

(3) 已知货币损失超过2.2%、该损失不超过5%的条件概率为$G(0.028) = 0.8995$，故已知货币损失超过2.2%、该损失超过5%的条件概率为$1-G(0.028) = 0.1005$。

因此，估算货币损失超过5%的无条件概率等于$0.1005 \times 150 / 3000 = 0.0050$。

(4) 查表可知，该概率等于$1 - \Phi(0.05/0.014) = 1 - \Phi(3.57) = 0.00018$。

在这一假设下，估计损失超过5%的概率远低于广义帕累托分布下的概率结果。鉴于对厚尾特性的预期，有理由相信广义帕累托分布比正常分布更为适用，使用正常分布预计会显著低估极端货币波动的风险。

(5) 分析师应当调查使用不同阈值的效果；

应当检验广义帕累托分布的拟合优度，例如使用卡方检验；

应当研究不同货币之间的波动性的联合表现；

应当对货币波动分布在时间上保持不变的假设进行验证。
