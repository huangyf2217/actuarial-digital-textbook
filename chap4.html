第4章 费率厘定

本章学习目标

  1. 了解非寿险准备金的类型。
  2. 认识未决赔款准备金的流量三角形（run-off triangle），定义进展因子并预测流量三角形的发展。
  3. 掌握链梯法（chain-ladder method）的原理和应用，并在考虑通货膨胀情况下对链梯法进行调整。
  4. 描述未决赔款准备金评估的其他常用方法，包括案均赔款法和B-F法（Bornhuetter-Ferguson method）。
  5. 讨论前述方法的应用假设。

4.1 流量三角形

非寿险准备金是经营非寿险业务的保险公司为履行保单赔付责任和其他相关义务所预留的资金储备，是经营过程中最重要的负债项目。准备金评估的准确性直接影响公司的偿付能力和财务稳定性。

保险公司非寿险业务准备金包括未到期责任准备金和未决赔款准备金。其中，未到期责任准备金是指在准备金评估日为尚未终止的保险责任而提取的准备金，包括未赚保费准备金（即以未满期部分保费收入为基础所计提的准备金）及保费不足准备金（即在进行保费充足性测试时，若未来净现金流出大于未赚保费准备金，则取其差额作为未到期责任准备金的一部分）。由于未到期责任准备金主要依赖保费的时间分布，评估过程相对简单，故不是非寿险精算研究的主要问题。

未决赔款准备金是指保险公司为保险事故已经发生但尚未最终结案的损失提取的准备金，包括已发生已报案未决赔款准备金、已发生未报案未决赔款（incurred but not reported，IBNR）准备金和理赔费用准备金三类。其中，已发生已报案未决赔款准备金是指为保险事故已经发生并已向保险公司提出索赔、但保险公司尚未结案的损失而提取的准备金，评估过程一般采用逐案估计法（对每个赔案的金额逐一进行估计）或案均赋值法（根据过去同类业务的平均赔款金额进行估计）；已发生未报案未决赔款准备金则主要面向以下四类赔案，包括：

(1) 保险事故已经发生但尚未向保险公司提出索赔；

(2) 已经提出索赔但保险公司尚未立案；

(3) 保险公司已立案但对事故损失估计不足，预计最终赔付将超过原估损值的；

(4) 保险事故已经赔付但有可能再次提出索赔。

理赔费用准备金是指为尚未结案的损失可能发生的直接或间接费用而提取的准备金。

未决赔款准备金的产生主要来自保险事故发生、报告和理赔完成之间的迟滞。在非寿险领域，由于案件调查、责任界定、估损争议、诉讼处理等流程的复杂性，导致赔付周期较长，延迟时间可能少则几天（如日常车险业务）、多则数年（如重大自然灾害或意外事故）。特别对于已经发生但尚未报案的事故，由于保险公司的信息不充分，IBNR准备金的评估存在较大的不确定性，需要依赖一定的假设和精算模型。因此，本章将重点介绍其评估方法。

流量三角形（run-off triangles）是非寿险未决赔款准备金评估的常用工具，通过分析索赔数据的历史发展趋势，预测未来索赔和准备金需求。考虑如下一个例子，假设保险公司长期经营某个险种或业务，在每个准备金评估时点（如2024年末），将不同年度发生的事故在此后每一年的累积赔款进展情况绘制成表格，得到结果如下：

4.2 链梯法

4.2.1 计算过程

观察前例的进展因子，易发现其分子部分可以写成：

$$\begin{gathered}\frac{1410}{786} \times 786 + \frac{1575}{904} \times 904 + \frac{1814}{995} \times 995 + \frac{2142}{1220} \times 1220\end{gathered}$$

因此，进展因子的计算可简化为进展年1与进展年0的累积赔款之比，即：

$$\frac{1410 + 1575 + 1814 + 2142}{786 + 904 + 995 + 1220} = 1.777$$

这种将前后两个进展年的累积赔款相除，逐步计算事故赔款在两个进展年之间的进展因子的方法，称为链梯法（chain-ladder method），比喻沿着梯子逐步向上攀爬的过程。链梯法的计算过程非常简单，通过加总合适年度的数据，可以计算每个进展年的进展因子如下：

定义 4.1（链梯法）

链梯法的基本假设是每个事故年的赔款都具有相同发展趋势，故在计算进展因子时，可以使用其他事故年的数据作为参考。此外，链梯法还假设未来的通货膨胀率保持相对稳定，由于价格因素已经被包含在进展因子中，若通货膨胀率突然上升，未来的赔款金额将很可能高出预期。

当进展因子的稳定性因索赔特性或外部环境而受到影响时，精算师还可以手动对进展因子进行调整。例如若保险公司引入自动化理赔系统， 改善其理赔政策和流程，则事故处理与赔款支付的时间可能加快；当保险产品条款、核保策略等因素发生变化，或在某个年度遭遇了重大事故时，赔款的累积模式也会表现出一定差异。此时，无论是直接根据数据计算进展因子，还是基于专家知识手动设定进展因子，利用流量三角形估计最终赔款的步骤都是一致的。

值得说明的是，链梯法的思想也可以扩展到更一般的统计模型。记$C_{ij}$表示第$i$个事故年在第$j$个进展年的增量赔款，则可以使用如下公式刻画其变化规律：

$$C_{ij} = s_i \cdot r_j \cdot x_{i+j} + e_{ij}$$

其中，$s_i$表示随事故年$i$变化的参数，例如风险单位数等；$r_j$是对应的进展因子，表示进展年$j$支付赔款所占的比例，且$r_j$与事故年$i$无关；$x_{i+j}$是随日历年（会计年）变化的参数，如通胀率等；$e_{ij}$为随机误差项。通过将增量赔款视为随机变量，并使用广义线性模型等技术进行建模，可以进一步提高准备金评估的有效性。

       512 660 750 .,
       611 700 . .,
       647 . . .};

  f = j(n-1, 1, 0);
  do j = 1 to n-1;
    s1 = 0; s2 = 0;
    do i = 1 to n-j;
      s1 = s1 + C[i, j+1];
      s2 = s2 + C[i, j];
    end;
    f[j] = s1 / s2;
  end;

  /* 累积进展因子 */
  F = j(n, 1, 1);
  do i = 2 to n;
    F[i] = F[i-1] * f[n-i+1];
  end;

  /* B-F法准备金 */
  reserve = j(n, 1, 0);
  do i = 2 to n;
    ult_loss = EP[i] * ELR;
    reserve[i] = ult_loss * (1 - 1/F[i]);
  end;
  total_reserve = sum(reserve);
  print "累积进展因子" F;
  print "B-F法各事故年准备金" reserve;

4.4 B-F法

赔付率或损失率是非寿险精算学的一个重要概念，表示保险公司在特定期间的赔款支出与（已赚）保费收入之比。基于历史数据、保险人经验或市场趋势得到的赔付率可以作为估计最终损失和未决赔款的基础，但由于索赔数据故有的波动性，赔付率估计值本身也具有较大的不确定性。

定义 4.2（Bornhuetter-Ferguson法）

B-F法（Bornhuetter-Ferguson法）的基本思想是将链梯法与赔付率法相结合：通过考虑索赔进展的模式，可以提高赔付率法的评估准确性；同时，赔付率法的结果也可以增加链梯法的稳定性，防止进展模式失真。

该方法的具体计算步骤如下：

(1) 应用链梯法，计算每个事故年已付赔款的累积进展因子$f$；

(2) 基于保费收入和预期赔付率假设，计算每个事故年的最终赔款的初始估计；

(3) 将最终赔款的估计值除以累积进展因子$f$，得到已付赔款的近似结果；

(4) 从最终赔款中减去已付赔款，得到未付赔款（即准备金）的估计结果。

其中，累积进展因子$f$的定义为：

$$\text{已付赔款} \times f = \text{最终赔款}$$

即$1/f=\text{已付赔款}/\text{最终赔款}$，表示已支付赔款的比例；$1-1/f$表示未支付赔款的比例。对上述步骤进行归纳，可以得到B-F法的准备金估计结果为：

$$\text{准备金} = \text{保费} \times \text{预期损失率} \times (1 - 1/f)$$

由于最终的估计结果同时考虑了索赔数据的历史经验（链梯法的累积赔款流量三角形）以及赔付率的先验假设信息，该方法也体现了贝叶斯统计学的思想。

(4) 对于事故年2024，未付赔款的估计值为$867 \times(1-1/1.4462)=267.50$，修正后的最终赔款估计值为$647+267.50=914.50$。

将所有未付赔款的估计值加总，得到准备金为413.82。

B-F法最早应用于已报案赔款进展的分析，然而，该方法也同样适用于已决赔款的进展。在原始计算过程中，累积进展因子$f$由链梯法计算得到，但应用其他方法计算的进展因子或调整因子，也可以得到准备金评估结果。此外，上述例子没有考虑对通货膨胀进行调整，但是可以采用链梯法中介绍的调整方法。

需要说明的是，应用B-F法需要对赔付率给出恰当的估计，且一般而言，也需要假设赔款进展具有与链梯法类似的稳定趋势。然而，B-F法对赔款进展中的异常值相对不敏感。故在处理波动性较大的赔款数据时，该方法常被视为更可靠的选择。

上述结果可通过对公前式变形来说明。由前例~和前例可知，B-F法对最终赔款的估计值也可以表示为：

$$\begin{aligned} \text{B-F法的最终赔款} &= \text{已付赔款} + \text{保费} \times \text{预期损失率} \times (1 - \frac{1}{f}) \\ &= \text{链梯法的最终赔款} \times \frac{1}{f} + \text{赔付率法的最终赔款} \times (1 - \frac{1}{f}) \end{aligned}$$

即两种方法预测结果的加权。

一般而言，链梯法的估计结果受到当前阶段已付赔款的强烈影响，特别对于靠近评估时点的事故年，由于其进展不完全、累积进展因子$f$较大，已付赔款的微小变动也可能导致最终赔款的估计结果发生显著差异。但在B-F法中，当$f$较大时，链梯法的权重下降而赔付率法的权重上升，此时的估计结果更多参考了先验知识的影响；对于靠前的事故年，由于其进展相对完全、累积进展因子$f$较小且接近于1，B-F法的结果基本等同于链梯法。

4.5 随机准备金模型

前述链梯法、案均赔款法、B-F法等准备金评估模型均为确定性方法，仅能给出准备金的（期望）预测值，无法给出其分布特征。因此在近年来，非寿险准备金评估的随机性模型，如Mack模型、广义线性模型方法等，受到了越来越多的关注。

4.5.1 Mack链梯法模型

Mack模型（Mack chain-ladder model）是对链梯法的一种随机性拓展，能够在链梯法假设的基础上，给出未来赔款进展的无偏估计并对其误差进行度量，在非寿险准备金评估中具有重要应用。

设随机变量$C_{i,j}$表示第$i$个事故年在第$j$个进展年的累积赔款（已付赔款、已发生赔款或已报案赔款），其中事故年$i \in \{1,\cdots,I\}$，进展年$j \in \{1,\cdots,J\}$。不失一般性，可以假设$I=J$。进一步假定$C_{i,j}$对于满足$i+j \leq I + 1$的日历年是可观测的随机变量，而对于$i+j>I+1$的日历年需要预测，构成一个流量三角形。通过对$\hat{C}_{i,I}$进行预测，即可计算事故年$i$的未付赔款$\hat{R}_i = \hat{C}_{i,I} - C_{i,I-i+1}$，加总后得到准备金。

定义个体进展因子$F_{i,j} = C_{i,j+1} / C_{i,j}$，其中$1 \leq i,j \leq I-1$。模型包含以下假设：

(1) 存在常数$f_j>0$，使得对所有$1 \leq i \leq I$和$1 \leq j \leq I-1$，有：

$$\mathbb{E}(F_{i,j} \mid C_{i,1},\cdots,C_{i,j}) = f_j$$

其中，参数$f_j$称为损失进展因子或链接比率（link ratios）。

(2) 存在常数$\sigma_j^2 > 0$，使得对所有$1 \leq i \leq I$和$1 \leq j \leq I-1$，有：

$$\text{Var}(F_{i,j} \mid C_{i,1},\cdots,C_{i,j}) = \frac{\sigma_j^2}{w_{i,j} \cdot C_{i,j}^\alpha}$$

其中，$w_{i,j} \in [0,1]$表示$C_{i,j}$的权重，参数$\sigma_j$称为方差参数。

(3) 各事故年的索赔数据$(C_{i,1},\cdots,C_{i,I})_{1 \leq i \leq I}$之间相互独立。

上式和上式也等价于：

$$\begin{gathered}\mathbb{E}(C_{i,j+1}|C_{i,j}) = f_j C_{i,j} \\ \text{Var}(C_{i,j+1}|C_{i,j}) = \frac{\sigma_j^2}{w_{i,j}} C_{i,j}^{2-\alpha}\end{gathered}$$

已知索赔信息$D_{\text{I}} = \{C_{ij}: i+j \leq I+1\}$，Mack模型给出了进展因子和方差参数的估计值，分别为：

$$\begin{gathered}\hat{f}_j = \frac{\sum\limits_{i=1}^{I-j} w_{i,j} C_{i,j}^\alpha F_{i,j}}{\sum\limits_{i=1}^{I-j} w_{i,j} C_{i,j}^\alpha},\ 1 \leq j \leq I-1,\ \alpha \in \{0,1,2\} \\ \hat{\sigma_j^2} = \frac{1}{I_j - 1} \sum\limits_{i=1}^{I-j} w_{i,j} C_{i,j}^\alpha (F_{i,j}-\hat{f}_j)^2,\ 1 \leq j \leq I-2,\ \alpha \in \{0,1,2\}\end{gathered}$$

其中，$I_j$代表权重$w_{i,j}$不为零的观测值（事故年）个数。对于第$I-1$个进展年，考虑到$f_{I-1}$仅能通过单个观测值$C_{1,I}/C_{1,I-1}$计算，无法直接估计$\hat{\sigma}_{I-1}^2$。此时，若$f_{I-1}=1$且假设赔款进展在第$I-1$年结束，则$\hat{\sigma}_{I-1}^2 = 0$；若赔款进展未结束，则可以使用外推公式$\hat{\sigma}_{I-3}/\hat{\sigma}_{I-2} = \hat{\sigma}_{I-2}/\hat{\sigma}_{I-1}$，求得估计值$\hat{\sigma}_{I-1}^2 = \min(\hat{\sigma}_{I-2}^4 / \hat{\sigma}_{I-3}^2, \min(\hat{\sigma}_{I-3}^2,\hat{\sigma}_{I-2}^2))$。

可以证明$\hat{f}_j,\hat{\sigma}_j^2$是对应参数$f_j,\sigma_j^2$的无偏估计量，且$\hat{f}_j$在所有表示为已观察进展因子$F_{i,j}$的加权平均值的无偏估计量中，方差最低。

在Mack模型中，参数$\alpha$决定了$f_j$的估计方法。不妨设权重$w_{i,j} \equiv 1$：

(1) 若$\alpha = 1$，即为经典链梯法对$f_j$的估计结果：

$$\begin{gathered}\hat{f}_j = \frac{\sum\limits_{i=1}^{I-j} C_{i,j} F_{i,j}}{\sum\limits_{i=1}^{I-j} C_{i,j}} = \frac{\sum\limits_{i=1}^{I-j} C_{i,j+1}}{\sum\limits_{i=1}^{I-j} C_{i,j}}\end{gathered}$$

此时有$\text{Var}(C_{i,j+1}|C_{i,j}) = \sigma_j^2 \cdot C_{i,j}$，即估计结果的不确定性会随索赔规模增加。

(2) 当$\alpha = 0$时，估计结果等于个体进展因子$F_{i,j}$的算术平均值：

$$\begin{gathered}\hat{f}_j = \frac{1}{I-j} \sum_{i=1}^{I-j} F_{i,j}\end{gathered}$$

(3) 当$\alpha = 2$时，估计结果是以$C_{i,j+1}$为因变量、以$C_{i,j}$为自变量的最小二乘回归系数（假设截距项等于0）：

$$\begin{gathered}\hat{f}_j = \frac{\sum\limits_{i=1}^{I-j} C_{i,j}^2 F_{i,j}}{\sum\limits_{i=1}^{I-j} C_{i,j}^2} = \frac{\sum\limits_{i=1}^{I-j} C_{i,j} C_{i,j+1}}{\sum\limits_{i=1}^{I-j} C_{i,j}^2}\end{gathered}$$

当$i+j > I+1$时，Mack模型对累积赔款的预测值等于：

$$\begin{gathered}\mathbb{E}(C_{i,j}|D_I) = C_{i,I+1-i} \cdot f_{i,I+1-i} \cdots f_{i,j-1}\end{gathered}$$

4.5.2 广义线性模型方法

广义线性模型方法通常使用增量赔款的流量三角形数据进行建模。记$Z_{ij}$表示保险公司的某项业务在第$i(1\leq i\leq I)$个年度发生的事故在第$j(1\leq j\leq J)$个进展年的增量赔款，是一个随机变量。不失一般性，可以假定$I=J$，即事故年和进展年的期限相等。流量三角形的上三角部分$Z_{o}=\{Z_{ij}:i+j\leq I+1\}$表示增量赔款的观测值，下三角部分$Z_{p}=\{Z_{ij}:i+j>I+1\}$表示待预测的增量赔款，将下三角部分的增量赔款预测值相加，即可得到准备金评估结果。

在对增量赔款数据进行建模前，通常需要将赔款转化为单位增量赔款，以剔除各个事故年的业务规模所造成的影响。记$Y_{ij}=Z_{ij}/\omega _{i}$表示标准化的增量赔款，其中$\omega_{i}$为事故年$i$的风险单位数，通常可以使用当年的已赚保费代替。

与链梯法等确定性模型相比，广义线性模型方法将各年度的标准化增量赔款视作随机变量，且假定其服从指数族中的某一分布，如伽马分布或逆高斯分布等（详见本书第7章）。记$F(\cdot)$表示标准化增量赔款的分布，则相应的准备金评估模型定义为：

$$Y_{ij} \sim F(y_{ij};~ \eta_{ij}, \boldsymbol{\gamma})$$

其中，$\eta_{ij}$和$\boldsymbol{\gamma}$为对应分布的参数。$\eta_{ij}$代表可以由增量赔款的事故年和进展年效应解释的参数，通常为分布的位置参数或形状参数；$\boldsymbol{\gamma}$为除$\eta_{ij}$外，分布的其他所有参数。

将赔款发生的事故年和进展年作为自变量引入模型，参照链梯法的思想，可以得到广义线性模型的结构如下：

$$g(\eta_{ij}) = \eta + \alpha _{i} + \beta _{j},\ 1 \leq i,j \leq I$$

其中，$\alpha_{i}$和$\beta_{j}$分别表示不同事故年和不同进展年的效应（$\alpha _{1} = \beta _{1} = 0$）；$g(\cdot)$为广义线性模型的连接函数。

在非寿险准备金评估中，通常使用的分布假设包括伽马分布和对数正态分布，能够较好刻画增量赔款数据的非负性和右偏特征。但当赔款数据的尾部较厚时，这些分布可能低估尾部风险，导致准备金评估不足。此时需要基于厚尾分布（如帕累托分布等）建立准备金评估模型，以更准确地衡量高额赔款的潜在影响，提高准备金的充足性。本书第11章将给出一个更复杂的案例。
