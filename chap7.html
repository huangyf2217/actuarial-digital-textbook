第7章 广义线性模型

本章学习目标

  1. 认识指数族的定义和性质，并将常见分布表示为指数族的形式。
  2. 掌握广义线性模型的框架及其对传统线性回归模型的拓展，包括分布假设、系统性成分（线性预测子）和连接函数。
  3. 说明广义线性模型的参数估计过程。
  4. 了解偏差和残差的概念，通过残差分析和拟合优度检验选择合适模型。
  5. 能够在R中熟练构建并解释广义线性模型。

7.1 指数族分布

7.1.1 指数族的定义和性质

指数族分布（the exponential family of distributions），或称指数型分布族，是统计和精算研究中最重要的参数分布族之一。其中包含了一系列离散和连续型分布，如二项分布、泊松分布、负二项分布、正态分布、伽马分布、逆高斯分布等。

若随机变量$Y$的分布属于指数族，则其概率密度函数可以写为如下形式：

定义 7.1（指数族分布）
$$f(y;\theta,\phi) = \exp\left[\frac{y\theta-b(\theta)}{a(\phi)}+c(y,\phi)\right]$$

其中，$a(\cdot),b(\cdot),c(\cdot,\cdot)$都是已知函数。

上式中各符号的解释如下：

  • $\theta$称为自然参数或正则参数，与分布的均值有关；
  • $\phi$称为尺度参数或离散参数，与分布的均值无关，仅与方差有关；
  • $a(\phi)>0$且为连续函数，通常的形式为$a(\phi) = \phi/\omega$，其中$\omega$是已知权重；
  • $b(\theta)$应二阶可导，且二阶导函数大于0；
  • $c(y,\phi)$与参数$\theta$无关。

在讨论指数族分布的性质前，首先介绍统计学中的两个著名性质。记$l(\theta,\phi ;y) = \ln f(y;\theta,\phi)$表示对数似然函数（对数概率密度函数），则有：

$$\begin{gathered}\mathbb{E}\left(\frac{\mathrm{d} l}{\mathrm{d} \theta}\right) = 0 \\[0.5em] \mathbb{E}\left(\frac{\mathrm{d}^2 l}{\mathrm{d} \theta^2}\right) + \mathbb{E}\left[\left(\frac{\mathrm{d} l}{\mathrm{d} \theta}\right)^2\right] = 0 \end{gathered}$$

证明：已知$\displaystyle \frac{\mathrm{d} l}{\mathrm{d}\theta} = \frac{\mathrm{d}l}{\mathrm{d}f} \frac{\mathrm{d}f}{\mathrm{d}\theta} = \frac{1}{f} \frac{\mathrm{d}f}{\mathrm{d}\theta}$，可得：

$$\mathbb{E} \left(\frac{\mathrm{d}l}{\mathrm{d}\theta} \right) = \int \left(\frac{1}{f} \frac{\mathrm{d}f}{\mathrm{d}\theta} \right) f \mathrm{d}y = \frac{\mathrm{d}}{\mathrm{d}\theta} \int f \mathrm{d}y = \frac{\mathrm{d}}{\mathrm{d}\theta} 1 = 0$$

根据定义，可计算$\displaystyle \frac{\mathrm{d}^2 l}{\mathrm{d} \theta^2}$等于：

$$\frac{\mathrm{d}^2 l}{\mathrm{d} \theta^2} = \frac{\mathrm{d}}{\mathrm{d}\theta} \left(\frac{1}{f} \frac{\mathrm{d}f}{\mathrm{d}\theta} \right) = \frac{1}{f} \left(\frac{\mathrm{d}^2f}{\mathrm{d}\theta^2} \right) - \frac{1}{f^2} \left(\frac{\mathrm{d}f}{\mathrm{d}\theta} \right)^2 = \frac{1}{f} \left(\frac{\mathrm{d}^2f}{\mathrm{d}\theta^2} \right) - \left(\frac{\mathrm{d} l}{\mathrm{d}\theta}\right)^2$$

对上式两边取期望，有：

$$\mathbb{E}\left(\frac{\mathrm{d}^2 l}{\mathrm{d} \theta^2}\right) + \mathbb{E}\left[\left(\frac{\mathrm{d} l}{\mathrm{d} \theta}\right)^2\right] = \mathbb{E} \left[\frac{1}{f} \left(\frac{\mathrm{d}^2f}{\mathrm{d}\theta^2} \right)\right] = \frac{\mathrm{d}^2}{\mathrm{d}\theta^2} \int f \mathrm{d}y = 0$$

命题得证。 $\square$

在上两式中，$\displaystyle U(\theta) = \frac{\mathrm{d}l}{\mathrm{d}\theta}$称为参数$\theta$的得分函数，$\displaystyle I(\theta) = \mathbb{E}[(\frac{\mathrm{d} l}{\mathrm{d} \theta})^2]$为参数$\theta$的Fisher信息量。上述性质表明，得分函数在参数真实值处的期望等于0，且Fisher信息量也可通过$\displaystyle I(\theta) = - \mathbb{E}(\frac{\mathrm{d}^2 l}{\mathrm{d} \theta^2})$计算。

性质 7.1（指数族分布的矩）

根据上述性质，可证明指数族分布的期望和方差分别为：

$$\begin{gathered}\mathbb{E}(Y) = b'(\theta) \\ \text{Var}(Y) = a(\phi) b''(\theta) \end{gathered}$$

证明：已知$\displaystyle l(\theta,\phi) = \ln f(y;\theta,\phi) = \frac{y\theta-b(\theta)}{a(\phi)}+c(y,\phi)$，则：

$$\frac{\mathrm{d}l}{\mathrm{d}\theta} = \frac{y - b'(\theta)}{a(\phi)},   \frac{\mathrm{d}^2l}{\mathrm{d}\theta^2} = -\frac{b''(\theta)}{a(\phi)}$$

应用上述性质，可得：

$$\begin{gathered}\mathbb{E} \left(\frac{\mathrm{d}l}{\mathrm{d}\theta} \right) = \frac{\mathbb{E}(Y) - b'(\theta)}{a(\phi)} = 0 ~\Rightarrow~ \mathbb{E}(Y) = b'(\theta) \\ \mathbb{E}\left[\left(\frac{\mathrm{d} l}{\mathrm{d} \theta}\right)^2\right] = -\mathbb{E}\left(\frac{\mathrm{d}^2 l}{\mathrm{d} \theta^2}\right) ~\Rightarrow~ \frac{\mathbb{E}[(Y-\mathbb{E}Y)^2]}{a^2(\phi)} = \frac{b''(\theta)}{a(\phi)} ~\Rightarrow~ \text{Var}(Y) = a(\phi) b''(\theta)\end{gathered}$$

命题得证。 $\square$

由上两式可知，指数族分布的期望与参数$\phi$无关，而方差由两部分组成：第一部分为$a(\phi)$，由尺度参数决定；另一部分为$b''(\theta)$，反映了期望与方差之间的函数关系。因此，指数族分布的方差与期望有关，通常可记为$\text{Var}(Y) = a(\phi)V(\mu)$，其中$V(\mu)$称为方差函数。

考虑简化情况$a(\phi) = \phi/\omega$，指数族分布的矩母函数可以表示为：

$$M(t) = \exp\left[\frac{b(\theta + t\phi/\omega) - b(\theta)}{\phi/\omega}\right]$$

证明：根据矩母函数的定义$M(t) = \mathbb{E}(e^{tY})$，有：

$$\begin{aligned}M_Y(t) &= \int \exp(t y) f(y;\theta,\phi) \mathrm{d}y \\ &= \int \exp(t y) \exp\left[\frac{y\theta - b(\theta)}{\phi/\omega} + c(y,\phi)\right] \mathrm{d}y \\ &= \exp\left[\frac{b(\theta + t\phi/\omega) - b(\theta)}{\phi/\omega}\right] \int \exp\left[\frac{y (\theta + t\phi/\omega) - b(\theta + t\phi/\omega)}{\phi/\omega} + c(y,\phi)\right] \mathrm{d}y \\ &= \exp\left[\frac{b(\theta + t\phi/\omega) - b(\theta)}{\phi/\omega}\right]\end{aligned}$$

其中，积分内是参数$(\theta^* = \theta + t\phi/\omega,~\phi^* = \phi)$的指数族分布的概率密度函数，积分结果为1。命题得证。 $\square$

例 7.1

设$Y_1,Y_2,\cdots\,Y_n$是独立同分布，且来自指数族分布的样本。试证明样本均值$\displaystyle \bar{Y} = \frac{1}{n} \sum\limits_{i=1}^{n} Y_i$仍然服从指数族分布。

不妨设$a(\phi) = \phi/\omega$，则$Y/n$的矩母函数为：

$$M_{Y/n}(t) = \mathbb{E}\left(e^{t y/ n}\right) = M_Y \left(\frac{t}{n}\right) = \exp\left\{\frac{b\left[\theta + t \cdot (\phi/n)/\omega\right] - b(\theta)}{\phi/\omega}\right\}$$

则$\displaystyle \bar{Y} = \frac{1}{n} \sum\limits_{i=1}^{n} Y_i$的矩母函数为：

$$M_{\bar{Y}}(t) = \left[M_{Y/n}(t)\right]^n = \exp\left\{\frac{b\left[\theta + t \cdot (\phi/n)/\omega\right] - b(\theta)}{(\phi/n)/\omega}\right\}$$

上式为参数$(\theta^* = \theta,~\phi^* = \phi/n)$的指数族分布的矩母函数。

7.1.2 常见的指数族分布

1. 二项分布

二项分布的概率分布列可以改写为：

$$\begin{aligned} f(y;n,p) &= \binom{n}{y} p^y (1-p)^{n-y} \\ &= \exp\left[y\ln(\frac{p}{1-p})+n\ln(1-p)+\ln\binom{n}{y}\right] \\ \end{aligned}$$

令$\theta = \ln[p/(1-p)],~\phi = a(\phi) = 1,~b(\theta) = n\ln(1+e^\theta),~c(y,\phi) = \ln\binom{n}{y}$，易知二项分布符合指数族的标准形式。

将上述结果代入上两式，可以得到二项分布的期望和方差为：

$$\begin{gathered}\mathbb{E}(Y) = b'(\theta) = \frac{ne^\theta}{1+e^\theta}=np\\ \text{Var}(Y) = a(\phi) b''(\theta)=\frac{ne^\theta}{(1+e^\theta)^2}=np(1-p)\end{gathered}$$

2. 泊松分布

泊松分布的概率分布列可以改写为：

$$f(y;\lambda) = \frac{\lambda^y e^{-\lambda}}{y!} = \exp[-\lambda + y\ln(\lambda) - \ln(y!)]$$

令$\theta = \ln\lambda,~\phi = a(\phi) = 1,~b(\theta) = e^\theta,~c(y,\phi) = -\ln(y!)$，易知泊松分布符合指数族的标准形式。

将上述结果代入上两式，可以得到泊松分布的期望和方差为：

$$\begin{gathered}\mathbb{E}(Y)=b'(\theta)=e^\theta=\lambda \\ \text{Var}(Y)=a(\phi)b''(\theta)=e^\theta=\lambda\end{gathered}$$

由于$a(\phi)=1,~b''(\theta) = b'(\theta)$，故泊松分布的期望和方差相等。

3. 负二项分布

在上式中，令$p=\beta/(1+\beta)$，则负二项分布的概率分布列可以改写为：

$$\begin{aligned} f(y;r,p) & = \binom{y+r-1}{y}p^y(1-p)^{r} \\ &= \exp\left[y\ln{p}+r\ln(1-p)+\ln\binom{y+r-1}{y}\right] \end{aligned}$$

令$\theta = \ln p,~\phi = a(\phi) = 1,~b(\theta) = -r\ln(1-e^\theta),~c(y,\phi) = \ln\binom{y+r-1}{y}$，易知负二项分布符合指数族的标准形式。

将上述结果代入上两式，可以得到负二项分布的期望和方差为：

$$\begin{gathered}\mathbb{E}(Y)=b'(\theta)=\frac{re^\theta}{1-e^\theta}=\frac{rp}{1-p}\\ \text{Var}(Y)=a(\phi)b''(\theta)=\frac{re^\theta}{(1-e^\theta)^2}=\frac{rp}{(1-p)^2}\end{gathered}$$

4. 正态分布

正态分布的概率密度函数可以改写为：

$$\begin{aligned} f(y; \mu ,\sigma^2 ) &= \frac{1}{\sqrt {2\pi \sigma ^2} } \exp\left[\frac{ -(y - \mu)^2}{2\sigma^2}\right] \\ &= \exp\left[-\frac{y^2 - 2y\mu + \mu^2}{2\sigma^2} - \frac{1}{2}\ln(2\pi\sigma^2)\right] \\ &= \exp\left\{\frac{{y\mu - {\mu ^2}/2}}{\sigma^2} - \frac{1}{2}\left[\frac{y^2}{\sigma ^2} + \ln(2\pi \sigma^2)\right]\right\} \end{aligned}$$

令$\theta = \mu,~\phi = \sigma^2,~a(\phi)=\phi,~b(\theta) = \theta^2/2,~c(y,\phi) = -\frac{1}{2}[y^2/\sigma^2+\ln(2\pi\sigma^2)]$，易知正态分布符合指数族的标准形式。

将上述结果代入上两式，可以得到正态分布的期望和方差为：

$$\begin{gathered}\mathbb{E}(Y) = b'(\theta)=\theta=\mu\\ \text{Var}(Y) = a(\phi)b''(\theta)=\phi=\sigma^2\end{gathered}$$

对于正态分布，由于$b''(\theta) = 1$，故方差与期望无关。

5. 伽马分布

伽马分布的概率密度函数可以改写为：

$$\begin{aligned} f(y;\alpha,\lambda) &= \frac{\lambda^\alpha}{\Gamma(\alpha)} y^{\alpha-1} e^{-\lambda y} \\ &\xlongequal{\mu=\alpha/\lambda} \frac{\alpha^\alpha}{\mu^\alpha\Gamma(\alpha)} y^{\alpha-1} e^{-y \alpha/\mu} \\ &= \exp\left[(-\frac{y}{\mu} - \ln \mu)\alpha + (\alpha-1) \ln y + \alpha \ln\alpha - \ln \Gamma(\alpha) \right] \end{aligned}$$

令$\theta = -1/\mu = -\lambda/\alpha,~\phi = \alpha,~ a(\phi)=1/\phi,~b(\theta) = -\ln(-\theta),~c(y,\phi) = (\alpha-1) \ln y + \alpha \ln\alpha - \ln \Gamma(\alpha)$，易知伽马分布符合指数族的标准形式。

将上述结果代入上两式，可以得到伽马分布的期望和方差为：

$$\begin{gathered}\mathbb{E}(Y) = b'(\theta) = -\frac{1}{\theta} = \mu = \frac{\alpha}{\lambda} \\ \text{Var}(Y) = a(\phi)b''(\theta) = \frac{1}{\phi} \cdot \frac{1}{\theta^2} = \frac{\alpha}{\lambda^2}\end{gathered}$$

6. 逆高斯分布

逆高斯分布的概率密度函数可以改写为：

$$\begin{aligned} f(y;\mu,\lambda) &= \sqrt{\frac{\lambda}{2\pi y^3}} \exp\left[-\frac{\lambda(y-\mu)^2}{2\mu^2 y}\right] \\ &\xlongequal{\sigma^2 = 1/\lambda} \exp \left[-\frac{y^2 - 2\mu y + \mu^2}{2\mu^2 \sigma^2 y} - \frac{1}{2}\ln(2\pi y^3 \sigma^2) \right] \\ &= \exp \left[\frac{-y/(2\mu^2) + (1/\mu)}{\sigma^2} - \frac{1}{2\sigma^2 y} - \frac{1}{2}\ln(2\pi y^3 \sigma^2) \right] \end{aligned}$$

令$\theta = -1/(2\mu^2),~\phi = \sigma^2 = 1/\lambda,~ a(\phi)=\phi,~b(\theta) = -\sqrt{-2\theta},~c(y,\phi) = - \frac{1}{2\sigma^2 y} - \frac{1}{2}\ln(2\pi y^3 \sigma^2)$，易知逆高斯分布符合指数族的标准形式。

将上述结果代入上两式，可以得到逆高斯分布的期望和方差为：

$$\begin{gathered}\mathbb{E}(Y) = b'(\theta) = (-2\theta)^{-\frac{1}{2}} = \mu \\ \text{Var}(Y)=a(\phi)b''(\theta) = \phi (-2\theta)^{-\frac{3}{2}} = \frac{\mu^3}{\lambda}\end{gathered}$$

7.2 广义线性建模框架

7.2.1 模型结构

广义线性模型（generalized linear models，GLM）是线性模型的推广和延伸，能够刻画多种类型因变量与自变量之间的灵活回归关系。其核心思想是应用合适的概率分布对因变量进行描述，并通过连接函数建立因变量期望值与自变量线性组合的关联。与线性模型相比，广义线性模型将分布假设拓展至指数族，允许数据服从非正态分布，同时还能在一定程度上捕捉因变量与自变量的非线性关系。

广义线性模型通常被应用于分类费率厘定和风险评估。在精算实务中，预测分析的一个重要目标是确定对风险具有重要影响的因素（风险因子），量化这些因子与风险之间的关系，以确定合适的保费水平。然而，保险损失数据通常不服从正态分布，例如索赔频次属于计数型变量（仅能在非负整数上取值），索赔金额虽然属于连续型变量，但其非负、右偏的特性也与正态分布不符。此时可以使用泊松分布拟合索赔频次，使用伽马分布或逆高斯分布拟合索赔金额，对应的模型即为广义线性模型。

考虑正态分布假设下的传统线性回归模型。记$Y$表示因变量，即建模预测和分析的目标，也称为预测变量或响应变量；$X_1,X_2,\cdots,X_p$是与因变量有关的因子，称为自变量或协变量。传统线性回归模型的形式可以表示为：

$$\left\{ \begin{aligned} &Y_i \sim \text{N}(\mu_i, \sigma^2) \\ &\mu_i = \beta_0 + \beta_1 X_{i,1} + \cdots + \beta_p X_{i,p} \end{aligned} \right.$$

其中，下标$i$表示第$i$个个体；$\boldsymbol{\beta} = (\beta_0,\beta_1,\cdots,\beta_p)^T$为回归系数，描述了自变量对因变量的影响。线性模型假设因变量$Y_i$服从正态分布，各因变量的方差为固定常数，而均值可以表示为自变量的线性组合。当因变量非正态、非连续，方差可能随均值变化，且因变量与自变量存在非线性关系时，线性回归模型的假设将被破坏，无法得到准确的回归分析结果。

广义线性模型对上述不足进行了拓展，其形式可以表示为：

定义 7.2（广义线性模型）
$$\left\{ \begin{aligned} &Y_i \sim \text{EF}(\mu_i, \phi) \\ &g(\mu_i) \triangleq \eta_i = \beta_0 + \beta_1 X_{i,1} + \cdots + \beta_p X_{i,p} \end{aligned} \right.$$

广义线性模型的核心组成部分包括以下三个方面：

(1) 分布假设（随机性成分）：随机样本$Y_i$相互独立且服从指数族中的一个分布，如二项分布、泊松分布、伽马分布等。

(2) 系统性成分$\eta_i$：表示为自变量的线性组合，也称为线性预测子。

(3) 连接函数$g(\cdot)$：单调、可微的函数，将因变量的期望与线性预测子相连，即$g(\mu_i) = \eta_i = \boldsymbol{x}_i^T \boldsymbol{\beta}$。

在实务中，精算师通常需要明确建模使用的因变量和自变量，根据因变量特征选择合适的分布假设，并指定连接函数。在此基础上，即可应用恰当的原则和方法（如最小化预测平方偏差、最大似然等）对广义线性模型的回归系数进行估计，确定适合数据的最优模型结构。后续章节将依次进行介绍。

例 7.2

经典回归模型是广义线性模型的一个特例。在模型()中，当假设因变量服从正态分布（也属于指数族）且连接函数为恒等连接函数$\mu = \eta$时，广义线性模型即退化为传统线性回归模型。

7.2.2 线性预测子

在非寿险业务中，由于个体风险的经验数据较少，通常需要将风险特征相似、潜在损失相近的个体进行归类，对同一风险类别的个体采用相同的费率，称为分类费率厘定。其中用于对个体风险进行分类的变量称为风险因子（费率因子），广义线性模式可以确定对风险具有显著影响的因子，同时解释每个因子对风险损失的边际效应。

以车险业务为例，代表因子如投保人的年龄，为连续型变量，理论上可以直接估计其回归系数。此外，常用的因子还包括许多离散型变量，例如投保人的性别，是具有两种类别（水平）的二分类变量；以及投保车辆的品牌和型号，是具有多个水平的多分类变量。此类变量通常需要处理为哑变量的形式，在选定基准水平后，对其他水平相对基准水平的影响进行估计。在分类费率厘定时，精算师通常也会对连续型变量进行离散化，例如将年龄划分成“18-25岁”、“26-40岁”、“40岁以上”等不同年龄段水平，再用于建模。这一操作可以更好刻画因变量与自变量的非线性关系，使费率更加平滑和具备可解释性。

在回归分析中，控制因子和随机因子是描述自变量性质的两个概念。其中，控制因子是指可以人为控制或观察得到的变量，例如投保人的年龄、性别等，在建模前需明确其水平或范围；而随机因子是指无法被完全解释或控制的因素，例如抽样误差等，主要体现在误差项中。对于控制因子，当水平数量较少且定义较清晰时，可认为每个水平对因变量都具有明确、固定的影响，并单独估计其效应。此时模型对因子的所有可能水平都进行了考察，各水平的效应不随样本而变化，称为固定效应模型。而当因子的水平数量较多、难以全部被控制，或该因子有某些水平未出现在样本中时，可以假设每个水平的效应是随机变量、来源于某个总体分布，应用样本信息对总体（包括未出现的水平）进行推断。此类模型更关注效应的分布特征，无法单独给出每个水平的效应估计，故称为随机效应模型。本章讲述的广义线性模型主要用于处理固定效应，对于随机效应模型，需要考虑拓展的广义线性混合模型（generalized linear mixed models，GLMM）。

为了刻画自变量与因变量（期望）的不同作用模式，广义线性模型的线性预测子可以设为如下三种形式：

(1) 主效应模型，例如：$\eta = \beta_0 + \beta_1 X_1 + \beta_2 X_2$，其中$\beta_1$和$\beta_2$可能为一系列系数向量，代表$X_1$和$X_2$的不同水平的效应。在该模型中，因变量的变化完全由各因子的独立作用决定，因子之间不存在交互关系。

(2) 交互效应模型，例如：$\eta = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \beta_3 (X_1 \times X_2)$，其中$X_1 \times X_2$为交互项，考虑了因子之间可能存在协同或相互影响的情况，因子的效应可能随另一个因子变化而变化。

(3) 嵌套设计模型，例如：$\eta = \beta_0 + \beta_1 X_1 + \beta_2 X_2(X_1)$。嵌套关系是指一个因子的水平（如$X_2$）完全包含在另一个因子的水平（如$X_1$）之内，$X_2$的效应在$X_1$的每个水平下可能不同，故需要区分因子间从属关系对因变量的不同层面影响。

主效应模型简单、直观，适合因子间无关联的情况。譬如，考虑一个包含性别和年龄的广义线性模型，$S=\{0,1\}$分别表示男性和女性，年龄记为$X$。则在不考虑交互效应的情况下，线性预测子$\eta$可以表示为：

$$\eta = \left\{ \begin{aligned} &\beta_0 + \beta_1 X, & S=0\text{(男性)} \\ &\beta_0 + \beta_1 X + \beta_2, & S=1\text{(女性)} \end{aligned} \right.$$

其中，$\beta_0$是截距项，在本例中可以理解为男性在年龄为0时的预测值；$\beta_1$是年龄的主效应，$\beta_0$和$\beta_1$对于男性和女性均相同；$\beta_2$是性别的主效应，表示从“男性”变为“女性”时的因变量变化情况。

若假设年龄和性别存在交互作用，则线性预测子变为：

$$\eta = \left\{ \begin{aligned} &\beta_0 + \beta_1 X, & S=0\text{(男性)} \\ &\beta_0 + \beta_1 X + \beta_2 + \beta_3 X, & S=1\text{(女性)} \end{aligned} \right.$$

其中，$\beta_3 X$是交互效应$\beta_3 (S \times X)$的简略，当$S=0$时为0，当$S=1$时为$\beta_3 X$。因此，$\beta_3$刻画了年龄对因变量的边际影响如何因性别而变化。若$\beta_3>0$，则表明女性的年龄效应$(\beta_1 + \beta_3)$大于男性。

在广义线性模型中，线性预测子还可以考虑多项式、自变量函数等多种形式，此时模型拓展为广义可加模型（generalized additive models，GAM）。

7.2.3 连接函数

在广义线性模型中，正则连接函数（canonical link function）是指使线性预测子等于指数族分布中的自然参数的连接函数，即：

$$\eta = g(\mu) = \theta$$

注意到指数族中$\mu = b'(\theta)$，故正则连接函数应为$b'(\theta)$的反函数。正则连接函数直接将线性预测子与自然参数相关联，在某些情况下可以简化模型参数的估计过程。根据前表，可以得到常见分布的正则连接函数。例如，伯努利分布（$n=1$）的正则连接函数为logit函数$g(\mu) = \ln(\frac{\mu}{1-\mu})$，泊松分布的正则连接函数为对数函数$g(\mu) = \ln(\mu)$，正态分布的正则连接函数为恒等函数$g(\mu) = \mu$。

在实际应用中，正则连接函数并不一定是建模的最优选择，通常还需要考虑连接函数对期望值$\mu$的可能影响。例如，对于服从伽马分布的数据，可以考虑使用倒数连接函数$g(\mu) = 1/\mu$或对数连接函数。但注意到$\mu = g^{-1}(\eta)$，在使用对数连接函数时，分布期望与线性预测子的关系可以表示为$\mu = \exp(\eta)$，即无论线性预测子为正值或负值，模型对伽马随机变量的期望预测值恒为正，符合伽马分布的特性；而倒数连接函数则无法保证这一性质。此时，广义线性模型可以在无约束条件下求解回归系数的估计值，估计效率更高、估计结果更加稳健。

当因变量的期望可以在$(-\infty,+\infty)$上取值时（例如正态分布），通常可使用恒等连接函数；当因变量的期望仅在$(0,1)$区间上取值时（例如伯努利分布），可使用logit连接函数；当因变量的期望恒为正数时（例如泊松分布、伽马分布等），可使用对数连接函数。在使用对数连接函数时，注意到$\mu = \exp(\beta_0 + \beta_1 X_1 + \cdots + \beta_p X_p) = \exp(\beta_0) \times \exp(\beta_1 X_1) \times \cdots \times \exp(\beta_p X_p)$，使自变量的加法关系转化为乘法关系，故十分适合用于分类费率厘定的研究（可以解释为每个因子使风险提高/降低的比例）。为了满足特定的建模目标，有时还需使用复杂的连接函数。

下表总结了部分常见的连接函数，以及对应的$\mu$的范围。

$$\ln(\frac{\pi_i^{*}/e_i}{1-\pi_i^{*}/e_i}) = \eta_i ~~\Rightarrow~~ \pi_i^{*} = e_i \frac{e^{\eta_i}}{1+e^{\eta_i}}$$

对于不完整保单的索赔概率建模问题，还需要根据上式修改连接函数。本书给出了在R软件中对logit连接函数进行调整的示例，调整完成后，只需在glm函数中指定“family = binomial(link = mlogit(exposure))”即可。


mlogit <- function(exposure = 1){
  linkfun <- function(mu){
    eta <- if(any(exposure-mu<=0)) log(mu/abs(mu-exposure)) else log(mu/(exposure-mu))
    eta
  }
  linkinv <- function(eta){
    thresh <- -log(.Machine$double.eps)
    eta <- pmin(thresh,pmax(eta,-thresh))
    exposure * exp(eta) / (1+exp(eta))
  }
  mu.eta <- function(eta){
    thresh <- -log(.Machine$double.eps)
    res <- rep(.Machine$double.eps,length(eta))
    res[abs(eta)<thresh] <- (exposure * exp(eta) / (1 + exp(eta))^2)[abs(eta) < thresh]
    res
  }
  valideta <- function(eta) TRUE
  link <- paste("logexp(",exposure,")",sep="") 
  structure(list(linkfun=linkfun,linkinv=linkinv,mu.eta=mu.eta,valideta=valideta,name=link), class="link-glm")
}

2. 索赔次数的模型：泊松回归模型

定义$N_i=\{0,1,2,\cdots\}$表示第$i$张保单在一个完整保单年度内的索赔次数，通常假设其服从泊松分布，期望索赔次数记为$\lambda_i$。此时可以使用对数连接函数，构建如下广义线性模型，称为泊松回归模型：

$$\left\{ \begin{aligned} &Y_i \sim \text{Poi}(\lambda_i) \\ &\ln(\lambda_i) = \eta_i = \beta_0 + \beta_1 X_{i,1} + \cdots + \beta_p X_{i,p} \end{aligned} \right.$$

在给定风险因子$\boldsymbol{X}_i$的条件下，第$i$张保单的期望索赔次数可以表示为$\lambda_i = \exp(\eta_i)$。

当存在风险暴露数$0

$$\ln(\lambda_i^{*}/e_i) = \eta_i ~~\Rightarrow~~ \ln(\lambda_i^{*}) = \ln(e_i) + \eta_i$$

其中，$\ln(e_i)$可称为抵消项（offset term），等价于回归方程中的解释变量、回归系数恒等于1。在R软件的glm函数中，直接指定“offset = log(exposure)”即可。

3. 索赔金额（索赔强度）的模型：伽马回归模型

定义$Y_i$表示第$i$次索赔的索赔金额，是取值非负的连续型变量。通常假设其服从伽马分布，期望索赔金额记为$\mu_i$。此时可以使用对数连接函数，构建如下广义线性模型，称为伽马回归模型：

$$\left\{ \begin{aligned} &Y_i \sim \text{Ga}(\mu_i, \phi) \\ &\ln(\mu_i) = \eta_i = \beta_0 + \beta_1 X_{i,1} + \cdots + \beta_p X_{i,p} \end{aligned} \right.$$

其中，伽马分布一般使用如下的重参数化形式：

$$p(y; \mu, \phi) = \frac{y^{(1/\phi^2-1)} \cdot \exp[-y/(\mu \phi^2)]}{(\mu \phi^2)^{(1/\phi^2)} \Gamma(1/\phi^2)}$$

在给定风险因子$\boldsymbol{X}_i$的条件下，第$i$次索赔的期望索赔金额可以表示为$\mu_i = \exp(\eta_i)$。

假设$S_i$为第$i$张保单的累积索赔金额，每张保单总共发生$n_i$次索赔，但每次索赔的金额未知，仅知道$Y_i = S_i/n_i$表示平均索赔金额（称为索赔强度）。此时，既可以采用与风险暴露数调整类似的方法，使用$S_i$作为因变量、$\ln(n_i)$作为抵消项，构建伽马回归模型；也可以使用$Y_i$作为因变量，但注意到数据中相当于对每个$Y_i$进行了$n_i$次重复观测，在建模时需要将$n_i$作为数据权重（weights），即$Y_i \sim \text{Ga}(\mu_i, \phi/n_i)$。

7.3 参数估计

7.3.1 极大似然估计

广义线性模型中的参数通常采用极大似然法进行估计。给定对应分布假设的对数概率密度函数$\ln f(y; \theta, \phi)$，在广义线性模型中，自然参数$\theta$可以表示成线性预测子的函数。因此，模型的联合对数似然函数可以写为（本节统一记$a(\phi) = \phi/\omega$）：

$$l = \sum_{i=1}^n \left[\frac{y_i \theta_i - b(\theta_i)}{\phi/\omega_i} + c(y_i,\phi) \right]$$

回归系数$\beta_j$的极大似然估计是下列方程组的解：

$$\frac{\mathrm{d}l}{\mathrm{d} \beta_j} = \sum\limits_{i = 1}^n {\frac{\omega_i (y_i - \mu_i) x_{i,j}}{\phi V(\mu _i ){g}'(\mu _i )}} = 0$$

其中，$\mu_i = g^{- 1}({\beta_0 + \beta_1 x_{i,1} + ... + \beta_p x_{i,p}})$。

证明：应用链式法则，上述导数可分解为：

$$\frac{\mathrm{d} l_i}{\mathrm{d} \beta_j} = \frac{\mathrm{d} l_i}{\mathrm{d} \theta_i} \frac{\mathrm{d} \theta_i}{\mathrm{d} \mu_i} \frac{\mathrm{d} \mu_i}{\mathrm{d} \eta_i} \frac{\mathrm{d} \eta_i}{\mathrm{d} \beta_j}$$

其中，各项导数分别为：

$$\left\{ \begin{aligned} &\frac{\mathrm{d} l_i}{\mathrm{d} \theta_i} = \frac{y_i - b'({\theta_i})}{\phi/\omega_i} = \frac{\omega_i(y_i - \mu_i)}{\phi} \\ &\frac{\mathrm{d} \theta_i}{\mathrm{d} \mu_i} = ({\frac{\mathrm{d} \mu_i}{\mathrm{d} \theta_i}})^{-1} = [{b''({\theta_i})}]^{-1} = [{V(\mu_i)}]^{-1} \\ &\frac{\mathrm{d} \mu_i}{\mathrm{d} \eta_i} = [g'(\mu_i)]^{-1} \\ &\frac{\mathrm{d} \eta_i}{\mathrm{d} \beta_j} = x_{i,j} \end{aligned} \right.$$

代入可得上式的结果，命题得证。 $\square$

特别当使用正则连接函数$g({\mu_i}) = \theta_i$时，有$g'({\mu_i}) =\frac{\mathrm{d} \theta_i}{\mathrm{d} \mu_i} = [{b''({\theta_i})}]^{-1} = [{V({\mu_i})}]^{-1}$，代入并化简可得：

$$\sum\limits_{i = 1}^n {\frac{\omega_i (y_i - \mu_i)x_{i,j}}{\phi} = 0} ~~\Rightarrow~~ \sum\limits_{i = 1}^n {\omega_i (y_i - \mu_i)x_{i,j} = 0}$$

即在正则连接函数下，极大似然估计的结果应使观测值的加权和等于拟合值的加权和。

例 7.3

假设某医疗保险的索赔额服从均值为$\mu_i$的指数分布：

$$f(y_i) = \frac{1}{\mu_i} e^{-y_i/\mu_i} = \exp \left(-\frac{y_i}{\mu_i} - \ln \mu_i\right)$$

保险人认为年龄会影响索赔额的大小：$\eta_i = \alpha + \beta x_i$。使用正则连接函数和随机样本$\{(x_1,y_1),\cdots,(x_n,y_n)\}$，求解$\alpha$和$\beta$的极大似然估计值。

首先，列出对数似然函数$\displaystyle l = -\sum_{i=1}^n \frac{y_i}{\mu_i} - \sum_{i=1}^n \ln \mu_i$。

易知指数分布的正则连接函数为倒数函数$g(\mu_i) = \frac{1}{\mu_i}$，令其与线性预测子相等$1/\mu_i = \eta_i$，则对数似然函数可以写成关于$\alpha$和$\beta$的函数:

$$\begin{aligned}l &= -\sum_{i=1}^n y_i \eta_i + \sum_{i=1}^n \ln \eta_i \\ &= -\sum_{i=1}^n y_i (\alpha + \beta x_i) + \sum_{i=1}^n \ln(\alpha + \beta x_i)\end{aligned}$$

分别对 $\alpha$ 和 $\beta$ 求导，可得极大似然估计值应满足如下方程组：

$$\left\{ \begin{aligned} &-\sum_{i=1}^n y_i + \sum_{i=1}^n \frac{1}{\alpha + \beta x_i} = 0 \\ &-\sum_{i=1}^n x_i y_i + \sum_{i=1}^n \frac{x_i}{\alpha + \beta x_i} = 0 \end{aligned} \right.$$

根据极大似然估计的相关性质，可以得到对参数估计的标准误的近似结果。然而，在没有计算机辅助的情况下，估计结果可能较难计算。

7.3.2 迭代加权最小二乘法

对于回归系数$\boldsymbol{\beta} = (\beta_0,\beta_1,\cdots,\beta_p)^T$，可定义得分函数$U(\boldsymbol{\beta}) = (U_0, U_1, \cdots, U_p)^T$，其中每个分量表示对数似然函数$l$关于参数$\beta_j$的偏导数。上式表明，回归系数的最大似然估计是令得分函数等于0的解。

定义 7.4（Fisher信息矩阵与Hessian矩阵）

定义$\displaystyle \boldsymbol{I}(\boldsymbol{\beta}) = \mathbb{E}[(\frac{\mathrm{d}l}{\mathrm{d} \boldsymbol{\beta}})^2]$表示回归系数的Fisher信息矩阵。若对数似然函数关于回归系数二阶可导，则$\boldsymbol{I}(\boldsymbol{\beta})$也可以表示为对数似然关于$\boldsymbol{\beta}$的二阶导数的负期望，即$\displaystyle \boldsymbol{I}(\boldsymbol{\beta}) = -\mathbb{E}[\frac{\mathrm{d}^2 l}{\mathrm{d} \boldsymbol{\beta}^2}]$。其中，$\displaystyle \boldsymbol{H}(\boldsymbol{\beta}) = \frac{\mathrm{d}^2 l}{\mathrm{d} \boldsymbol{\beta}^2} = [\frac{\mathrm{d}^2l}{\mathrm{d}\beta_j \mathrm{d}\beta_k}]_{jk}$称为Hessian矩阵。

在广义线性模型中，回归系数的Fisher信息矩阵可以表示为：

$$\begin{gathered}\boldsymbol{I}(\boldsymbol{\beta}) = \boldsymbol{X}^T \boldsymbol{W} \boldsymbol{X} \\ I_{jk}(\boldsymbol{\beta}) = \sum_{i=1}^n \frac{\omega_i x_{i,j} x_{i,k} }{\phi V(\mu _i ) [g'(\mu_i)]^2}\end{gathered}$$

其中，$I_{jk}(\boldsymbol{\beta})$表示信息矩阵中第$j$行、第$k$列的元素，$\boldsymbol{X}$和$\boldsymbol{W}$的定义为：

$$\boldsymbol{X} = \begin{bmatrix} 1 & {x_{1,1} } & \cdots & {x_{1,p} } \\ 1 & {x_{2,1} } & \cdots & {x_{2,p} } \\ \vdots & \vdots & \cdots & \vdots \\ 1 & {x_{n,1} } & \cdots & {x_{n,p} } \\ \end{bmatrix},    \boldsymbol{W} = \text{diag}\left[\frac{\omega_i}{\phi V(\mu_i) [g'(\mu_i)]^2} \right]_{n \times n}$$

分别为模型的设计矩阵和权重矩阵。

证明：根据定义，$\displaystyle I_{jk}(\boldsymbol{\beta}) = \mathbb{E}(\frac{\mathrm{d}l}{\mathrm{d}\beta_j} \frac{\mathrm{d}l}{\mathrm{d}\beta_k}) = \sum_{i=1}^n \mathbb{E}(\frac{\mathrm{d}l_i}{\mathrm{d}\beta_j} \frac{\mathrm{d}l_i}{\mathrm{d}\beta_k})$，其中：

$$\frac{\mathrm{d}l_i}{\mathrm{d} \beta_j} = {\frac{\omega_i (y_i - \mu_i) x_{i,j}}{\phi V(\mu _i ){g}'(\mu _i )}},   \mathbb{E}(\frac{\mathrm{d}l_i}{\mathrm{d}\beta_j} \frac{\mathrm{d}l_i}{\mathrm{d}\beta_k}) = \mathbb{E} \left[\frac{\omega_i^2 (y_i-\mu_i)^2 x_{i,j} x_{i,k}}{\phi^2 [V(\mu_i)]^2 [g'(\mu_i)]^2} \right]$$

注意到$\mathbb{E}[(y_i-\mu_i)^2] = \text{Var}(Y_i) = \phi V(\mu_i)/\omega_i$，代入上式得到：

$$\mathbb{E}(\frac{\mathrm{d}l_i}{\mathrm{d}\beta_j} \frac{\mathrm{d}l_i}{\mathrm{d}\beta_k}) = \frac{\omega_i x_{i,j} x_{i,k}}{\phi V(\mu_i) [g'(\mu_i)]^2},   I_{jk}(\boldsymbol{\beta}) = \sum_{i=1}^n \frac{\omega_i x_{i,j} x_{i,k} }{\phi V(\mu _i ) [g'(\mu_i)]^2}$$

命题得证。 $\square$

当自变量的维度较高、模型参数较多时，似然方程组()的解可能难以直接求出。此时可以考虑应用数值优化算法，通过迭代方式逐步更新参数值。Fisher-Scoring算法是其中一种经典的优化算法，其迭代求解公式为：

$$\boldsymbol{\beta}^{(m+1)} \approx \boldsymbol{\beta}^{(m)} + [\boldsymbol{I}(\boldsymbol{\beta}^{(m)})]^{-1} U(\boldsymbol{\beta}^{(m)})$$

其中，上标$(m)$表示第$m$次迭代的参数估计结果。从初始值$\boldsymbol{\beta}^{(0)}$开始，不断计算得分函数和Fisher信息矩阵并对结果进行更新，即可得到稳定的估计结果。

对于广义线性模型，Fisher-Scoring算法的迭代求解公式可以写成如下形式：

$$\boldsymbol{\beta}^{(m+1)} \approx (\boldsymbol{X}^T \boldsymbol{W}^{(m)} \boldsymbol{X})^{-1} \boldsymbol{X}^T \boldsymbol{W}^{(m)} \boldsymbol{z}^{(m)}$$

其中，$\boldsymbol{z}^{(m)} = \boldsymbol{X} \boldsymbol{\beta}^{(m)} + \boldsymbol{G}^{(m)} (\boldsymbol{y} - \boldsymbol{\mu}^{(m)})$，$\boldsymbol{G} = \text{diag}[g'(\mu_1), \cdots, g'(\mu_n)]_{n \times n}$。

证明：首先考察得分函数$U(\boldsymbol{\beta})$，其中每个元素$U_j = \frac{\mathrm{d}l}{\mathrm{d}\beta_j} = \sum\limits_{i=1}^n \frac{\omega_i (y_i-\mu_i) x_{i,j}}{\phi V(\mu_i) g'(\mu_i)}$。

记$\boldsymbol{y} = (y_1,\cdots,y_n)^T,\ \boldsymbol{\mu} = (\mu_1,\cdots,\mu_n)^T,\ \boldsymbol{G} = \text{diag}[g'(\mu_1), \cdots, g'(\mu_n)]_{n \times n}$，则得分函数$U(\boldsymbol{\beta}^{(m)})$可以表示为：

$$U(\boldsymbol{\beta}^{(m)}) = \boldsymbol{X}^T \boldsymbol{W}^{(m)} \boldsymbol{G}^{(m)} (\boldsymbol{y} - \boldsymbol{\mu}^{(m)})$$

将信息矩阵$\boldsymbol{I}(\boldsymbol{\beta}) = \boldsymbol{X}^T \boldsymbol{W} \boldsymbol{X}$和得分函数代入()，即可得到：

$$\boldsymbol{\beta}^{(m+1)} \approx (\boldsymbol{X}^T \boldsymbol{W}^{(m)} \boldsymbol{X})^{-1} \boldsymbol{X}^T \boldsymbol{W}^{(m)} [\boldsymbol{X} \boldsymbol{\beta}^{(m)} + \boldsymbol{G}^{(m)} (\boldsymbol{y} - \boldsymbol{\mu}^{(m)})]$$

令$\boldsymbol{z}^{(m)} = \boldsymbol{X} \boldsymbol{\beta}^{(m)} + \boldsymbol{G}^{(m)} (\boldsymbol{y} - \boldsymbol{\mu}^{(m)})$，命题得证。 $\square$

在上式中，$\boldsymbol{z}^{(m)}$为工作变量（working response）的向量，其中第$i$个元素$z_i = \eta_i + g'(\mu_i) (y_i - \mu_i)$是线性预测子与残差项$(y_i-\mu_i)$的线性组合。注意到上述估计值等价于优化问题$\arg\min~ Q(\boldsymbol{\beta})= (\boldsymbol{z}^{(m)} - \boldsymbol{X} \boldsymbol{\beta})^T \boldsymbol{W}^{(m)} (\boldsymbol{z}^{(m)} - \boldsymbol{X} \boldsymbol{\beta})$的解，即以$\boldsymbol{z}^{(m)}$为因变量的加权最小二乘估计问题，因此，该方法又称为迭代加权最小二乘法（iteratively weighted least squares，IWLS）。

当使用对数连接函数时，上述迭代公式可以写为：

$$\boldsymbol{\beta}^{(m+1)} \approx \boldsymbol{\beta}^{(m)} + (\boldsymbol{X}^T \boldsymbol{W}^{(m)} \boldsymbol{X})^{-1} \boldsymbol{X}^T \boldsymbol{W}^{(m)} \boldsymbol{k}^{(m)}$$

其中，$k_i^{(m)} = (y_i - \mu_i^{(m)})/\mu_i^{(m)}$，是工作变量向量$\boldsymbol{k}^{(m)}$中的第$i$个元素。

7.3.3 尺度参数的估计

1. 极大似然法

由于指数族分布的期望仅与参数$\theta$有关、与参数$\phi$无关，即$\phi$的取值不会影响回归系数的极大似然估计结果，可以单独估计。事实上，在似然方程组()中同时消去参数$\phi$，参数$\beta_j$的估计值依然相同。

因此，尺度参数$\phi$的极大似然估计等于对数似然方程$\displaystyle \frac{\mathrm{d}l}{\mathrm{d}\phi} = 0$的解。在不同分布假设下，极大似然法的估计结果没有统一形式，需要具体分析。

2. 矩估计法

矩估计法求解尺度参数较为简单，且具有良好的性质。假设因变量$Y_i$服从指数族分布，根据方差上式，可以建立如下关系：

$$\text{Var}[Y_i] = \frac{\phi V(\mu_i)}{\omega_i} ~~\Rightarrow~~ \phi = \frac{\text{Var}[Y_i]\omega_i }{V(\mu_i)}$$

应用样本进行替换，可以得到$\phi$的矩估计为：

$$\hat{\phi} = \frac{1}{n - p - 1} \sum\limits_{i = 1}^n {\frac{\omega_i (y_i - \hat{\mu}_i)^2}{V(\hat {\mu}_i)}}$$

其中，分母是考虑回归系数的$p+1$个估计量后，对自由度进行了调整。

可以证明矩估计是尺度参数的渐进无偏估计。

证明：定义Pearson卡方统计量$\chi^2$和尺度调整Pearson卡方统计量$\chi^2/\phi$如下：

$$\begin{gathered}\chi^2 = \sum_{i=1}^n \frac{\omega_i (y_i - \hat{\mu}_i)^2}{V(\hat{\mu}_i)} \\ \chi^2/\phi = \sum_{i=1}^n \frac{\omega_i (y_i - \hat{\mu}_i)^2}{\phi V(\hat{\mu}_i)}\end{gathered}$$

则$\phi$的矩估计量可以写为$\hat{\phi} = \chi^2/(n-p-1)$。

由于尺度调整Pearson卡方统计量渐进服从自由度为$n-p-1$的卡方分布，应用卡方分布的期望性质，可求得：

$$\mathbb{E}(\chi^2 /\phi) \approx n-p-1 ~~\Rightarrow~~ \mathbb{E}(\hat{\phi}) = \frac{\mathbb{E}(\chi^2)}{n-p-1} \approx \phi$$

命题得证。 $\square$

在估计尺度参数时，通常建议使用未经汇总的原始数据。这是由于数据汇总会减少观测样本的数量$n$，从而降低自由度，影响尺度参数估计结果的稳定性。此外，汇总数据由于丢失了原始数据中的变异信息，可能导致估计结果不够精确。然而，数据汇总不会对回归系数的估计结果产生影响，因为其主要考察观测值的总体趋势。

7.3.4 回归系数估计的标准误

由极大似然方法的理论可知，估计量$\hat{\boldsymbol{\beta}}_{mle}$的渐进方差等于信息矩阵的倒数，即$\boldsymbol{\Sigma} = \text{Var}(\hat{\boldsymbol{\beta}}_{mle}) = [\boldsymbol{I}(\boldsymbol{\beta})]^{-1}$，其中的对角线元素$\Sigma_{jj}(j=0,1,\cdots,p)$为参数估计值$\hat{\beta}_j$的方差，$\sqrt{\Sigma_{jj}}$为估计值的标准差。已知广义线性模型中$\boldsymbol{I}(\boldsymbol{\beta}) = \boldsymbol{X}^T \boldsymbol{W} \boldsymbol{X}$，故可以应用理论信息矩阵，直接推导估计量的协方差矩阵。

理论信息矩阵不依赖观测数据，可以基于模型分布假设和权重矩阵直接计算。但当样本分布偏离模型假设时，有可能低估参数估计值的误差。此时，可以使用观测的信息矩阵或Hessian矩阵来替代。根据定义，观测的Hessian矩阵是基于实际数据计算的负对数似然函数的二阶导数，即$\displaystyle H(\boldsymbol{\beta}) = [\frac{\mathrm{d}^2l}{\mathrm{d}\beta_j \mathrm{d}\beta_k}]_{jk}$，则$I^{*}(\boldsymbol{\beta}) = -H(\boldsymbol{\beta})$代表观测的信息矩阵。因此，估计量的协方差矩阵也可以写为$\Sigma = -[H(\boldsymbol{\beta})]^{-1}$。通常情况下，应用观测值计算的结果对数据偏差更敏感，能够反映模型中未捕获的变异。

7.4 模型诊断与评估

在得到回归系数的估计结果后，将连接函数的反函数应用于线性预测子，即可计算出因变量的预测值。例如当使用对数连接函数时，预测值$\hat{\mu}_i = \exp(\eta_i)$。此时，可以进一步通过残差图和显著性检验等方式，对分布假设的合理性、参数估计的准确性和模型拟合效果进行验证。理论上讲，广义线性模型的残差应当呈现随机性；若残差具有某种模式，则表明模型可能遗漏了部分与因变量相关的重要因素，或需要采用其他模型。对于该问题，通常可以绘制残差图，对样本残差的形状和离群值进行检验。

参数显著性检验的过程同样重要，目的是评估模型中的每个参数是否对因变量具有显著影响。通常情况下，回归系数显著的因子对模型具有重要贡献，应优先保留在模型中。而对于回归系数不显著的因子，如果其对模型拟合效果的影响有限，则可以在简化模型时予以排除；若其对模型的解释性或预测能力至关重要，则应基于实际问题背景和专业知识，将其保留在模型中。

尽管统计推断理论提供了重要的分析工具，但在寻找最佳模型的过程中，不需要过分拘泥于具体的检验结果。残差、偏差和显著性分析仅是模型选择的参考，而不是绝对的规则。结合实际问题的背景和数据特性，灵活运用这些方法，可以更有效地构建适合的广义线性模型。

7.4.1 偏差分析

偏差（deviance）是广义线性模型中的重要概念，用于评估不同模型对数据的整体拟合效果。为了理解偏差的概念，首先需要介绍饱和模型和空模型：

(1) 饱和模型（saturated model）：理想化的模型，能够完美拟合所有样本的因变量观测值。该模型相当于为每个样本都赋予一个参数，故参数个数等于观测值的数量$n$，模型的自由度为0。饱和模型的对数似然达到了所有可能模型中的最大值，因此常作为模型比较的基准，用于评估其他模型的拟合效果。 (2) 空模型（null model）：最简单的模型，不考虑任何自变量，仅使用所有样本的因变量平均值进行预测。

应用上述概念，可以定义一个模型的偏差（残差偏差）等于：

$$\begin{aligned} D &= -2 \times (\text{当前模型的对数似然}-\text{饱和模型的对数似然}) \\ &= 2[l(\hat{\boldsymbol{\beta}}_{max}) - l(\hat{\boldsymbol{\beta}})] \end{aligned}$$

其中，$\hat{\boldsymbol{\beta}}$是当前模型的回归系数估计值。偏差越小，则当前模型越接近饱和模型，对数据的拟合效果越好。

例 7.4

假设有一组样本~$(\boldsymbol{x}_i, y_i),~i=1,2,\cdots,n$，因变量~$Y_i \sim \text{Poi}(\lambda_i)$。广义线性模型给出了对样本因变量的期望估计值$\hat{\lambda}_i$~。

(1) 饱和模型：对每个~$i$，显然当~$\lambda_i = y_i$~时，模型“完美”、对数似然函数达到最大。此时饱和模型的对数似然值等于~$\sum\limits_{i=1}^n \ln{p(y_i; \lambda = y_i)}$。

(2) 当前模型：对数似然值等于~$\sum\limits_{i=1}^n \ln{p(y_i; \lambda = \hat{\lambda}_i)}$。

代入泊松分布的概率分布列，可以得到当前模型的偏差为：

$$\begin{aligned}&d(\hat{\lambda}_i, y_i) = \left\{ \begin{aligned} & 2 \hat{\lambda}_i, &\text{if}~~ y_i = 0, \\ & 2 [y_i \ln(y_i / \hat{\lambda}_i) - y_i + \hat{\lambda}_i], &\text{if}~~ y_i > 0. \end{aligned} \right. \\[0.5em] &D(y_i, \hat{\lambda}_i) = \sum_{i=1}^n \omega_i d(y_i, \hat{\lambda}_i)\end{aligned}$$
例 7.5

对于指数族分布$f(y;\mu,\phi)$，当使用正则连接函数$g(\mu)=\theta$时，偏差可以表示为：

$$\begin{aligned} D &= 2 \sum_{i=1}^n \left[l(\mu=y_i, \phi; y_i) - l(\mu = \hat{\mu}_i, \phi; y_i)\right] \\ &= 2 \sum_{i=1}^n \left\{\frac{y_i g(y_i) - b(g(y_i))}{\phi/\omega_i} - \frac{y_i g(\hat{\mu}_i) - b(g(\hat{\mu}_i))}{\phi/\omega_i} \right\} \\ &= \frac{2}{\phi} \sum_{i=1}^n \omega_i \left\{y_i [g(y_i) - g(\hat{\mu}_i)] - b(g(y_i)) + b(g(\hat{\mu}_i)) \right\} \end{aligned}$$

考虑正态分布，$\theta = \mu,~ b(\theta) = \theta^2/2,~ \phi=\sigma^2$，正则连接为恒等连接函数，则：

$$\begin{aligned} D &= \frac{2}{\phi} \sum_{i=1}^n \omega_i \{y_i [g(y_i) - g(\hat{\mu}_i)] - b(g(y_i)) + b(g(\hat{\mu}_i)) \} \\ &= \frac{2}{\sigma^2} \sum_{i=1}^n \omega_i [y_i (y_i - \hat{\mu}_i) - y_i^2/2 + \hat{\mu}_i^2/2] \\ &= \frac{1}{\sigma^2} \sum_{i=1}^n \omega_i (y_i - \hat{\mu}_i)^2 \end{aligned}$$

即正态分布下的模型偏差等价于残差平方和。

尺度调整偏差是在残差偏差的基础上，进一步考虑尺度参数$\phi$的影响，定义为：

$$D^{*} = D/\phi$$

尺度调整偏差对残差偏差进行了标准化，因而更适用于具有不同尺度参数的分布。

当模型假设正确、尺度参数$\phi$已知或被正确估计、且样本量$n$足够大时，尺度调整偏差近似服从自由度为$n-p-1$的卡方分布，即$D^{*} \sim \chi^2(n-p-1)$。应用该性质，可以推出另一种基于偏差的尺度参数估计方法：

$$\hat{\phi} = \frac{D}{n - p - 1}$$

证明：由于$D^{*} \sim \chi^2(n-p-1)$，计算卡方分布的期望，有：

$$\begin{gathered}\mathbb{E}(D^*) \approx n - p - 1,\ \mathbb{E}(D) = \mathbb{E}(\phi D^*) \approx (n - p - 1) \phi \\[0.5em] \mathbb{E}(\hat{\phi}) = \frac{\mathbb{E}(D)}{n - p - 1} \approx \frac{(n - p - 1) \phi}{n - p - 1} = \phi\end{gathered}$$

即上式给出的估计量是尺度参数$\phi$的无偏估计。 $\square$

7.4.2 残差分析

残差（residuals）主要关注单个样本的拟合误差。在广义线性模型中，常用的残差包括原始残差、Pearson残差和偏差残差。

原始残差是因变量的真实值$y_i$与模型拟合值$\hat{\mu}_i$间的差值，即$r_i = y_i - \hat{\mu}_i$。该残差衡量了真实观测值与预测值之间“误差”或“偏离程度”，较小的残差通常表明模型具有更好的拟合效果。但在广义线性模型中，由于$Y_i$的方差通常并非常数、而是依赖于均值$\mu_i$，直接使用原始残差可能受到不同样本观测值的方差影响。此时可以进一步引入Pearson残差，定义为：

$$\begin{gathered}r_i^P = \frac{y_i - \hat{\mu}_i}{\sqrt{V(\hat{\mu}_i)}}\end{gathered}$$

Pearson残差对原始残差进行了标准化，以消除不同方差的影响。同时，对Pearson残差进行标准化，可以得到：

$$\begin{gathered}r_i^{StdP} = \frac{r_i^p}{\sqrt{\hat{\phi} (1-\hat{h}_{ii})}}\end{gathered}$$

其中，$\hat{h}_{ii}$代表第$i$个观测值的杠杆值，是帽子矩阵$\hat{H}$对角线上的第$i$个元素；帽子矩阵的定义为：

$$\begin{gathered}\hat{H} = \hat{W}^{1/2} X (X^T \hat{W} X)^{-1} X^T \hat{W}^{1/2} \\ \hat{W} = \text{diag} \left[\frac{\omega_i}{\phi V(\hat{\mu}_i) [g'(\hat{\mu}_i)]^2}\right]_{n \times n}\end{gathered}$$

在理想情况下，标准化Pearson残差的均值为0、方差接近于1。因此，可以观察其绝对值，找出与预测结果具有明显偏离的样本观测值（可能为异常值）。

偏差残差（deviance residuals）是一种借助对数似然函数，衡量每个样本的观测值与拟合值的相对偏离程度的指标。其定义为：

$$r_i^D = \text{sign}(y_i - \hat {\mu}_i) \times \sqrt{D_i} = \text{sign}(y_i - \hat {\mu}_i) \sqrt{2[l(y_i,\phi;y_i) - l(\hat{\mu}_i,\phi;y_i)]}$$

其中$\text{sign}(\cdot)$表示符号函数，$D_i$表示每个样本对应的残差偏差。

理想情况下的deviance残差应近似服从正态分布。对于来自正态分布的数据（因变量），Pearson残差的结果等价于deviance残差；但对于非正态数据，Pearson残差的分布通常是有偏的，不再具有正态近似。实际应用中，可以对deviance残差做进一步调整，使其更加近似于正态分布的性质，即：

$$\begin{gathered}r_i^{D_a} = r_i^{D} + \frac{1}{6} \mathbb{E} \left[\frac{Y_i - \hat{\mu_i}}{\sqrt{\text{Var}(Y_i)}}\right]\end{gathered}$$

近年来，还有学者提出了一种新的残差，称为分位残差（quantile residuals）。该残差同样考虑了预测分布的整体性质，定义为：

$$r_i^Q = \left\{ \begin{aligned} & \Phi^{-1}(F(y_i; \hat{\mu}_i, \phi)), &Y_i~\text{为连续型}, \\ & \Phi^{-1}(u_i),~ u_i \sim \text{U} \left[\lim_{y \to y_i^{-}} F(y_i; \hat{\mu}_i, \phi), F(y_i; \hat{\mu}_i, \phi) \right], &Y_i~\text{为离散型}. \end{aligned} \right.$$

其中，$F(\cdot)$是因变量对应的分布函数，$\Phi(\cdot)$是标准正态的分布函数。当$Y_i$为离散型变量时，计算分位残差的$u_i$是从均匀分布中抽取的随机数，故又称为随机分位残差。

理想的分位残差应近似服从标准正态分布。因此，可以绘制样本分位残差的Q-Q图，通过残差与标准正态分布的贴合程度来评估模型。

7.4.3 模型选择和比较

模型选择和比较是统计建模过程中的关键步骤，目的在于从多个候选模型中找出最适合数据的模型。同时，模型选择还可以帮助筛选对预测结果最具贡献的自变量，从而简化模型、提高其可解释性和稳定性。对于嵌套模型，通常可使用基于偏差的$\chi^2$检验和F检验；而对于非嵌套模型，则需要考虑AIC、BIC等辅助指标。

嵌套模型和非嵌套模型是回归分析中两类不同的模型关系。其中，嵌套模型是指两个模型之间存在层次关系，一个模型可以通过对另一个模型中的某些参数施加约束（通常是将某些参数设为0）来简化得到，例如：

  • 模型M1：$g(\mu_i) = \beta_0 + \beta_1 x_{i,1} + \beta_2 x_{i,2}$；
  • 模型M2：$g(\mu_i) = \beta_0 + \beta_1 x_{i,1} + \beta_2 x_{i,2} + \beta_3 x_{i,3}$。

则称M1是M2的子模型。注意两个模型的分布假设和连接函数须相同。

非嵌套模型是指两个模型之间不存在直接的层次关系，例如：

  • 模型使用了不同的分布假设或连接函数；
  • 线性预测子的结构不同，例如$\beta_0 + \beta_1x +\beta_2x^2$与$\beta_0 + \beta_1 \ln x$。

由于嵌套模型之间存在明确的统计关系，可以使用$\chi^2$检验和F检验，对两个模型进行比较和选择。具体而言，当数据分布的尺度参数已知时（例如二项和泊松分布，尺度参数恒为1），则两个嵌套模型的尺度调整偏差之差近似服从卡方分布，即：

$$\begin{gathered} D_2^{*} - D_1^{*} \sim \chi^2(k)\end{gathered}$$

其中，$k$为两个模型的自由度的差值。注意到自由度为$\nu$的卡方分布$\chi^2(\nu)$在5%显著性水平下的临界值约为$2\nu$，若模型2具有$p$个参数、模型1具有$q$个参数，则当$D_1^{*} - D_2^{*} > 2(p-q)$时，认为模型2的改进显著、相对更优。

若数据分布的尺度参数未知，则可以应用如下的F统计量：

$$\begin{gathered}F = \frac{D_2-D_1}{k \hat{\phi}} \sim F(k,n-p-1)\end{gathered}$$

证明：根据上两式，可知：

$$\begin{gathered}D_2^{*} - D_1^{*} = (D_2 - D_1)/\phi \sim \chi^2(k) \\[0.5em] \frac{(n-p-1)\hat{\phi}}{\phi} = \sum\limits_{i = 1}^n {\frac{\omega_i (y_i - \hat{\mu}_i )^2}{\phi V(\hat{\mu}_i)}} \sim \chi^2(n-p-1)\end{gathered}$$

应用卡方分布和F分布的关系，可得：

$$\begin{gathered}F = \frac{[{(D_2 - D_1) / \phi}] / k}{[(n - p - 1)\hat{\phi} / \phi] / (n - p - 1)} = \frac{D_2 - D_1}{k\hat{\phi}} \sim F(k,n-p-1)\end{gathered}$$

命题得证。 $\square$

对于非嵌套模型，尽管仍然可以通过偏差（或尺度调整偏差）来衡量模型的拟合效果，但模型之间差异的显著性无法通过检验来判断。此时可以引入如下一些评价指标：

  • 赤池信息准则（AIC）：$\text{AIC} = -2l + 2d$；
  • 修正AIC：$\text{AICc} = \text{AIC} + \frac{2d(d+1)}{n-d-1}$；
  • 贝叶斯信息准则（BIC）：$\text{BIC} = -2l + d\ln(n)$；
  • Hannan–Quinn信息准则（HQC）：$\text{HQC} = -2l + 2d \ln(\ln(n))$。

上述信息准则在考虑模型拟合优度（对数似然）的同时，对模型的复杂度（参数个数$d$）进行了惩罚。通常情况下，AIC对复杂模型的惩罚相对较轻；BIC的惩罚较重，倾向于选择简单模型；而HQC介于AIC和BIC之间。AICc则主要针对样本量较小且参数个数较多的情况，对AIC的应用效果进行了修正。在同一个信息准则下，可以对候选模型进行横向比较：若某个模型的信息准则指标较小，表明其可以应用较简单的模型结构取得较好的拟合效果，在实际应用中可能是一个理想的选择。

7.6 GLM拓展：GAM与GAMLSS

广义可加模型（generalized additive models，GAM）是广义线性模型的拓展，允许对自变量和因变量之间的关系进行更加灵活的建模。

广义可加模型的基本假设与广义线性模型一致，即因变量$Y_i$服从指数族分布。但与上式相比，广义可加模型将系统性成分的形式拓展为如下结构：

$$g(\mu_i) \triangleq \eta_i = \beta_0 + s_1(X_{i,1}) + s_2(X_{i,2}) + \cdots + s_p(X_{i,p})$$

其中，$s_j(\cdot)$称为平滑函数，可以通过多种方法构造，用于捕捉因变量（系统性成分）与自变量之间的非线性关系。各自变量对$\eta_i$的影响仍然通过加法的形式进行组合，在简单情况下，$s_j(\cdot)$可以取为$\beta_j X_{i,j}$，即退化为广义线性模型。

常见的平滑函数
平滑函数函数形式
B-样条$s(x) = \sum\limits_{k=1^K \alpha_k B_k(x)$，其中$B_k(\cdot)$是一个多项式基函数。
平滑样条$\arg\min_s~ \sum\limits_{i=1^n [y_i - s(x_i)]^2 + \lambda \int[s''(x)]^2 dx$，即最小化回归损失。
局部回归（Loess）选择若干个点，对每个观测点附近的数据进行多项式回归，估计
平滑曲线。

在广义可加模型中，还可以对自变量的交互项添加平滑函数，如$s(X_{i,1},X_{i,2})$，刻画对因变量（系统性成分）的共同影响。常见的多元平滑函数包括张量积（tensor product splines）、多元自适应样条（multivariate adaptive regression splines，MARS）等。

由前所述，对于分布特征较复杂的数据，可以采用基于位置、尺度和形状参数的广义可加模型（generalized additive model for location shape and scale，简称GAMLSS模型）。该模型通过直接对分布参数进行建模，进一步放宽了因变量的分布假设。假设因变量服从一个至多为四参数的分布$F(\mu, \sigma, \nu, \tau)$，四个参数分别影响分布的位置、尺度和形状（偏度和峰度），则GAMLSS的结构可以表示为：

$${\left\{ \begin{aligned} &Y_i \sim F(\mu_i, \sigma_i, \nu_i, \tau_i) \\ &g_1(\mu_i) = \beta_0^{(1)} + s_1^{(1)}(X_{i,1}) + \cdots + s_p^{(1)}(X_{i,p}) \\ &g_2(\sigma_i) = \beta_0^{(2)} + s_1^{(2)}(X_{i,1}) + \cdots + s_p^{(2)}(X_{i,p}) \\ &g_3(\nu_i) = \beta_0^{(3)} + s_1^{(3)}(X_{i,1}) + \cdots + s_p^{(3)}(X_{i,p}) \\ &g_4(\tau_i) = \beta_0^{(4)} + s_1^{(4)}(X_{i,1}) + \cdots + s_p^{(4)}(X_{i,p}) \end{aligned} \right. }$$

其中，该模型允许因变量分布的所有参数都随样本而变化，各自变量的效应首先按照加法进行组合、再通过连接函数与参数相联系。不同参数的连接函数以及系统性成分中不同自变量的平滑函数，都可以根据实际问题进行选择。

尽管GAMLSS模型的形式较为复杂，其回归系数仍然可以借助极大似然思想进行求解。R软件的gamlss程序包也提供了对一些常见分布建立GAMLSS模型的方法。

例 7.7

假定一组投保人的索赔频率服从零膨胀泊松分布$\text{ZIP}(\pi,\lambda)$，且零膨胀概率$\pi$和泊松均值$\lambda$均可能随投保人的风险因子而变化，需要同时构建回归模型。

已知ZIP的概率分布列等于：

$$p(y;\pi,\lambda) = \left\{ \begin{aligned} & \pi+(1-\pi)e^{-\lambda}~,    y=0 \\ & (1-\pi) \lambda^y e^{-\lambda} / y!~,~~ y=1,2,\cdots \end{aligned} \right.$$

则ZIP-GAMLSS模型可以定义为：

$$\left\{ \begin{aligned} & Y_i \sim \text{ZIP}(\pi_i, \lambda_i) \\ & \text{logit}(\pi_i) = \eta_{i1} = \boldsymbol{X}_i \boldsymbol{\beta}_1 \\ & \ln(\lambda_i) = \eta_{i2} = \boldsymbol{X}_i \boldsymbol{\beta}_2 \end{aligned} \right.$$

其中，$\boldsymbol{X}_i = (1, X_{i,1}, X_{i,2}, \cdots, X_{i,p})$表示样本的自变量向量，$\boldsymbol{\beta}_1 = (\beta_{0}^{(1)}, \beta_{1}^{(1)}, \cdots, \beta_{p}^{(1)})^T$和$\boldsymbol{\beta}_2 = (\beta_{0}^{(2)}, \beta_{1}^{(2)}, \cdots, \beta_{p}^{(2)})^T$分别为两个参数的回归系数。连接函数的选取主要考虑了各参数的取值范围。

在此基础上，列出模型的对数似然函数及其一阶和二阶偏导数，分别为：

$$l = \sum_{i=1}^n \ln{p}(y_i; \pi_i, \lambda_i)$$
$$-\frac{\mathrm{d} \ln{p}(\pi, \lambda)}{\mathrm{d} \pi} = \left\{ \begin{aligned} & (1 - e^{-\lambda}) / (\pi + (1 - \pi) e^{-\lambda}) ,    y=0 \\ & 1/(1-\pi) ~,                     ~ y=1,2,\cdots \end{aligned} \right.$$

$$-\frac{\mathrm{d} \ln{p}(\pi, \lambda)}{\mathrm{d} \lambda} = \left\{ \begin{aligned} & (1 - \pi)/(1-\pi + \pi e^{\lambda}) ~,    y=0 \\ & 1 - y/\lambda ~,                   y=1,2,\cdots \end{aligned} \right.$$

应用复合函数求导的链式法则，即可进一步转化成关于$\beta_{j}^{(1)}, \beta_{j}^{(2)}$的偏导。令偏导数等于零、求解对数似然方程组（一般需应用数值方法），即可得到回归系数的极大似然估计值。

(1) 将上式转化为指数族分布的形式，并确定$a(\phi), b(\theta), c(y,\phi)$的表达式。

(2) 利用指数族分布的性质，推导$Y$的数学期望和方差。

【解答】(1) 将原表达式转换成$\displaystyle \exp[\frac{y\theta-b(\theta)}{a(\phi)}+c(y,\phi)]$的形式，结果为：

$$\text{P}(Y=y) = \exp\left[y\ln(1-p)+k\ln{p}+\ln{\binom{k+y-1}{y}}\right]$$

记$\theta=\ln(1-p),~\phi=a(\phi)=1,~b(\theta)=-k\ln{p}~,c(y,\phi)=\ln{\binom{k+y-1}{y}}$，显然可知该分布属于指数族。

(2) 由$p=1-e^\theta$，有：

$$\begin{gathered}\mathbb{E}(Y)=b'(\theta)=\frac{\mathrm{d}}{\mathrm{d}\theta}(-k\ln(1-e^\theta))=-k\frac{-e^\theta}{1-e^\theta}=k\frac{1-p}{p}\\ \text{Var}(Y)=a(\phi)b''(\theta)=\frac{\mathrm{d}}{\mathrm{d}\theta}(k\frac{e^\theta}{1-e^\theta})=k\frac{e^\theta}{(1-e^\theta)^2}=k\frac{1-p}{p^2}\end{gathered}$$

2. 设$Y$的概率密度函数符合指数族的形式：

$$\begin{gathered}f(y) = \exp\left[\frac{y\theta-b(\theta)}{a(\phi)}+c(y,\phi)\right]\end{gathered}$$

(1) 使用$b(\theta)$和$a(\phi)$表示$Y$的均值和方差。

(2) 证明均值为$\mu$的指数分布的密度函数可以写成上述形式。

(3) 求出该分布的自然参数和方差函数。

【解答】(1) 易知指数族分布的均值和方差为：$\mathbb{E}(Y)=b'(\theta),~\text{Var}(Y)=a(\phi)b''(\theta)$。

(2) 对于均值为$\mu$的指数分布，其概率密度函数可以写为：

$$f(y) = \frac{1}{\mu} \exp\left(-\frac{y}{\mu}\right) = \exp\left( -\frac{y}{\mu} + \ln\frac{1}{\mu} \right)$$

与题目中的标准形式比较，分别取：

$$\begin{gathered}\theta = -\frac{1}{\mu},   \phi = a(\phi) = 1,   b(\theta) = -\ln\frac{1}{\mu} = -\ln(-\theta),   c(y, \phi) = 0\end{gathered}$$
