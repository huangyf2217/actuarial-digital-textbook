本章学习目标

  1. 理解损失的随机性，以及描述随机变量的数学工具。
  2. 掌握常见的离散型、连续型分布及其性质。

在非寿险研究中，损失（例如索赔发生与否、索赔频率、每次事故的索赔金额、累积损失金额等）通常无法被准确预测，故可以用随机变量来表示。本章从随机变量及其分布的基本概念出发，系统介绍非寿险精算中常用的离散型和连续型分布，并讨论分布变换、参数估计和拟合优度检验等方法。

1.1 随机变量及其分布

随机变量可以理解为一个不确定的、反映随机事件结果的数值，通常记为大写英文字母 $X, Y$ 等。根据取值范围，可分为离散型随机变量（取值范围为有限个或可列个）和连续型随机变量（可取某个区间中的一切值）。

1.1.1 随机变量的描述

随机变量的取值规律可以运用累积分布函数、概率分布列/概率密度函数、生存函数等进行刻画。

累积分布函数（CDF） 随机变量 $X$ 的累积分布函数定义为： $$F(x) = \text{P}(X \le x), \quad -\infty < x < +\infty$$ 分布函数具有：（1）单调性；（2）$\lim_{x\to -\infty}F(x)=0$，$\lim_{x\to +\infty}F(x)=1$；（3）右连续性。

对于离散型随机变量，使用概率分布列描述其取每个具体值的概率：$p(x_i) = \text{P}(X = x_i)$。

对于连续型随机变量，使用概率密度函数（pdf）描述其取值规律。若存在非负可积函数 $f(x) \geq 0$，使 $F(x)=\int_{-\infty}^{x}f(t)\mathrm{d}t$，则称 $f(x)$ 为随机变量 $X$ 的概率密度函数。显然，$f(x)=F'(x)$，$\int_{-\infty}^{\infty} f(x) \mathrm{d}x = 1$。

生存函数 给定随机变量 $X$ 及其分布函数 $F(x)$，其生存函数定义为： $$S(x) = 1 - F(x) = \text{P}(X > x)$$ 生存函数描述了随机变量 $X$ 的取值大于 $x$ 的概率。

1.1.2 随机变量的数字特征

1. 数学期望

数学期望反映随机变量的中心趋势。离散型：$\mathbb{E}X = \sum_{i}x_ip(x_i)$；连续型：$\mathbb{E}X = \int_{-\infty}^{\infty}xf(x)\mathrm{d}x$。

期望的运算性质：$\mathbb{E}(aX+b) = a\mathbb{E}(X) + b$；$\mathbb{E}(X + Y) = \mathbb{E}(X) + \mathbb{E}(Y)$；若 $X$ 与 $Y$ 不相关，则 $\mathbb{E}(XY) = \mathbb{E}(X)\mathbb{E}(Y)$。

2. 方差

方差衡量随机变量取值的离散程度：$\text{Var}(X) = \mathbb{E}[(X-\mathbb{E}X)^2] = \mathbb{E}(X^2)-\mathbb{E}^2(X)$。

3. 有限期望值和平均超越值

有限期望值 描述存在赔偿限额 $d$ 时的期望损失： $$\mathbb{E}(X \wedge d) = \int_{0}^{d} x f(x) \mathrm{d}x + d [1 - F(d)]$$ 其中 $X \wedge d = \min(X, d)$。
平均超越值 描述存在免赔额 $u$ 时的期望损失： $$e(u) = \mathbb{E}(X - u \mid X > u) = \frac{\int_{u}^{\infty} (x - u) f(x) \mathrm{d}x}{1 - F(u)}$$

4. 矩

$k$ 阶原点矩：$\mu_k' = \mathbb{E}(X^k)$；$k$ 阶中心矩：$\mu_k = \mathbb{E}[(X - \mathbb{E}X)^k]$。期望是1阶原点矩，方差是2阶中心矩。

由此可定义：变异系数 $\sigma/\mu$；偏度系数 $skew(X) = \mu_3/\sigma^3$；峰度系数 $kurt(X) = \mu_4/\sigma^4$。

5. 分位数

给定 $0 < q < 1$，随机变量 $X$ 的 $q$ 分位数 $x_q$ 定义为：$F(x_q) = q$。$q=0.5$ 时的分位数称为中位数。

1.1.3 概率母函数、矩母函数和累积量生成函数

概率母函数（PGF） 对于取值范围为 $\{0,1,2,\cdots\}$ 的随机变量 $X$： $$P(z) = \mathbb{E}(z^X) = \sum_{x=0}^{\infty} z^x \cdot \text{P}(X=x)$$ 当期望存在时，$\mathbb{E}(X) = P'(1)$；当方差存在时，$\text{Var}(X) = P''(1) + P'(1) - [P'(1)]^2$。
矩母函数（MGF） $$M_X(t) = \mathbb{E}(e^{tX}) = \sum_{x}e^{tx} \cdot \text{P}(X=x) \quad \text{或} \quad \int e^{tx}f(x)\mathrm{d}x$$ 对矩母函数求导可得各阶原点矩：$\mathbb{E}(X^k) = M_X^{(k)}(0)$。

累积量生成函数（CGF）定义为 $C_X(t) = \ln M_X(t)$。在0点处求导可得：$\mu = C_X'(0)$，$\sigma^2 = C_X''(0)$，$\mathbb{E}[(X-\mathbb{E}X)^3] = C_X'''(0)$。

1.2 常见的离散型分布

在非寿险精算中，离散型分布广泛用于保险索赔次数的建模。本节介绍二项分布、泊松分布和负二项分布。

1.2.1 二项分布（Binomial distribution）

B 二项分布 $\text{B}(n, p)$

若随机变量 $N$ 服从参数为 $n$ 和 $p$ 的二项分布，记为 $\text{B}(n, p)$，其中 $n$ 为正整数且 $0 < p < 1$。当 $n = 1$ 时退化为伯努利分布。

概率分布列 $$p_k = \binom{n}{k} p^k (1-p)^{n-k}, \quad k = 0, 1, \cdots, n$$ 其中 $\binom{n}{k} = \frac{n!}{k!(n-k)!}$。
数字特征 $$\mathbb{E}(N) = np, \quad \text{Var}(N) = np(1-p)$$ $$P(z) = [1 + p(z-1)]^n$$

性质

  1. 可描述 $n$ 个独立同分布风险所组成的风险集合的索赔频率。
  2. 索赔频率存在极大值 $n$（如一次自然灾害造成的伤亡人数）。
  3. $p = 0.5$ 时分布对称；$p < 0.5$ 时正偏；$p > 0.5$ 时负偏。
  4. 众数为 $[(n+1)p]$，若 $(n+1)p$ 为整数则具有双众数。
  5. 方差小于均值——与泊松分布和负二项分布的主要区别。

1.2.2 泊松分布（Poisson distribution）

P 泊松分布 $\text{Poi}(\lambda)$

若随机变量 $N$ 服从参数为 $\lambda > 0$ 的泊松分布，记为 $\text{Poi}(\lambda)$。

概率分布列 $$p_k = \frac{\lambda^k}{k!}e^{-\lambda}, \quad k = 0, 1, 2, \ldots$$
数字特征 $$\mathbb{E}(N) = \text{Var}(N) = \lambda$$ $$P(z) = \exp\{\lambda(z-1)\}$$

性质

  1. 均值和方差相等，都等于参数 $\lambda$。
  2. 可加性：若 $N_1 \sim \text{Poi}(\lambda_1)$，$N_2 \sim \text{Poi}(\lambda_2)$ 独立，则 $N_1+N_2 \sim \text{Poi}(\lambda_1+\lambda_2)$。
  3. 可分解性：总索赔次数 $N \sim \text{Poi}(\lambda)$，按概率 $p_1, p_2$ 分解为两类独立的泊松变量。
  4. $\lambda$ 较小时近似二项分布；$\lambda$ 较大时近似正态分布。
  5. 众数为 $[\lambda]$，若 $\lambda$ 为整数则具有双众数。
  6. 若事故发生时间间隔服从指数分布，则固定时间内事故次数服从泊松分布。

1.2.3 负二项分布（Negative Binomial distribution）

NB 负二项分布 $\text{NB}(r, \beta)$

若随机变量 $N$ 服从参数为 $r > 0$ 和 $\beta > 0$ 的负二项分布，记为 $\text{NB}(r, \beta)$。

概率分布列 $$p_k = \binom{k+r-1}{k} \left(\frac{1}{1+\beta}\right)^r \left(\frac{\beta}{1+\beta}\right)^k, \quad k = 0, 1, 2, \ldots$$
数字特征 $$\mathbb{E}(N) = r\beta, \quad \text{Var}(N) = r\beta(1+\beta)$$ $$P(z) = [1 - \beta(z-1)]^{-r}$$

性质

  1. 具有两个参数，概率函数具有更大的灵活性。
  2. 方差大于均值——与泊松分布的重要差别。若观察值方差大于均值（过离散），负二项分布拟合更好。
  3. 是一种混合泊松分布：泊松参数服从伽马分布时，混合结果为负二项分布。
  4. 当索赔频率随时间线性增加时，适合使用负二项分布建模。
  5. 众数为 $[(r-1)\beta]$，若 $(r-1)\beta$ 为整数则具有双众数。

当 $r = 1$ 时，负二项分布退化为几何分布：$p_k = p(1-p)^k, \quad k = 0, 1, 2, \ldots$

1.2.4 (a,b,0)与(a,b,1)分布类

二项分布、泊松分布和负二项分布（几何分布）均属于 (a, b, 0) 分布类，其取值概率满足递归关系：

(a,b,0)递归关系 $$p_k = \left(a + \frac{b}{k}\right)p_{k-1}, \quad k = 1, 2, \cdots$$
分布$a$$b$$p_0$
二项分布$-\frac{p}{1-p}$$\frac{(n+1)p}{1-p}$$(1-p)^n$
泊松分布$0$$\lambda$$e^{-\lambda}$
负二项分布$\frac{\beta}{1+\beta}$$\frac{(r-1)\beta}{1+\beta}$$(1+\beta)^{-r}$
几何分布$\frac{\beta}{1+\beta}$$0$$(1+\beta)^{-1}$

在非寿险实务中，常需对零点概率进行调整，得到 (a, b, 1) 分布类。其递推关系从 $p_1$ 开始计算，$p_0$ 需单独设定。

零截断分布 ZT-(a,b,1) 令 $p_0 = 0$，零点概率被完全截断： $$p_0^T = 0, \quad p_k^T = \frac{p_k}{1 - p_0}, \quad k = 1, 2, 3, \ldots$$
零调整分布 ZM-(a,b,1) 令 $0 < p_0 < 1$，将零点概率调整为任意值： $$p_0^M \in (0,1), \quad p_k^M = \frac{1 - p_0^M}{1 - p_0} \cdot p_k, \quad k = 1, 2, 3, \ldots$$
例 1.1 在R软件中分别绘制零截断、零调整负二项分布的概率分布。 

## 零截断负二项分布的概率
x <- 0:10
p <- dnbinom(x, 4, 0.7)       # 负二项分布的概率
p0 <- p[1]                    # 零点的概率

1.3.1 指数分布（Exponential distribution）

E 指数分布 $\text{Exp}(\lambda)$
密度函数与分布函数 $$f(x) = \lambda e^{-\lambda x}, \quad x \geq 0$$ $$F(x) = 1 - e^{-\lambda x}, \quad x \geq 0$$
数字特征 $$\mathbb{E}(X) = \frac{1}{\lambda}, \quad \text{Var}(X) = \frac{1}{\lambda^2}$$ $$M(t) = \frac{\lambda}{\lambda - t}, \quad t < \lambda$$

有限期望函数存在显式解：$\mathbb{E}[(X \wedge d)^k] = \frac{\gamma(k+1, \lambda d)}{\lambda^k} + d^k e^{-\lambda d}$。

性质

  1. 变异系数等于1，众数等于0。
  2. 若索赔次数服从参数为 $\lambda$ 的泊松分布，则两个相邻损失发生的时间间隔服从参数为 $1/\lambda$ 的指数分布。
  3. 无记忆性：$\text{P}(X \geq s+t \mid X\geq s) = \text{P}(X \geq t)$。
  4. 指数分布是伽马分布和威布尔分布中当 $\alpha = 1$（或 $\tau = 1$）时的特殊情况。
  5. 通常作为轻尾分布和厚尾分布的分水岭。

1.3.2 伽马分布（Gamma distribution）

Γ 伽马分布 $\text{Ga}(\alpha, \lambda)$
密度函数 $$f(x) = \frac{\lambda^\alpha}{\Gamma(\alpha)}x^{\alpha-1}e^{-\lambda x}, \quad x > 0$$ 其中 $\Gamma(s)=\int_0^{\infty} t^{s-1}e^{-t}\mathrm{d}t$ 为伽马函数，$\alpha$ 为形状参数，$\lambda$ 为速率参数。

令 $\theta = 1/\lambda$，可重参数化为 $f(x) = \frac{1}{\Gamma(\alpha) \theta^\alpha} x^{\alpha-1} e^{-x/\theta}$，记为 $\text{Ga}(\alpha, \theta)$，$\theta$ 为尺度参数。

数字特征 $$\mathbb{E}(X) = \alpha \theta, \quad \text{Var}(X) = \alpha \theta^2$$ $$M(t) = (1-\theta t)^{-\alpha}, \quad t < 1/\theta$$

有限期望函数：$\mathbb{E}[(X \wedge d)^k] = \frac{\theta^k \gamma(\alpha + k, d/\theta)}{\Gamma(\alpha)} + d^k \left[1-\frac{\gamma(\alpha,d/\theta)}{\Gamma(\alpha)}\right]$。

性质

  1. 变异系数等于 $\alpha^{-0.5}$；当 $\alpha > 1$ 时，众数为 $(\alpha-1)\theta$。
  2. 伽马分布呈右偏（正偏）；当 $\alpha \to \infty$ 时，近似于正态分布。
  3. 固定尺度参数具有可加性：若 $X_1 \sim \text{Ga}(\alpha_1,\theta)$，$X_2 \sim \text{Ga}(\alpha_2,\theta)$ 独立，则 $X_1+X_2 \sim \text{Ga}(\alpha_1+\alpha_2,\theta)$。
  4. 伽马随机变量 $X$ 乘以常数 $r$ 后仍服从伽马分布，参数变为 $(\alpha, r\theta)$。
  5. 当 $\alpha = 1$ 时，伽马分布退化为指数分布。

1.3.3 逆高斯分布（Inverse Gaussian distribution）

IG 逆高斯分布 $\text{IG}(\mu, \lambda)$
密度函数 $$f(x) = \sqrt{\frac{\lambda}{2\pi x^3}} \exp\left[-\frac{\lambda(x-\mu)^2}{2\mu^2 x}\right], \quad x > 0$$ 其中 $\mu > 0$ 为均值参数，$\lambda > 0$ 为形状参数。令 $\sigma^2=1/\lambda$，可重参数化为 $\text{IG}(\mu,\sigma^2)$。
数字特征 $$\mathbb{E}(X) = \mu, \quad \text{Var}(X) = \frac{\mu^3}{\lambda}$$ $$M(t) = \exp\left[\frac{\lambda}{\mu}\left(1 - \sqrt{1 - \frac{2\mu^2 t}{\lambda}}\right)\right]$$

性质

  1. 变异系数为 $(\mu/\lambda)^{0.5}$，众数为 $\mu \left[(1+\frac{9\mu^2}{4\lambda^2})^{0.5} - \frac{3\mu}{2\lambda} \right]$。
  2. 当 $\lambda \to \infty$ 时，逆高斯分布近似于正态分布。
  3. 逆高斯分布呈右偏，尾部较伽马分布更厚。
  4. 逆高斯分布和伽马分布同属于广义逆高斯分布（GIG）的特例。

1.3.4 帕累托分布（Pareto distribution）

Pa 帕累托分布 $\text{Pa}(\alpha, \lambda)$
分布函数与密度函数 $$F(x) = 1 - \left(\frac{\lambda}{\lambda + x}\right)^\alpha, \quad x \geq 0$$ $$f(x) = \frac{\alpha \lambda^\alpha}{(\lambda + x)^{\alpha + 1}}, \quad x \geq 0$$ 其中 $\alpha > 0$ 为形状参数，$\lambda > 0$ 为尺度参数。
数字特征 $$\mathbb{E}(X) = \frac{\lambda}{\alpha - 1}, \quad \alpha > 1$$ $$\text{Var}(X) = \frac{\alpha \lambda^2}{(\alpha - 1)^2 (\alpha - 2)}, \quad \alpha > 2$$

有限期望函数（$k=1$ 时）： $$\mathbb{E}[X \wedge d] = \begin{cases} \frac{\lambda}{\alpha-1} \left[ 1 - \left( \frac{\lambda}{\lambda+d} \right)^{\alpha-1} \right], & \alpha \neq 1 \\ -\lambda \ln \left( \frac{\lambda}{\lambda+d} \right), & \alpha = 1 \end{cases}$$

性质

  1. 变异系数为 $\sqrt{\frac{\alpha}{\alpha-2}}$，众数等于0。
  2. 帕累托分布呈右偏，是典型的厚尾分布。
  3. 尾部概率 $\text{P}(X>x)=\left(\frac{\lambda}{\lambda + x}\right)^\alpha$，衰减速度由 $\alpha$ 决定，故 $\alpha$ 又称尾部指数。
  4. 当 $\alpha$ 较小时，尾部概率衰减较慢，极端值出现概率相对较高，分布的高阶矩可能不存在。

帕累托分布对极端事件较为敏感，能够帮助保险公司评估和管理高风险事件的潜在大额损失（如飓风、洪水等巨灾事件）。

1.3.5 对数正态分布（Log-normal distribution）

LN 对数正态分布 $\text{LN}(\mu, \sigma^2)$

若 $\ln X$ 服从正态分布 $\text{N}(\mu,\sigma^2)$，则 $X$ 服从对数正态分布，记为 $X \sim \text{LN}(\mu,\sigma^2)$。

密度函数与分布函数 $$f(x) = \frac{1}{\sqrt{2\pi} \sigma x}\exp\left[-\frac{1}{2}\left(\frac{\ln x - \mu}{\sigma}\right)^2\right], \quad x > 0$$ $$F(x) = \Phi\left(\frac{\ln x - \mu}{\sigma}\right), \quad x > 0$$ 其中 $\Phi(\cdot)$ 是标准正态分布的分布函数。
数字特征 $$\mathbb{E}(X) = \exp\left(\mu + \frac{\sigma^2}{2}\right), \quad \text{Var}(X) = \left[\exp(\sigma^2) - 1\right]\exp\left(2\mu + \sigma^2\right)$$ $$\mathbb{E}(X^k) = \exp\left(k\mu + \frac{1}{2}k^2 \sigma^2\right)$$

性质

  1. 变异系数等于 $[\exp(\sigma^2) - 1]^{0.5}$，仅与 $\sigma$ 有关。众数为 $\exp(\mu-\sigma^2)$。
  2. 对数正态分布可通过正态分布的指数变换得到，正态分布可通过对数正态分布的对数变换得到。
  3. 对数正态分布呈右偏，尾部较伽马分布、逆高斯分布更厚。
  4. 均值和方差是参数 $(\mu,\sigma^2)$ 的增函数。固定 $\mu$，当 $\sigma \to 0$ 时，均值收敛于 $e^\mu$，方差收敛于0。
  5. 给定正实数 $r,t$，若 $X \sim \text{LN}(\mu,\sigma)$，则 $Y = rX^t$ 服从参数为 $(t\mu + \ln r, t^2 \sigma)$ 的对数正态分布。

1.3.6 威布尔分布（Weibull distribution）

W 威布尔分布 $\text{W}(\tau, \theta)$
分布函数与密度函数 $$F(x) = 1 - \exp\left[-\left(\frac{x}{\theta}\right)^\tau\right], \quad x \geq 0$$ $$f(x) = \frac{\tau}{\theta^\tau} x^{\tau-1} \exp\left[-\left(\frac{x}{\theta}\right)^\tau\right], \quad x \geq 0$$ 其中 $\tau > 0$ 为形状参数，$\theta > 0$ 为尺度参数。
数字特征 $$\mathbb{E}(X) = \theta \cdot \Gamma\left(1 + \frac{1}{\tau}\right)$$ $$\text{Var}(X) = \theta^2 \left[\Gamma\left(1 + \frac{2}{\tau}\right) - \Gamma^2\left(1 + \frac{1}{\tau}\right)\right]$$ $$\mathbb{E}(X^k) = \theta^k \Gamma\left(1 + \frac{k}{\tau}\right), \quad k > -\tau$$

威布尔分布可认为是对指数分布的推广。当 $\tau=1$ 时，威布尔分布等价于指数分布；通常取 $0<\tau<1$，此时威布尔分布的尾部较指数分布更厚。

性质

  1. 当 $\tau \leq 1$ 时，众数等于0；当 $\tau > 1$ 时，众数为 $\theta\left(\frac{\tau-1}{\tau}\right)^{1/\tau}$。
  2. 若 $X \sim \text{W}(\tau,\theta)$，则 $rX \sim \text{W}(\tau, r\theta)$。
  3. 威布尔分布属于广义指数分布。
  4. 威布尔分布在 $\tau = 3.6$ 时近似对称，$\tau > 3.6$ 时右偏，$\tau < 3.6$ 时左偏。

1.3.7 常见连续型分布与广义伽马、贝塔分布间的关系

在非寿险精算模型中，理解常见连续型分布与广义伽马分布（GG）、广义贝塔分布（GB）之间的关系非常重要。广义伽马分布和广义贝塔分布可以看作是分布族中的"母分布"，许多常见分布都是它们的特例。

广义伽马分布（GG） $$f(x; a, d, p) = \frac{p/a^d}{\Gamma(d/p)} x^{d-1} e^{-(x/a)^p}, \quad x > 0$$ 其中 $d>0, p>0$ 是形状参数，$a$ 是尺度参数。

许多常见分布是广义伽马分布的特例：

  • 指数分布：当 $p = d = 1$ 时。
  • 伽马分布：当 $p = 1$ 时。
  • 威布尔分布：当 $d = p$ 时。
第二类广义贝塔分布（GB2） $$f(x; a,b,p,q) = \frac{1}{\text{B}(p,q)} \frac{|a|x^{ap-1}}{b^{ap} [1+(x/b)^a]^{p+q}}, \quad x > 0$$

广义伽马分布是GB2的极限情况：$\text{GG}(a,\beta,p) = \lim_{q \to \infty} \text{GB2}(a,b=q^{1/a}\beta,p,q)$。

1.4 损失分布的变换

在非寿险精算中，对数变换和指数变换是两种常见的随机变量变换方式。这些变换常用于调整数据的分布形态，使其更符合分析模型的假设要求。

1.4.1 对数变换和指数变换

对数变换是一种常用的非线性变换，对随机变量 $X$ 取对数，得到新的随机变量 $Y = \ln(X)$。根据概率密度函数的变换公式：

密度函数变换公式 $$f_Y(y) = f_X(x(y)) \left| \frac{dx}{dy} \right|$$

考虑对数变换的逆变换 $X = e^Y$，易解得 $\frac{dx}{dy} = e^y$，代入上式可知 $Y$ 的密度函数为：

$$f_Y(y) = f_X(e^y) \cdot e^y$$

$Y$ 的分布函数可以通过定义直接求解：$F_Y(y) = \text{P}(Y \leq y) = \text{P}(\ln X \leq y) = \text{P}(X \leq e^y) = F_X(e^y)$。

例 1.2 若随机变量 $X$ 服从参数为 $\lambda$ 的指数分布，即 $X \sim \text{Exp}(\lambda)$，计算对数变换后随机变量 $Y = \ln X$ 的分布。

已知 $f_X(x) = \lambda e^{-\lambda x},\ x \geq 0$，代入密度函数变换公式，可得 $Y$ 的概率密度函数为：

$$f_Y(y) = \lambda e^y e^{-\lambda e^y}, \quad y \in \mathbb{R}$$

注意，随机变量 $Y$ 的取值范围为 $y \in \mathbb{R}$。该结果表明，对数变换可以将原本定义在非负实数上的随机变量转换为定义在整个实数轴上的随机变量。

指数变换也有类似结果。记 $X$ 为原始的随机变量，经过指数变换可以得到新的随机变量 $Y = e^X$。若 $X$ 具有密度函数和分布函数 $f_X(\cdot), F_X(\cdot)$，则 $Y$ 的分布满足：

$$f_Y(y) = \frac{1}{y} f_X(\ln y), \quad F_Y(y) = F_X(\ln y)$$

由前所述，正态分布经过指数变换可以得到对数正态分布，而对数正态分布经过对数变换可以得到正态分布。

1.4.2 组合分布（composite distribution）

组合分布泛指一类通过对多个分布进行拼接得到的复杂分布类，通常被用于描述具有多重特性的数据（例如损失中既存在大量小额赔付，又存在大额极端赔付的情况）。其核心思想是用不同的分布分别刻画不同区间的数据，再将它们组合在一起形成一个整体分布。

组合分布 对于一个在 $[0,+\infty)$ 上取值的随机变量 $X$，以 $c_0 = 0 < c_1 < \cdots < c_{r-1} < c_r = +\infty$ 作为分界点，将随机变量的取值范围划分成 $r$ 个互不相交的区间。应用 $f_i(\cdot)$ 分别描述区间 $[c_{i-1},c_i)$ 上的数据，则组合分布的概率密度函数为： $$f(x) = \frac{p_i f_i(x)}{A_i}, \quad x \in [c_{i-1}, c_i), \quad i = 1, 2, \cdots, r$$ 其中 $p_i \geq 0$ 为权重系数，且 $\sum_{i=1}^{r} p_i = 1$；$A_i = F_i(c_i) - F_i(c_{i-1})$。

为使组合分布的密度函数在分界点处保持连续，可以添加条件：$p_i \frac{f_i(c_i)}{A_i} = p_{i+1} \frac{f_{i+1}(c_i)}{A_{i+1}}$。若令 $f'(c_i^-) = f'(c_i^+)$，可进一步增强光滑性。在某些简化情况下，也可以放宽连续性要求，此时被称为剪接模型（splicing models）。

例 1.3 若使用 $\text{Exp}(1)$ 描述区间 $[0, 2)$ 上的数据；使用 $\text{Gamma}(2, 1)$ 描述区间 $[2, 4)$ 上的数据；使用 $\text{Pareto}(2, 2)$ 描述区间 $[4, \infty)$ 上的数据。请构造组合分布 $f(x)$，且 $f(x)$ 在分界点 $x = 2$ 和 $x = 4$ 处是连续的。

已知三种分布的概率密度函数分别为：$f_1(x) = e^{-x}$，$f_2(x) = xe^{-x}$，$f_3(x) = \frac{8}{(x+2)^3}$。

首先计算每个区间内的积分 $A_i$：

$$A_1 = \int_0^2 e^{-x} \mathrm{d}x = 1 - e^{-2} \approx 0.8647$$ $$A_2 = \int_2^4 x e^{-x} \mathrm{d}x \approx 0.3144$$ $$A_3 = \int_4^\infty \frac{8}{(x+2)^3} \mathrm{d}x \approx 0.1111$$

为使组合分布的密度函数连续，需要解以下方程组：

$$\begin{cases} \frac{p_1}{0.8647} \cdot e^{-2} = \frac{p_2}{0.3144} \cdot 2e^{-2} \\ \frac{p_2}{0.3144} \cdot 4e^{-4} = \frac{p_3}{0.1111} \cdot \frac{8}{6^3} \\ p_1 + p_2 + p_3 = 1 \end{cases}$$

最终解得权重值：$p_1 \approx 0.7640$，$p_2 \approx 0.1389$，$p_3 \approx 0.0971$。

因此，新的组合分布可以表示为：

$$f(x) = \begin{cases} 0.8835 \cdot e^{-x}, & x \in [0, 2) \\ 0.4418 \cdot x e^{-x}, & x \in [2, 4) \\ 0.8740 \cdot \frac{8}{(x + 2)^3}, & x \in [4, \infty) \end{cases}$$

1.4.3 混合分布（mixture distribution）

混合是另一种构造复杂分布的方法，通过将多个不同的分布按照一定的权重进行加权，得到新的分布。混合分布由若干个"成分分布"构成，每个成分的贡献通过相应的权重控制。在非寿险精算中，混合分布常用于刻画具有异质性的损失数据。

混合分布 假设一个混合分布由 $k$ 个成分构成，每个成分对应的分布函数和密度函数记为 $F_i(\cdot)$ 和 $f_i(\cdot)$，则混合分布的形式为： $$f(x) = \omega_1 f_1(x) + \omega_2 f_2(x) + \cdots + \omega_k f_k(x)$$ $$F(x) = \omega_1 F_1(x) + \omega_2 F_2(x) + \cdots + \omega_k F_k(x)$$ 其中 $\omega_i \geq 0$ 是成分权重，且 $\sum_{i=1}^k \omega_i = 1$。

对混合分布进行推广。假设 $X$ 的分布形状与参数值 $\Theta$ 有关，$\Theta$ 是一个离散型随机变量，概率分布为 $\text{P}(\Theta = \theta_i) = \omega_i$，则 $X$ 的分布可以表示为：

$$f_X(x) = \sum_{i=1}^k P(\Theta = \theta_i) f_{X\vert\Theta}(x\vert \theta_i) = \sum_{i=1}^k w_i f_{X\vert\Theta}(x\vert \theta_i)$$

当参数 $\Theta$ 为连续型随机变量且具有概率密度函数 $f_\Theta(\theta)$ 时，$X$ 的分布可用积分表示为：$f_X(x) = \int_{-\infty}^\infty f_\Theta(\theta) f_{X\vert\Theta}(x\vert\theta) d\theta$。

例 1.5（指数-伽马混合 = 帕累托） 假设保单组合中每个个体的损失都服从指数分布 $X_i \sim \text{Exp}(\lambda_i)$。对于大型保单组合，所有保单持有人的平均损失 $\lambda$ 可以视为一个随机变量，假定 $\lambda \sim \text{Ga}(\alpha,\delta)$。求混合后的损失分布。

记 $X$ 表示保单组合中的某个个体的损失，由此产生的损失分布即为混合分布，相应的概率密度函数为：

$$f_X(x) = \int_0^\infty f_\Lambda(\lambda) f_{X|\Lambda}(x|\lambda) \mathrm{d}\lambda = \int_0^\infty \frac{\delta^\alpha}{\Gamma(\alpha)} \lambda^{\alpha-1} e^{-\delta\lambda} \cdot \lambda e^{-\lambda x} \mathrm{d}\lambda$$

整理被积函数为 $\text{Ga}(\alpha + 1, x+\delta)$ 分布的形式：

$$f_X(x) = \frac{\delta^\alpha}{\Gamma(\alpha)} \cdot \frac{\Gamma(\alpha+1)}{(x+\delta)^{\alpha+1}} \int_0^\infty \frac{(x+\delta)^{\alpha+1}}{\Gamma(\alpha+1)} \lambda^\alpha e^{-(x+\delta)\lambda} \mathrm{d}\lambda$$

积分结果等于1，因此得到：

$$f_X(x) = \frac{\alpha \delta^\alpha}{(x+\delta)^{\alpha+1}}, \quad x > 0$$

上式为帕累托分布 $\text{Pa}(\alpha, \delta)$ 的概率密度函数。即如果损失服从参数为 $\lambda$ 的指数分布，并且 $\lambda \sim \text{Ga}(\alpha,\delta)$，则混合后的损失分布就是 $\text{Pa}(\alpha, \delta)$。

例 1.6（伽马-伽马混合 = 广义帕累托） 当 $X \sim \text{Ga}(k,\lambda)$ 且 $k=1$ 时，损失分布退化成 $\text{Exp}(\lambda)$，由例1.5可知混合结果为帕累托分布。当 $k$ 取一般数时，伽马-伽马混合分布的结果为何种分布？

对于一般的 $k$，可类似求得 $X$ 的概率密度函数为：

$$f_X(x) = \int_0^\infty \frac{\delta^\alpha}{\Gamma(\alpha)} \lambda^{\alpha-1} e^{-\delta\lambda} \cdot \frac{\lambda^k}{\Gamma(k)} x^{k-1} e^{-\lambda x} \mathrm{d}\lambda = \frac{\delta^\alpha x^{k-1}}{\Gamma(\alpha)\Gamma(k)} \int_0^\infty \lambda^{\alpha+k-1} e^{-(\delta+x)\lambda} \mathrm{d}\lambda$$

将被积函数整理成 $\text{Ga}(\alpha+k, x+\delta)$ 分布的形式，积分结果等于1，因此得到：

$$f_X(x) = \frac{\Gamma(\alpha+k)}{\Gamma(\alpha)\Gamma(k)} \frac{\delta^\alpha x^{k-1}}{(\delta+x)^{(\alpha+k)}}$$

上式为广义帕累托分布 $\text{Pa}(\alpha, \delta, k)$ 的概率密度函数。

典型的混合分布还包括泊松-伽马混合分布，即假设泊松分布的参数 $\lambda$ 服从伽马分布，混合后的结果为负二项分布。

1.5 分布拟合与参数估计

在分布拟合中，问题可以表述为：假定随机变量 $X$ 服从某分布 $F(x;\theta)$，其中 $\theta$ 是分布中的未知参数。若已知 $(X_1, X_2, \dots, X_n)$ 是来自该分布的一组简单随机样本，需要从数据中估计参数值 $\hat{\theta}$。

1.5.1 矩估计

矩估计法（moment estimation，ME）是一种通过样本矩近似总体矩来估计分布参数的方法。给定一组样本 $(x_1, x_2, \dots, x_n)$，分布的样本矩可以计算为：

$$\hat{\mu}_k' = \frac{1}{n} \sum_{i=1}^n x_i^k, \quad \hat{\mu}_k = \frac{1}{n} \sum_{i=1}^n (x_i-\bar{x})^k$$

若分布的参数 $\theta$ 可以表示为某些总体矩的函数 $\theta = g(\mu_1',\cdots,\mu_r'; \mu_2,\cdots,\mu_s)$，通过计算对应的样本矩，并用其替代参数中的总体矩，得到估计量 $\hat{\theta} = g(\hat{\mu}_1',\cdots,\hat{\mu}_r'; \hat{\mu}_1,\cdots,\hat{\mu}_s)$，称为参数 $\theta$ 的矩估计量。

例如对伽马分布 $\text{Ga}(\alpha,\theta)$，用样本均值 $\bar{x}$ 替代数学期望、用样本方差 $s_n^2$ 替代方差，即可得到关系：

$$\begin{cases} \bar{x} = \alpha \theta \\ s_n^2 = \alpha \theta^2 \end{cases} \Rightarrow \hat{\alpha} = \frac{\bar{x}^2}{s_n^2}, \quad \hat{\theta} = \frac{s_n^2}{\bar{x}}$$

1.5.2 极大似然估计

1. 似然函数与极大似然思想

极大似然估计法（maximum likelihood estimation，MLE）是一种通过最大化样本数据的似然函数来估计分布参数的方法。当样本观察值已知、而模型参数 $\boldsymbol{\theta}$ 未知时，似然函数为：

$$L(\boldsymbol{\theta}) = \prod_{i=1}^{n} f(x_i; \boldsymbol{\theta}), \quad l(\boldsymbol{\theta}) = \ln L(\boldsymbol{\theta}) = \sum_{i=1}^n \ln f(x_i; \boldsymbol{\theta})$$

极大似然法是对似然函数求极大化，即 $\hat{\boldsymbol{\theta}} = \arg\max_{\boldsymbol{\theta} \in \Theta} L(\boldsymbol{\theta})$。对于单参数场景，若 $l(\theta)$ 可以对参数 $\theta$ 求导，则极大似然估计 $\hat{\theta}$ 应满足 $\frac{\mathrm{d}}{\mathrm{d} \theta}l(\hat{\theta}) = 0$。

例 1.7（指数分布的MLE） 保险公司用指数分布来研究遭受事故的车辆维修费用。已知1000台车辆的样本的平均维修费用是2200英镑，求平均费用 $1/\lambda$ 的极大似然估计。

记 $X_1, X_2, \ldots, X_n$ 表示个体维修费用（$n=1000$），$X_i \sim \text{Exp}(\lambda)$。似然函数为：

$$L = \prod_{i=1}^n \lambda e^{-\lambda x_i} = \lambda^n e^{-\lambda n\bar{x}}$$

对数似然函数：$\ln L = n\ln \lambda - \lambda n\bar{x}$。求导并令导函数等于0：

$$\frac{\mathrm{d}}{\mathrm{d} \lambda}\ln L = \frac{n}{\lambda} - n\bar{x} = 0 \Rightarrow \hat{\lambda} = \frac{1}{\bar{x}} = \frac{1}{2200}$$

因此，平均成本的极大似然估计为 $1/\hat{\lambda} = 2200$，恰好等于样本均值。

例 1.8（对数正态分布的MLE） 对数正态分布的参数估计是直接的，因为 $\mu$ 和 $\sigma^2$ 可以通过对数变换，在正态分布下进行估计。记 $X \sim \text{LN}(\mu,\sigma^2)$，$x_1, x_2, \ldots, x_n$ 为样本观察值，令 $y_i = \ln x_i$。由于正态分布 $Y \sim \text{N}(\mu,\sigma^2)$ 的极大似然估计分别为样本均值和样本方差（分母为 $n$），则在对数正态分布下 $\mu$ 和 $\sigma^2$ 的极大似然估计为 $\bar{y}$ 和 $s_y^2$。

例 1.9（威布尔分布的百分位数估计） 假定 $X \sim \text{W}(c,\gamma)$，样本的下四分位数为401，上四分位数为2836.75，用百分位数法估计威布尔分布的参数。

根据威布尔的分布函数，可建立关于 $c$ 和 $\gamma$ 的方程组：

$$\begin{cases} F(401) = 1-\exp(-c \cdot 401^\gamma) = 0.25 \\ F(2836.75) = 1-\exp(-c \cdot 2836.75^\gamma) = 0.75 \end{cases}$$

解得参数的百分位数估计值 $\tilde{\gamma} = 0.8038$，$\tilde{c} = 0.002326$。注意 $\tilde{\gamma} < 1$，表明此时具有比指数分布更厚的尾部。

2. 极大似然估计的方差

若 $f(x; \theta)$ 满足正则条件，则当 $n \to \infty$ 时，参数的极大似然估计量 $\hat{\theta}_n$ 渐进服从正态分布 $\text{N}(\theta, [I(\theta)]^{-1})$。其中 $I(\theta)$ 称为 Fisher信息量：

$$I(\theta) = - \mathbb{E} \left[ \frac{\mathrm{d}^2}{\mathrm{d}\theta^2} l(\theta) \right] = \mathbb{E} \left[ \left(\frac{\mathrm{d}}{\mathrm{d}\theta} l(\theta) \right)^2 \right]$$

当待估参数个数大于1时，极大似然估计量渐进服从多元正态分布，协方差矩阵等于Fisher信息矩阵的逆。

Delta方法 记 $\hat{\boldsymbol{\theta}}_n$ 渐进服从多元正态分布，均值为 $\boldsymbol{\theta}$、协方差矩阵为 $\boldsymbol{\Sigma}/n$。令 $g(\cdot)$ 是可微函数，则 $G_n = g(\hat{\boldsymbol{\theta}}_n)$ 也渐进服从多元正态分布，均值为 $g(\boldsymbol{\theta})$、协方差矩阵为 $(A^T \boldsymbol{\Sigma} A)/n$，其中 $A = (\frac{\mathrm{d}g}{\mathrm{d}\theta_1}, \cdots, \frac{\mathrm{d}g}{\mathrm{d}\theta_r})^T$。对单参数模型，$g(\hat{\theta})$ 渐进服从正态分布，方差为 $[g'(\theta)]^2 \cdot \text{Var}(\hat{\theta})$。

3. 置信区间

当极大似然估计量 $\hat{\boldsymbol{\theta}}$ 渐进服从正态分布时，可以为每个参数 $\theta_j$ 构造独立的置信区间，例如置信水平 $100(1-\alpha)\%$ 的区间为 $\hat{\theta}_j \pm z_{1-\alpha/2} \cdot \sqrt{\text{Var}(\hat{\theta}_j)}$。基于似然的方法，可构造置信域 $\{\boldsymbol{\theta}: l(\boldsymbol{\theta}) \geq c\}$，其中 $c = l(\hat{\boldsymbol{\theta}}) - 0.5 \cdot \chi_{1-\alpha}^2(r)$。

1.5.3 拟合优度检验

1. 卡方拟合优度检验

卡方拟合优度检验主要思想是将数据分组后，比较每组观测频率与理论频率的差异。原假设为 $H_0$：随机变量 $X$ 的分布为 $F(x;\theta)$。

将实数轴划分为若干个子区间 $(c_{j-1},c_j]$，应用分布函数计算理论概率 $p_j = F(c_j; \hat{\theta}) - F(c_{j-1}; \hat{\theta})$，理论频数 $E_j = np_j$。记 $O_j$ 为落入各区间的样本数量，则卡方检验统计量为：

$$\chi^2 = \sum_{j=1}^r \frac{(O_j-E_j)^2}{E_j}$$

当样本量 $n$ 足够大时，检验统计量应收敛于自由度 $k = r-1-m$ 的卡方分布（$m$ 为需要估计的参数个数）。需要注意，卡方拟合优度检验要求每个区间的期望频数 $E_j \geq 5$。

例 1.10（维修费用的卡方检验） 假设例1.7中的维修成本观察值可以被划分为以下区间，检验指数分布是否能够有效拟合该样本的损失。
维修费用区间频数
$(0, 1000]$250
$(1000, 2000]$300
$(2000, 3000]$250
$(3000, 4000]$150
$(4000, 5000]$100
$(5000, \infty)$0

假设维修费用服从指数分布，利用估计值 $\hat{\lambda} = 1/2200$，可以计算每个区间上的理论概率。例如，个体观察值落入区间 $(2000,3000]$ 的概率为：

$$\int_{2000}^{3000} \lambda e^{-\lambda x} \mathrm{d}x = e^{-2000\lambda} - e^{-3000\lambda} = 0.1472$$

该区间的理论频数为 $1000 \times 0.1472 = 147.2$。其他区间的理论频数分别为：365.3, 231.8, 147.2, 93.4, 59.3, 103.0。

计算卡方统计量：

$$\chi^2 = \frac{(250-365.3)^2}{365.3} + \frac{(300-231.8)^2}{231.8} + \cdots + \frac{(0-103.0)^2}{103.0} = 293.48$$

总共6个区间，存在1个待估参数，故检验统计量的自由度应为 $6-1-1=4$。而卡方检验统计量的值远大于14.86（自由度为4的卡方分布的99.5%分位数），故有很大把握拒绝原假设 $H_0$，认为维修成本不服从指数分布。

# 卡方拟合优度检验：维修费用数据
lower = np.array([0, 1000, 2000, 3000, 4000, 5000])
upper = np.array([1000, 2000, 3000, 4000, 5000, np.inf])
freq = np.array([250, 300, 250, 150, 100, 0])

# 指数分布MLE
midpoints = np.array([500, 1500, 2500, 3500, 4500, 5500])
n_total = freq.sum()
xbar = np.sum(freq * midpoints) / n_total
lambda_hat = 1 / xbar

# 期望频数
p_exp = stats.expon.cdf(upper, scale=1/lambda_hat) - \
        stats.expon.cdf(lower, scale=1/lambda_hat)
exp_freq = n_total * p_exp

# 卡方统计量
chi_sq = np.sum((freq - exp_freq)**2 / exp_freq)
df = len(freq) - 1 - 1
p_value = 1 - stats.chi2.cdf(chi_sq, df)
print(f"卡方拟合优度检验（指数分布）:")
print(f"  lambda_hat = {lambda_hat:.6f}")
print(f"  chi_sq = {chi_sq:.4f}, df = {df}, p-value = {p_value:.4f}")

2. K-S检验

K-S检验（Kolmogorov-Smirnov tests）是一种适用于连续型分布的检验方法，通过对累积分布函数进行比较，判断一组样本是否来自某个特定分布。给定样本 $(X_1, X_2, \cdots, X_n)$，其经验分布函数为：

$$F_n(x) = \begin{cases} 0, & x < X_{(1)} \\ k/n, & X_{(k)} \leq x < X_{(k+1)} \\ 1, & x \geq X_{(n)} \end{cases}$$

对于单样本K-S检验，其检验统计量为 $D_n = \sup_{-\infty < x < +\infty} |F_n(x) - F_0(x)|$。当 $D_n$ 的值较小时，可以认为 $F_n(\cdot)$ 与 $F_0(\cdot)$ 相等；否则应拒绝原假设。K-S检验的临界值 $D_{n,\alpha}$ 依赖于样本量 $n$ 以及检验的显著性水平 $\alpha$。
